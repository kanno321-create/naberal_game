## 🎨 다크판타지 정수 (5 답변 횡단 작법 원칙)

### 1. Showing 우선 / Telling 절대 회피
- 권력 위계: *시종 → 길드장 → 공작* 단계 배치 (Telling = "그는 신분이 높았다" 금지)
- 차별: *광장 단역 한 마디 + 시선/공기 + 세계관 용어/금기* (Telling = "마족은 차별받았다" 금지)
- 페널티: *옆구리 젖음 + 무릎 무너짐 + 시야 잿빛* (Telling = "그는 절망감에 휩싸였다" 금지)

### 2. 미세 비트 50자 이내 + em-dash 호흡
- 라운드 2 새 톤 그대로 유지
- 비트는 *부담 없이 짧게* — 1~2 줄 박제로 충분
- em-dash (—) = 한 박자 깨짐 / 시간차 / 자아 균열 표시

### 3. 시간차 누수 (Time-Lag of Pain)
- 행위 직후가 아닌 *며칠 후 발현*
- 새로운 에피소드 시작 아침 / 휴식 씬 / 시점 전환 시 박제
- 베르세르크 + 진격의 거인 + Bleach 검증 패턴

### 4. 곁의 사람 부식 (감정 누수 전염)
- 주인공 절망/감정 누수 → 동료 *원인 모를 우울* → 육체적 부식
- Vol.1 미세 → Vol.2 병적 우울증 → Vol.3 물리적 부식 점층
- Made in Abyss + Drakengard/Nier 검증 패턴
- **현재 미리암 4회 누수** = Vol.1 권장 빈도 정합

### 5. 씨앗 → 회수 → 폭발 (오버로드식 미학)
- 1권 = 씨앗만 박제 (독자가 *무언가 있다* 직감)
- Vol.2 = 점층 회수 (조기 만족 + 중기 확장 + 후기 자아 균열)
- Vol.2 클라이맥스 = *진실의 일거 폭발* (모든 씨앗이 한 번에 의미를 가짐)
- 진격의 거인 + Drakengard/Nier + 오버로드 검증 패턴
