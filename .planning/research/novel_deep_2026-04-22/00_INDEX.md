## 🎯 나이트 프로젝트 특화 크로스 레퍼런스

### "나이트" 캐릭터 설계 축 (Vol.2 전용)
- 정체성 = 쿨데레 + "한결같음의 의미를 모르는 수호자" [Vol.2 §3-1]
- DNA 합성 = 아인즈(오버로드) + 2B(Nier) + 해리(디스코) [Vol.2 §3-4 / §3-6]
- 약점 구조 = 물리×·심리·존재·관계·서사 4축 [Vol.2 §3-4]
- 갭모에 균열 = Act 2 에서 1회 노출 [Vol.2 §3-2]
- 음성화 = ~다 기본·주어 생략·침묵이 감정 [Vol.2 §2-8 / §3-6]

### Act 1 (1인칭 Ch.01~11) 적용 규칙
- POV 전환 = 챕터 경계에서만 [Vol.2 §2-2]
- 종결어미 분포 = ~다 70%+ / ~지 10~15% [Vol.2 §2-8]
- 감정 표현 등급 = 4~7 주력, 1번 사용 ≤ 3회/챕터 [Vol.2 §2-4]
- 합쇼체 기본 · 위기 시 해라체 독백 낙차 [Vol.1 §3.2 / Vol.3 §8-1]

### Act 2~3 (3인칭 + 다중POV) 적용 규칙
- 적 진영 POV 챕터 = Act 2 첫 챕터 권장 [Vol.2 §권고]
- 오버로드式 다중 POV 교차 = 왕국편·성왕국편 패턴 [Vol.3 §5-2]
- 서사 아이러니 3층 (코미디·공포·비극) [Vol.3 §5-3]
- 수호자의 의도 과대해석 구조 반복 활용 [Vol.1 §3.7 / Vol.3 §5-3]

### 엔딩 5구조 × 게임 IP 연동
- Happy / Bad / True / Normal / Secret 배치 전략 [Vol.3 §적용권고]
- True Ending = 다른 4개 경험 후만 해금 (비주얼노벨식) [Vol.3 §적용권고]
- 수정 1/2 이중 관리의 진실 = True Ending 핵심 [Vol.3 §적용권고]

### 세계관 철학 3조와의 연동
- ①불완전성 ↔ Mirror 악역 설계 "완벽한 악역 없음" [Vol.2 §3-7]
- ②나이트 한결같음 ↔ 음성화 · 침묵 · 감정 등급 5~7 [Vol.2 §3-6]
- ③영혼 평등 ↔ As-You-Know-Bob 회피 · 위계적 언어 금지 [Vol.1 §3.2]

## 📋 상위 1% 체크리스트 — 통합 (40항목)

### 포맷·문법 (9)
- [ ] 단락 3문장 초과 여부
- [ ] 대화문 「」 내 마침표 없음
- [ ] 삼점리더 …… 짝수 사용
- [ ] 대시 —— 짝 사용
- [ ] 오노마토페 3회 연속 경고
- [ ] 숫자 아라비아/한글 혼용 기준
- [ ] 회차 분량 3K~5K자 범위
- [ ] 문장부호 일관성 (「」 또는 " " 고정)
- [ ] 「」 스타일 유지 여부 (LN 감성)

### 대화·화자 (8)
- [ ] "당신도 알다시피" 테스트 (As-You-Know-Bob)
- [ ] No-tag Test (대화 태그 제거 후 화자 구별)
- [ ] 감정 형용사 지문 최소화
- [ ] 동일 발화 동사 3연속 이상 없음
- [ ] 내적 독백/외부 대사 구분
- [ ] 침묵/공백 의도적 ≥1회
- [ ] 성우 감정 해석 공간 확보
- [ ] 어체 혼용 없음 (합쇼/해요/해라)

### 문체·리듬 (10)
- [ ] 문장 길이 표준편차 ≥ 8어절
- [ ] 한 줄 단락 1~3개 전략 배치
- [ ] 감정 표현 등급 4~7 주력
- [ ] 등급 1 사용 ≤ 3회/챕터
- [ ] POV 전환은 챕터 경계에서만
- [ ] 클리셰 비유 회피 (칼날 눈빛·심장 쿵쿵)
- [ ] Act 1 종결어미 ~다 70%+
- [ ] 챕터 rise→impact→transition 3단계
- [ ] 적 POV 챕터에서 나이트를 "경이"로 묘사
- [ ] 주어 생략 감정절정 3~5문장 연속

### 캐릭터·세계관 (10)
- [ ] 나이트 "우리 포인트" 한 문장
- [ ] 갭모에 균열 지점 Act 2 이후 노출
- [ ] 캐릭터 시트 Will/Need/Wound/Misbelief
- [ ] 나이트 약점 4영역 설계
- [ ] 조연 No-tag Test 통과
- [ ] 악역 Mirror 기법 (동일 목표·다른 방법)
- [ ] 앙상블 6역할 중 3+ 확보
- [ ] 나이트 일러스트 5요소 + 금지항목
- [ ] 악역 "한 번은 옳은" 장면 Act 2 내
- [ ] 나이트 발화량 장면당 3~5문장 한도

### 구조·상업 (3)
- [ ] 단권 기승전결 전(転) 40% 비율
- [ ] 아크 서파급 10:80:10
- [ ] 복선 3층 설계 (1·2·3층 각 1+)
