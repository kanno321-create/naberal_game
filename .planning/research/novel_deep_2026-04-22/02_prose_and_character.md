# 라이트노벨 딥 리서치 Vol.2 — 문체·템포 & 캐릭터 공학

## 1. TL;DR (200자 요약)

LN 문체의 핵심은 "짧은 문장 → 긴 문장 → 침묵"의 리듬 건축이며, 오버로드式은 Tell을 전략적으로 사용하는 냉정 서술로 독자 거리감을 의도적으로 설계한다. 캐릭터 공학의 정수는 갭모에 설계 + 음성화(고유 어휘·리듬) + 동기의 이중 구조(표면 동기 vs 심층 동기)이다. "나이트"는 2B·아인즈의 DNA를 한국어 존댓말 문체로 재구성한다.

## 2. 영역 3 — 문체·템포

### 2-1. Staccato 리듬 공학 — 문장 길이 설계

**핵심 원리**: 문장 길이는 독자의 심박수를 직접 제어한다. 짧은 문장은 페이지를 가속하고, 긴 문장은 감속한다. 단, LN에서는 이 규칙이 더 극단적으로 적용된다.

**실전 공식 (3층 리듬 구조)**:
```
긴장 상승  →  짧은 문장 연속 (3~7어절)
이완 구간  →  긴 문장 + 내면 묘사 (15~25어절)
전환점    →  한 줄 단락 (1~3어절)
```

**구체 기법**:
- **Hard -ed / 1음절 어휘**: 행동 장면에서 단음절·동사형 어휘 집중 → 심박 가속
- **-ing형(진행형) 문장**: 이완 구간에서 진행형·연속형 어미로 여운 연장
- **의도적 단편 문단**: 감정 절정 직전 또는 직후 단 한 줄 문단 배치
- **표준편차 설계**: 한 챕터 내 문장 길이 표준편차 ≥ 8어절 유지 권장 (단조로움 방지)

**오버로드 실적용 예시 (Vol.1 기반 분석)**:
```
[냉정 서술 구간]
아인즈 울 고운은 그 광경을 바라보았다.
감정이 없었다.
정확히는, 감정을 처리할 기관이 소멸해 있었다.

→ 3문장. 8/5/12어절. 짧음→더 짧음→중간 길이의 역순 구조.
→ 마지막 문장에서 "소멸해 있었다"라는 과거 완료형이 거리감 생성.
```

**Before / After 예시**:
```
[Before — 평범한 묘사]
나이트는 전장을 보면서 아군이 쓰러지는 것을 바라보았고, 그것이 슬프다고
느꼈지만 어떻게 해야 할지 몰랐다.

[After — Staccato + 냉정 서술]
전장이 무너졌다.
나이트는 그것을 보고 있었다.
슬픔인지 확인할 방법이, 그에게는 없었다.
```

**추천 도구**: 퇴고 시 각 챕터의 문장 길이를 스프레드시트에 나열하고 표준편차 측정. 표준편차 < 5이면 리듬 단조로움 의심.

### 2-2. POV 시점 전략 — 1인칭 vs 3인칭 제한 vs 오버로드式 다중 POV

**대표님 프로젝트 적용 시점 구조**:
| Act | 시점 | 근거 |
|-----|------|------|
| Act 1 (Ch.01~11) | 1인칭 | 독자 = 나이트와 동일 시야. 정보 제한 극대화 |
| Act 2 (Ch.12~) | 3인칭 제한 | 세계관 확장 + 다중 진영 묘사 |
| Act 3 | 혼용 | 오버로드式: 챕터 단위로 POV 전환 |

**각 시점의 공학적 특성**:

**1인칭 밀착**:
- 장점: 독자 감정이입 최대화, 내면 독백 자연스러움, 정보 비대칭 연출 용이
- 단점: 주인공이 모르는 정보 서술 불가, 장기 연재 시 시야 협소화
- LN 표준: 페이지의 60~70%가 주인공 내면 + 관찰로 채워짐
- **나이트 적용 주의**: 1인칭이지만 감정 직접 서술 최소화 → "감정을 설명하지 않는 1인칭"

**3인칭 제한**:
- 장점: 서술자 거리 조정 가능, 여러 진영 묘사, 독자에게 "신의 시각" 부분 허용
- 단점: 1인칭보다 감정이입 미세 감소
- 오버로드의 사례: 3인칭이지만 아인즈 내면 독백 폭발적 → "내면 독백이 서술보다 많은 3인칭"

**POV 전환 타이밍 규칙**:
```
허용: 챕터 경계 (가장 안전)
조건부 허용: 섹션 경계 (*** 구분선 필수)
원칙 금지: 단락 내 무단 전환 (독자 혼란 유발)
```

**오버로드式 다중 POV 기법**:
- 메인 POV (아인즈) + 적 진영 POV를 번갈아 구성
- 적 진영 챕터에서 아인즈는 "공포의 대상"으로만 서술 → 주인공 신비감 유지
- 이 기법은 독자가 주인공을 "외부에서 보는" 경험을 제공 → 캐릭터 신격화 효과

**대표님 프로젝트 권고**:
- Act 1: 순수 1인칭 유지. 나이트가 모르면 독자도 모른다.
- Act 2 시작: 챕터 1개를 적 진영 POV로 시작 → 독자에게 "나이트가 얼마나 강한가"를 간접 체감시킴
- Act 3 혼용: 챕터 단위 전환, 섹션 내 전환 금지

### 2-3. Show vs Tell — LN의 현대적 재해석

**LN이 Tell을 더 많이 쓰는 이유 (실증적 분석)**:

1. **매체 특성**: LN은 주 1회 연재 → 독자 기억 보조 서술 필요
2. **내면 독백 문화**: 일본 LN은 서양 MFA 기준 "Show"보다 내면 서술("Tell")이 구조적으로 더 많음
3. **모바일 가독성**: 긴 장면 묘사 → 독자 이탈. Tell이 더 빠른 정보 전달
4. **웹소설 진화**: 한중일 모두 실용적 Tell 우선. "동작을 통한 감정 시사" 대신 "직접 설명 후 장면 전환"

**LN 최적 혼합 비율 (업계 경험칙)**:
```
액션/긴장 구간: Show 70% / Tell 30%
감정/성격 묘사 구간: Show 40% / Tell 60%
세계관 설명 구간: Show 20% / Tell 80%
```

**"냉정한 Tell"의 기술** (오버로드式):
- 감정을 설명하되 감정을 느끼지 않는 듯 서술
- 주인공이 자신의 감정에 "분석적으로" 접근
```
[일반 Tell] 그는 슬펐다.
[냉정한 Tell] 슬픔이라고 분류해야 할 감각이, 그의 논리 회로 어딘가에 걸려 있었다.
```

### 2-4. 감정 표현 스펙트럼 — 7단계 등급

| 등급 | 방식 | 예시 | 적합 장면 |
|------|------|------|----------|
| 1 | 직접 선언 | "그는 분노했다" | 지양 (LN에서 가장 약한 표현) |
| 2 | 감각 서술 | "주먹이 떨렸다" | 액션 장면 보조 |
| 3 | 환경 투영 | "창밖 비가 거세졌다" | 분위기 조성 |
| 4 | 내면 독백 단절 | "—생각이 멈췄다" | 충격 장면 |
| 5 | 논리적 분석 (냉정 Tell) | "이 반응는 두려움과 유사하다" | 나이트·아인즈형 캐릭터 |
| 6 | 타인의 반응을 통한 시사 | "옆에 있던 병사가 한 발 물러섰다" | 오버로드式 주력 기법 |
| 7 | 완전 침묵 (행동만) | "나이트는 등을 돌렸다" | 절정 감정 표현 |

**나이트 캐릭터 권고**: 등급 4~7 주력. 등급 1~2는 작중 20회 미만으로 제한.

### 2-5. 수사학·비유 — "뻔한 비유" 회피 리스트

**사용 금지 비유 (클리셰)**:
- "심장이 쿵쿵 뛰었다" → 대체: "심장이 제 리듬을 잊었다"
- "눈물이 볼을 타고 흘렀다" → 대체: "시야가 번졌다. 이유를 확인하지 않았다"
- "그의 눈이 빛났다" → 대체: "동공이 수축했다"
- "칼날 같은 눈빛" → 대체 필요 (기계적 비유 → 우주적 비유)
- "폭풍 같은 감정" → 대체: 구체적 물리 현상

**참신한 비유 공식 (세계관 연동)**:
```
공식: [세계관 고유 개념] + [인간 감각]
예시:
- "Karzor의 마력 파문처럼, 그 분노는 중심이 아닌 경계에서 먼저 드러났다"
- "수호자의 무관심 — 별이 자신을 도는 행성에 이름을 붙이지 않듯"
- "11왕국이 균형을 잡듯, 나이트의 표정도 어떤 중립을 유지했다"
```

### 2-6. 호흡·박자 제어 — 챕터 내 Micro-Cycle 설계

**챕터 감정 아크 표준 구조** (LN 업계 실증 패턴):
```
[챕터 2,000~4,000자 기준]

도입 (10~15%): 장소·상황 그라운딩
상승 (30~40%): 긴장 축적 (짧은 문장 증가)
절정 (10~15%): 감정 피크 (한 줄 단락 배치)
여운 (15~20%): 이완 (긴 문장·내면 독백)
전환 (10%): 다음 챕터 Hook (미해결 질문 투기)
```

**Jean-Louis Vill 분석에서 도출한 원칙**:
- "감정 밀도가 페이지 수보다 중요하다"
- "리듬은 가속 장치가 아니라 건축이다"
- "챕터는 반드시 rise → impact → transition 3단계를 포함해야 한다"

**한 줄 단락의 전략적 배치**:
```
올바른 사용: 반전 직후, 침묵 직전, 챕터 마지막 줄
잘못된 사용: 설명 구간, 세계관 서술, 감정 누적 전
```

### 2-7. 니시오 이신·마루야마 쿠가네 문체 분석

**니시오 이신 (이야기 시리즈)**:
- 특징: 만연체 + 언어유희 + 만담 + 캐릭터 위주 전개
- 쉬운 내용을 "축약 없이 꼬아 쓰는" 역설적 구조
- 대화가 메인 스토리만큼 길어지는 드라마 CD 대본 형식
- 캐릭터 이름에 언어유희 내장 (아라라기 코요미 = 달력)
- **대표님 프로젝트 적용**: 나이트의 독백에 제한적 적용 가능. 과용 금지.

**마루야마 쿠가네 (오버로드)**:
- 특징: 강자가 선이고 약자가 악이라는 냉정한 세계관을 서술 문체에도 반영
- 아인즈 내면 독백 = 감정 없는 분석가의 보고서 형식
- 적 진영 챕터 = 아인즈를 "경이"로 서술하여 주인공 신비감 구축
- Tell이 많지만 Tell 자체가 "냉정함"이라는 Show 역할을 수행
- **오버로드의 역설**: "말이 많은 Tell이 오히려 캐릭터의 감정 부재를 보여준다"
- **대표님 프로젝트 핵심 기법**: 나이트에게 직접 적용 가능

**레:제로 나가츠키 탓페이**:
- 특징: 3인칭이지만 산문보다 대화 선호. "설명이 필요한 내용을 설명이 아닌 대화로 쓰는 것"을 좋아함
- 서브루의 약점 = 물리 능력이 아닌 심리적 내구력으로 설계 (죽음을 반복하는 감정 소모)
- 에밀리아를 먼저 구상하고 스바루는 그를 위한 수난 캐릭터로 역설계

### 2-8. 한국어 특수 기법

**주어 생략의 시적 효과**:
- 한국어는 구조적으로 주어 생략이 자연스러움
- "주어를 꼬박꼬박 쓰면 외국인이 쓴 것처럼 보인다"
- **의도적 생략 기법**: 감정이 고조된 장면에서 주어를 탈락시켜 행위 자체에 집중
```
[주어 있음] 나이트는 칼을 내렸다.
[주어 생략] 칼을 내렸다.
            쓸 이유가 없었다.
```

**종결어미 운용 가이드**:
| 어미 | 기능 | 소설 적용 |
|------|------|----------|
| ~다 | 중립적 선언 | 서술 기본형, 냉정 서술에 적합 |
| ~네 | 새롭게 지각한 사실 | 발견·놀람 장면. 절제해서 사용 |
| ~지 | 청자가 알고 있다는 전제 | 독자와 공유하는 내면 독백 |
| ~군/~구나 | 감탄·새로운 인식 | 감정 표현 시. 과용 시 진부함 |
| ~거든 | 화자만 알고 있는 정보 | 1인칭 독자 밀착 서술 |
| ~을 것이다 | 추론·예상 | 나이트의 분석형 독백에 자주 사용 |

**나이트의 종결어미 전략 (Act 1)**:
- 기본: ~다 (80%) → 냉정·중립
- 감정 표현: ~지 (10%) → 독자와 은밀하게 공유하는 감각
- 놀람: ~네 (5%) → 나이트가 예상하지 못한 순간에만
- 기타: ~군/~거든 (5%) → 극히 절제

## 3. 영역 4 — 캐릭터 공학

### 3-1. 캐릭터 "우리(賣り)" 포인트 — 한 문장 요약 공식

**공식**: `[외형/직함] + [내면 모순] + [독자가 응원하는 이유]`

**예시 (참고작)**:
- 아인즈 울 고운: "전능한 마왕이지만 실제로는 겁쟁이 직장인 — 강자의 가면을 쓴 약자라는 공감"
- 2B (NieR): "완벽한 전투 기계지만 사랑하는 사람을 반복 살해해야 하는 존재 — 의무와 감정의 비극적 모순"
- 스바루 (레:제로): "약하고 실패만 하지만 절대 포기하지 않는 — 의지력이 유일한 능력"

**나이트의 우리 포인트 (설계 제안)**:
```
"우주의 균형을 지키는 수호자지만, 자신이 무엇을 지키고 있는지 확인할 수 없는 존재
— 한결같음이 힘인가, 아니면 선택의 부재인가"
```

### 3-2. 갭모에 설계 — 8종 아키타입 심화

**갭모에의 정의**: 겉모습과 내면이 충돌할 때 발생하는 매력. 정적 귀여움이 아닌 동적 반전으로 작동.

**핵심 공식**: `지배적 인상 확립 → "균열" 노출 → 반전 충격 → 매력 고착`

**8종 아키타입과 갭모에 설계**:

| 아키타입 | 외면 | 내면 갭 | LN 대표 캐릭터 |
|---------|------|---------|--------------|
| 쿨데레 | 냉정·무표정 | 은밀한 충성·고양이 집착 | 나이트, 아인즈, 2B |
| 츤데레 | 공격·거부 | 숨겨진 부드러움 | 토라도라 타이가 |
| 얀데레 | 외형 완벽·상냥 | 집착·파괴성 | 미래일기 유노 |
| 천연 | 순수·무지 | 의도 없는 독설 | 반응집 카가미네 |
| 현자형 | 지식·분석 | 실생활 무능 | 현자의 제자 |
| 샤이(부끄러움) | 위축·무성 | 극도의 열정 보유 | 히카루가 죽었다 |
| 보쿠코 | 남성적 언어 | 여성성 또는 취약성 | 일반적 보이시 캐릭터 |
| 오네에 | 성숙·포용 | 동심·약점 | 오버로드 알베도 |

**갭모에 설계 4단계 프로세스**:
1. 캐릭터의 첫 인상 확정 (독자 기대 설정)
2. "균열 지점" 하나 선택 (너무 많으면 갭이 아니라 일관성 없음)
3. 균열을 언제 드러낼지 타이밍 설계 (등장 후 최소 2챕터 이후)
4. 균열이 캐릭터 핵심 갈등과 연결되게 설계

### 3-3. 캐릭터 시트 — 편집자 표준 비교

**일본 편집자 표준 (이치문코 계열 기준)**:
```
필수 항목:
- 이름 / 나이 / 성별 / 역할
- 외모 (일러스트 가이드용 5요소: 눈색·머리·체형·의상 포인트·특징 소품)
- 성격 핵심 키워드 3개
- 동기 (원하는 것 vs 필요한 것 구분)
- 말투 특징 (종결어미·어휘 수준·말수)
- 다른 캐릭터와의 관계 (1:1 관계도)
- 비밀 (독자가 모르는 것)
- 성장 아크 (시작 → 중간 → 끝 상태)
```

**한국 웹소설 표준 (카카오페이지·네이버 계열)**:
```
추가 항목:
- 능력치 수치 (독자 기대 관리)
- 호칭 체계 (다른 캐릭터가 어떻게 부르는가)
- 갈등 유발 패턴 (이 캐릭터가 어떤 상황에서 갈등을 만드는가)
- 독자 페르소나 여부 (대리인 역할인가)
```

**영미 MFA 표준 (Robert McKee 방식)**:
```
추가 항목:
- Will vs Need (겉으로 원하는 것 vs 실제로 필요한 것의 괴리)
- Wound (핵심 상처 — 성격 형성의 트라우마)
- Ghost (과거 사건 중 현재를 지배하는 기억)
- Misbelief (잘못된 신념 — 아크 해소 대상)
```

**대표님 프로젝트 권고 하이브리드 시트**:
```
영역 1: 외형 (일러스트 가이드 5요소) — 일본 표준
영역 2: 내면 구조 (Will/Need/Wound/Misbelief) — MFA 표준
영역 3: 음성화 (말투·어휘·리듬) — 캐릭터 보이스 전용
영역 4: 관계도 (주인공 기준 거리) — 한국 웹소설 표준
영역 5: 비밀 목록 (독자 공개 타이밍 포함) — 별도 관리
```

### 3-4. 주인공 설계 — 강한 주인공형 약점 공학

**강한 주인공 3대 실패 패턴**:
1. 갈등이 없어 서사가 평탄해짐
2. 독자가 감정이입할 약점이 없어 관계 형성 실패
3. 강함만 보여주다 캐릭터 깊이 부재

**"아인즈 울 고운" 약점 설계 분석**:
- 물리적 약점: 거의 없음
- 심리적 약점: 부하들에게 완벽한 지도자로 보여야 한다는 강박
- 존재적 약점: 인간 감정을 잃어버렸다는 근원적 상실
- 관계적 약점: 진심으로 신뢰하는 동료가 없음 (모두 자신보다 나자릭 우선)
- **이 구조의 핵심**: "강하지만 아무도 구원해주지 않는다"

**2B (NieR Automata) 약점 설계**:
- 외형: 완벽한 전투 기계 → 취약점 없어 보임
- 실제: 자신이 사랑하는 9S를 반복 처형해야 하는 역할 → 강함이 고통의 원인
- "감정 금지"라는 규칙이 오히려 감정의 크기를 암시
- 침묵이 말하는 것 이상의 감정을 전달

**나이트에게 권고하는 약점 구조**:
```
물리: 없거나 거의 없음 (수호자로서 필요)
심리: "한결같음"이 선택의 부재인지 확신의 부재인지 자신도 모름
존재: 영혼 평등 원칙을 따르면서도 특정 존재를 더 오래 바라보게 됨
관계: 인간들이 나이트를 신으로 대하지만 나이트는 그것을 거부할 방법을 모름
서사적: 강함 때문에 오히려 아무것도 배울 수 없다는 고립
```

### 3-5. 조연 설계 — 주인공 돋보이기 + 독자 페르소나

**조연의 3가지 기능**:
1. **대조군**: 주인공의 특성을 역방향으로 부각. 나이트가 침묵이면 조연은 언어과잉.
2. **독자 페르소나**: 독자가 모르는 것을 대신 질문. 세계관 설명 자연화.
3. **갈등 발생기**: 주인공이 직접 유발하면 비호감인 갈등을 조연이 트리거.

**앙상블 캐스트 4~8인 역할 분담 (던전밥 기반 분석)**:
| 역할 | 기능 | 던전밥 사례 |
|------|------|------------|
| 리더 | 목표 방향 설정 | 라이오스 |
| 감정 앵커 | 독자 감정이입 대리 | 마르실 |
| 실무 전문가 | 현실적 문제 해결 | 센시 |
| 회의주의자 | 독자 의심 대변 | 칠치쿠 |
| 야생 변수 | 예측 불가 행동 | (이즈츠미) |
| 희생/감동 | 감정 피크 생성 | 서브 캐릭터들 |

**핵심 원칙**: 던전밥은 주인공이 조연보다 더 스포트라이트를 받지 않는다 → "앙상블 형" 구조. 일반 이세계물과 반대.

### 3-6. 캐릭터 보이스 (Character Voice) — 음성화 7기법

독자가 대화 태그 없이도 누가 말하는지 알아야 한다 ("No-tag Test").

**7가지 음성화 기법**:

1. **어휘 수준 차별화**: 나이트 = 간결·기술적 어휘 / 일반 병사 = 구어체
2. **문장 길이 패턴**: 냉정형 = 짧은 문장 연속 / 감정형 = 긴 문장 + 이탈
3. **종결어미 고착**: 캐릭터마다 주력 어미 1~2개 고정 (나이트: ~다, ~것이다 / 조연 A: ~잖아요, ~거든요)
4. **메타포 영역**: 각 캐릭터의 비유를 그들의 배경 영역에서만 추출 (군인 = 전술 비유, 상인 = 거래 비유)
5. **침묵의 패턴**: 나이트는 언제 침묵하는가를 일관되게 설계
6. **과거 트라우마 언어 반향**: 특정 단어가 나올 때 특정 방식으로 반응
7. **반복 구절(Catchphrase)**: 과용 금지. 1캐릭터 당 1구절. 나이트에게는 권장하지 않음 (오히려 의미 손상)

**나이트 전용 음성화 설계**:
```
말수: 극단적으로 적음 (장면 평균 발화 3~5문장)
어휘: 추상적 개념 대신 구체 현상 기술 ("슬픔" 대신 "이 상황의 비효율")
구조: 주어+동사만. 수식어 최소.
침묵: 감정적 질문에 대답하지 않거나 다른 주제로 전환
예외 (균열 지점): 특정 대상 앞에서만 문장이 길어지거나 주어가 생략됨
```

### 3-7. 악역 설계 — "불완전성" 원칙

**세계관 철학 ①불완전성과의 연동**:

완벽한 악역 = 서사 실패. 이유:
1. 독자가 감정이입할 수 없음
2. 주인공 승리가 도덕적으로 너무 단순해짐
3. 세계관의 "우주적 균형" 철학과 충돌

**Antagonist Mirror 기법 (비추천 작법에서 권장 작법으로)**:
```
단순 악역 (금지): "세상을 지배하고 싶다" + 도덕적 결함만 있음
Mirror 악역 (권장): 주인공과 같은 목표를 다른 방법으로 추구
                    나이트가 지키려는 것을 악역도 지키려 한다
                    단지 그 방법이 나이트와 양립 불가능할 뿐
```

**"믿음의 악역" 설계 공식**:
1. 악역의 목표가 이해 가능해야 함 (최소한 그들의 논리 안에서)
2. 주인공과의 충돌이 순수한 도덕 문제가 아닌 "양립 불가능한 필연"
3. 악역이 한 번은 옳아야 함 (주인공 실수를 지적하거나 올바른 결론 도출)
4. 악역의 "불완전성"을 노출 (강함 속의 공포, 확신 속의 의심)

---

### 3-8. 일러스트 가이드 작성 기법

**작가→일러스트레이터 전달 업계 표준 (A6 LN 기준)**:

**필수 5요소**:
1. **눈**: 색상 + 눈매 형태 (날카로움/둥근/처진) + 특수 효과 (빛 없음/별자리형)
2. **머리**: 색상 + 길이 + 스타일 + 질감 (매끄러움/거침)
3. **체형**: 키·체중 느낌 + 근육형/슬림형 + 자세 특징
4. **의상 포인트**: 1~2가지 상징적 디테일 (망토의 가장자리 처리, 문양 위치)
5. **표정 레퍼런스**: 감정 없을 때의 표정 묘사 (특히 쿨데레형)

**추가 전달 항목**:
- "이 장면에서 캐릭터의 감정 상태" 1문장
- "피해야 할 표현" (예: 눈을 반짝이게 그리지 말 것)
- "강조해야 할 요소" (예: 피로한 눈 주위, 정돈되지 않은 머리카락 한 가닥)

**나이트 일러스트 가이드 초안**:
```
눈: 어두운 색상 계열. 초점이 흐리거나 먼 곳을 보는 느낌.
    눈 안의 하이라이트 최소화 또는 생략 (감정 없음 암시).
머리: 짧지 않음 (우주적 여행자 이미지). 관리되지 않은 자연스러움.
체형: 전투형보다 균형 잡힌 중간형. 어깨 약간 아래로 처짐 (긴장 없음).
의상: 우주 수호자의 상징물이 1개 이상. 특정 색에 집착하지 않음.
표정: 무표정이되 차갑지 않음. "피로한 지혜"의 느낌.
금지: 눈을 빛내거나 환하게 웃는 표정. 과장된 포즈.
```

## 4. 상위 1% 체크리스트

### 문체·템포 체크리스트
- [ ] 챕터 내 문장 길이 표준편차 ≥ 8어절 (리듬 변동성 확보)
- [ ] 챕터마다 한 줄 단락 1~3개 전략적 배치
- [ ] 감정 표현 등급 4~7 주력 (등급 1 사용 횟수 챕터당 3회 이하)
- [ ] POV 전환은 챕터 경계에서만 (섹션 내 전환 금지)
- [ ] 클리셰 비유 리스트 검토 후 대체 비유 적용
- [ ] Act 1 종결어미 분포: ~다 70%+, ~지 10~15%
- [ ] 챕터 구조: rise→impact→transition 3단계 확인
- [ ] 적 진영 POV 챕터에서 나이트를 "경이"로만 묘사
- [ ] Tell 구간에서 "냉정한 Tell" 기법 적용 (분석가 보고서 형식)
- [ ] 주어 생략 기법: 감정 절정 장면 3~5문장 연속 생략

### 캐릭터 공학 체크리스트
- [ ] 나이트의 "우리 포인트" 한 문장 완성
- [ ] 갭모에 균열 지점 1개 설정 + 등장 후 2챕터 이후 노출 타이밍
- [ ] 캐릭터 시트: Will/Need/Wound/Misbelief 4요소 모두 작성
- [ ] 나이트 약점: 물리 외 심리·존재·관계·서사 4영역 모두 설계
- [ ] 조연: 각자의 "No-tag Test" 통과 (대화 태그 없이 구별 가능)
- [ ] 악역: "Mirror 기법" 적용. 나이트와 동일 목표 다른 방법 구조 확인
- [ ] 앙상블 캐스트: 6역할 중 3개 이상 확보 (리더·감정앵커·실무전문가)
- [ ] 나이트 일러스트 가이드: 5요소 + 금지 항목 작성
- [ ] 악역 "한 번은 옳은" 장면 1개 이상 Act 2 내 배치 계획
- [ ] 나이트 발화량: 장면당 3~5문장 한도 설정

## 6. 대표님 프로젝트 적용 권고

### 추천 기법 (나이트 캐릭터 + 오버로드式 하이브리드)

**최우선 적용 (즉시)**:
1. **감정 표현 등급 5~7 전략**: 나이트의 감정을 직접 선언하지 말고 "논리적 분석" 또는 "타인 반응"으로만 시사. 독자가 감정을 "발견"하게 만드는 구조.
2. **한 줄 단락 배치**: 각 챕터 1~3개. 감정 피크 직후 또는 전환점에 배치. 가장 빠른 템포 효과.
3. **종결어미 ~다 기본화**: Act 1 서술 90% 이상 ~다 형식 유지. 감정 표현 시에만 ~지 허용.
4. **No-tag Test**: 매 챕터 완성 후 대화 태그 제거하고 화자 구별 가능한지 확인.

**중기 적용 (Act 2 시작 전)**:
5. **적 진영 POV 챕터**: Act 2 첫 챕터를 적 진영 시각으로 시작 → 나이트를 "신화적 존재"로 외부 묘사. 독자의 기대감 재설정.
6. **캐릭터 시트 MFA 추가**: 기존 일본식 설정에 Will/Need/Wound/Misbelief 4항목 추가. 악역 포함 전 캐릭터 적용.
7. **Mirror 악역 설계**: Karzor 진영 주요 악역을 나이트의 "다른 선택지"로 설계. 양립 불가능하되 이해 가능한 논리 부여.

**장기 적용 (Act 3 준비)**:
8. **갭모에 균열 노출 타이밍**: Act 2에서 나이트의 "균열 지점"(특정 존재 앞에서의 변화)을 1회 노출 → Act 3에서 심화.
9. **POV 혼용 설계**: Act 3의 챕터 분류표 작성. 나이트 POV / 적 POV / 조연 POV 비율 계획.

### 비추천 (경계선)

**사용 금지**:
- 나이트에게 등급 1 감정 표현 ("그는 슬펐다" 형식) 3회 이상/챕터
- 챕터 내 POV 전환 (섹션 구분선 없이)
- 클리셰 비유 리스트의 비유 직접 사용 (특히 "칼날 같은 눈빛", "심장이 쿵쿵")
- 조연들의 말투 혼동 (No-tag Test 실패 용납 금지)
- 악역의 완전한 "악" 설정 (세계관 철학 ①불완전성 위배)

**제한적 사용**:
- 니시오 이신式 언어유희: 나이트 독백에 1~2회/챕터 최대. 나이트 캐릭터 기조와 충돌 위험 높음.
- Catchphrase 기법: 나이트에게 적용 금지. 조연에게만 1개 한정.
- 내면 독백 과다 사용: 오버로드 모방 시 번역 투 서술 위험. "냉정한 Tell"은 인정하되 연속 5문단 이상 독백 금지.
