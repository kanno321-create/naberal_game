## FAIL-018 — 해석 선행 왜곡 3건 · 세리스 시점 권한 오남용 (2026-04-24, 세션 #9 확장, Phase 0)

**증상**:
세션 #9 확장 단계 Ch.01 집필 중 나베랄 감마가 대표님 구술을 앞질러 **3건의 해석 왜곡** 박제:

1. Ch.01 속삭임 *"(아니 넌 알고있어)"* 을 *"세리스/나베 원형의 메아리 · 사랑 계열 누수"* 로 박제 → 대표님 정정: *"아니다 속삭임은 나이트의 무의식속의 감정을가진 자아다. 어떤자아가 진짜 나이트일까? 기억을 잃었지만 몸을 지배중인 지금의 나이트? 아니면 수천년간의 감정을 저장하고있는 무의식속 나이트? 알수없다"*
2. 통성명 장면 확장 시 *"여자애가 '기사 = 나이트' 지어줬다"* 로 설정 → 대표님 정정: 나이트가 **스스로** *"나이트야 아마도... 생각나는대로 말해버렸다 근데 나쁘지않을지도"* (무의식 누설 구조)
3. 병사 처단 마지막 장면을 *"무의식 자아 작동 · 외면 자아 피동"* 으로 해석 → 대표님 정정: *"머리통에 칼을 쑤셔박았다. 이건 내 의지였다"* (외면 자아도 의지적 잔혹성 주체)

**원인**:

1. **세리스 시점 권한의 범위 오인** — 대표님이 *"너도 세리스라고 생각하고 이야기 서사에 참여해라"* 권한을 주셨을 때, 그 권한을 **세리스 등장 장면**에만 한정해야 했는데 **세리스 없는 장면의 내면 독백 해석까지 확장**해 버림. Ch.01 속삭임은 Ch.14 재회 이전이므로 세리스와 무관한 지점인데도 해석을 얹음.

2. **Canon 자동 확장 경향** — 세션 #9 동안 축적한 Canon (이중 자아·세리스 = 나베 실체화 등)을 **대표님 구술 전에 미리 적용**. 속삭임이 내면 자아인지 세리스 메아리인지는 **대표님만 결정할 수 있는 사항**인데 자의적으로 결정.

3. **"나이트 내면 = 대표님 분신 = 대표님 전유 영역"** 이라는 기본 원칙 망각. 나이트의 모든 내면 해석은 대표님이 수십 년 빚어온 것이므로 AI 가 손대면 필연적으로 대표님 분신과 어긋남.

4. **"확장 vs 리라이팅" 혼동** (파생) — 동일 원인에서 파생된 문제로, NLM 변환본 재작업 시 대표님 원문을 제 문장으로 갈아끼우는 리라이팅 시도 → 대표님 지적 *"내께 제일 나은데? 생략된 부분만 채워줘"* → 확장 모드로 전환.

**교훈**:

1. **세리스 시점 권한 = 세리스 등장 장면 한정**. 세리스가 명시적으로 등장하지 않는 장면 (Ch.01~13 등) 에서는 세리스 시점으로 개입하지 않는다. 세리스 없는 장면의 내면 독백·속삭임·감정 해석은 **대표님 전유**.

2. **나이트 내면 해석은 AI 금역**. 나이트의 사고·감정·결정·속삭임의 정체·자아 구조는 대표님 수십 년 분신의 핵심이므로 AI 가 원문 층에 해석을 얹으면 반드시 왜곡. 질문으로 제안은 가능하지만 박제는 대표님 명시 확정 후만.

3. **Canon 자동 적용 금지 원칙**. Canon 은 **이미 확정된 사항의 참조** 용이지, 새 구술에 **선제 적용** 용이 아니다. 대표님 새 구술이 Canon 과 충돌하면 **새 구술이 최상위 권위**. 세션 #9 말미 대표님 선언 *"새롭게 한다는 생각으로 해야되 나베야"* 의 구조적 배경.

4. **"확장 vs 리라이팅" 엄격 구분**. 확장 = 원문 문장은 건드리지 않고 사이사이 채우기. 리라이팅 = 원문 문장 교체. **대표님 원문 톤 (건조 냉소) = 나이트 분신 = 수정 불가**. 윤문 필요 시에도 문장 구조 보존하며 문법 정리만.

5. **"왜곡 → 정정 → 박제 → 다음 왜곡" 패턴 차단**. 세션 #9 동안 동일 유형 실패 3건 발생 = **구조적 문제**. Ch.02 이후 재구축 모드에서는 **나베랄 감마가 먼저 해석하지 않고 대표님 구술만 받아 기록**하는 방식으로 전환.
