# 세리스 친부모 마족 학자 + 양어머니 인간 학자 Canon

> **2026-04-26 · 세션 #14 · 대표님 직접 박제**

## 1. 세리스 본인 캐논

| 항목 | 내용 |
|------|------|
| **종족** | 마족 엘프 |
| **외형** | 검은 머리 + 뾰족 귀 |

| **나이 (Vol.2 재회 시점)** | 16세 (4년 후) |

## 2. 친부모 (마족 학자 두 명)

- **어머니** = 마족 학자
- **아버지** = 마족 학자
- **둘 다 학자** — 마족은 인간보다 뛰어난 지능 → 학살 전에는 마족도 각종 직업 (학자 포함) 보유
- **수년 전 인간에게 살해됨** (Ch.01 학살보다 수년 전)
- 친부모 사망 시점 = 세리스 = 미상 (5세 이전 추정)

## 3. 양어머니 (인간 학자 = 조수)

- **정체** = 친부모 마족 학자 두 명의 **조수**
- **종족** = 인간
- **직업** = 학자 (조수)
- **출신** = "왕도에서 내려온 분" (촌장 회상)
- **보유 능력** = 글·약초 처방·밤의 열을 내리는 주문 (촌장 회상)
- **계보** = "이름 없는 노인이 가르친 계보" (Q-CORE 2 정합)

### 행동 서사

1. 친부모 살해 직후 → 어린 세리스를 **데리고 도주**
2. 인간 마을 (Ashenveil 또는 인근) 정착 → **인간인 척 키움**
3. 마을 아이들에게 글·약초 가르침 → 마을의 인정 받음
4. 어느 날 **외형(검은 머리 + 뾰족 귀)이 들킴** → **도망자 신세**
5. **Ch.01 학살 시 살해됨**

✅ **양어머니 인간 학자 캐논 완전 정합 (모순 해소 완료)**:
- 양어머니 = 갈색 머리 인간 여인 (외형 명시) — 인간 캐논 정합
- 시간선 모호화: *4년 전* → *지난* (Ch.01 학살 시점 = 1년 6개월 전 정합)
- 양어머니 옆 *검은 머리 엘프 아이* = 세리스 (친부모 마족 = 외형 정합)
- "마족 친부모를 잃은 검은 머리 엘프 아이를 인간인 척 길러온 학자" = 1권 line 385 촌장 회상과 완전 정합

### Vol.2 마족 차별 빌드업 결정적 씨앗

- **양어머니 = 인간** 이지만 *마족 아이 보호*  → 인간 사회의 마족 차별 광기로 양어머니 살해
- 세리스 = 친부모 살해 + 양어머니 살해 둘 다 *인간에 의해* 반복
- Vol.2 학살의 진실 = *나이트 본인 (조종)* → 세리스의 *인간 증오* 정당성 + Vol.2 마족 차별 폭발 진실 폭로 정합

## 4. Q-CORE 2 학자 계보 정합

- Q-CORE 2 = "이름 없는 노인이 가르친 학자 계보" (생활 마법 배포)
- **계보 구성** = 마족 학자 + 인간 학자 둘 다 포함
- 친부모 (마족 학자) + 양어머니 (인간 학자) = 모두 같은 계보
- *마족 = 인간보다 뛰어난 지능* (대표님 박제) → 학자 = 마족 다수
