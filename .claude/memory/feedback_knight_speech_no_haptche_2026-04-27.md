# 나이트 말투 = 합쇼체 절대 X (대표님 critical 정정 지시)

## 0. 대표님 원문 (불변)

### 0.1. 1단계 정정 지시 (합쇼체 폐기)

> *"나이트의 말투가 합쇼체인데 절대 합쇼체 아니야,,,일반적인 말투야....그냥 무덤덤할뿐 너무 단답형도아니고....이건 반드시 수정하고 지나가야된다."*
>
> — 대표님 직접 paste · 2026-04-27 · Q23+Q24 자문 박제 직후

### 0.2. 2단계 정정 지시 (격식 평문체도 폐기) ⭐ 신규

> *"씁 맞다. ..알았다... 이런말투 누가쓰노 ㅡ.ㅡ 좀더 평범하게 안되는가"*
>
> — 대표님 직접 추가 지시 · 2026-04-27 · 1단계 정정 직후

**해석**: 격식 평문체 (~알았다/~맞다/~그렇다/~모른다 등 ~다 종결 단답) = 너무 무뚝뚝·옛스러움 = 일반인이 잘 안 쓰는 말투. 더 평범한 일상 어체로.

---

## 1. 어체 강제 박제 (합쇼체 X + 격식 평문체 X / 일상 어체 O)

### 1.1. 절대 금지 1 — 합쇼체 어미 (1단계 정정)

**나이트 발화에서 일절 사용 금지**:

| 어미 | 예시 (금지) |
|------|------------|
| ~합니다 | "감사합니다" / "그렇게 합니다" |
| ~습니다 | "갔습니다" / "먹었습니다" / "모릅니다" |
| ~입니다 | "예입니다" / "사실입니다" |
| ~십시오 | "가십시오" / "들어주십시오" |
| ~입니까 | "맞습니까" / "그렇습니까" |
| ~합니까 | "어떡합니까" |
| ~지요 (격식) | "그렇지요" |

**판정 기준**: *공무원·하인·격식 사회 인사* 어조 = 합쇼체 = 나이트에 부적합.

### 1.2. 절대 금지 2 — 격식 평문체 ⭐ 2단계 정정 (대표님 신규 지시)

**나이트 발화에서 일절 사용 금지**:

| 어미 | 예시 (금지) | 사유 |
|------|------------|------|
| ~다 종결 단답 | 「알았다.」 / 「맞다.」 / 「그렇다.」 / 「모른다.」 / 「모르겠다.」 / 「진짜다.」 / 「아니다.」 | 너무 무뚝뚝·옛스러움 / 일반인 잘 안 씀 |
| ~다 단정 | 「있다.」 / 「없다.」 / 「한다.」 / 「된다.」 | 격식 평문체 / 부자연 |

**대표님 원문**: *"씁 맞다. ..알았다... 이런말투 누가쓰노 ㅡ.ㅡ 좀더 평범하게"*

**판정 기준**: *근엄한 노인·고풍 사극* 어조 = 격식 평문체 = 나이트에 부적합.

### 1.3. 허용 어체 (일상 어체 — 평범한 사람의 말투)

#### 어른 (모르는 / 어른) 대화 = **해요체** (~해요 / ~예요 / ~네요)

**원칙**: 정중하지만 *격식 X* / 부드러운 일반인 말투.

**예시 (1단계 + 2단계 정정)**:

| 금지 (합쇼체/평문체) | 허용 (해요체) |
|---------------------|--------------|
| 「감사합니다.」 | 「······고마워요.」 |
| 「죄송합니다.」 | 「······미안해요.」 |
| 「그렇습니다.」 / 「그렇다.」 | 「······그래요.」 |
| 「입니까?」 | 「······예요?」 |
| 「알겠습니다.」 / 「알았다.」 | 「······알았어요.」 |
| 「먹지 못합니다.」 | 「······많이 못 먹어요.」 |
| 「가십시오.」 | 「······가요.」 |
| 「뭐라고 하셨습니까?」 | 「······뭐라고 했어요?」 |
| 「맞다.」 | 「······맞아요.」 |
| 「없다.」 | 「······없어요.」 |
| 「있다.」 | 「······있어요.」 |
| 「아니다.」 | 「······아니에요.」 |

#### 동료 (프레이야·록솔·벨릭·안나·미리암·펠란·세리스 등) = **일상 반말**

**원칙**: 동료에게는 *무덤덤 + 일상 반말* (격식 평문체 X / 일상 반말 O).

**예시 (2단계 정정)**:

| 금지 (격식 평문체) | 허용 (일상 반말) |
|-------------------|-----------------|
| 「알았다.」 | 「······알았어.」 |
| 「맞다.」 | 「······맞아.」 |
| 「그렇다.」 | 「······그래.」 |
| 「모른다.」 | 「······몰라.」 |
| 「모르겠다.」 | 「······모르겠어.」 |
| 「있다.」 | 「······있어.」 |
| 「없다.」 | 「······없어.」 |
| 「진짜다.」 | 「······진짜야.」 |
| 「아니다.」 | 「······아니야.」 |

#### 평문체 자체 폐기 ⭐

**기존 박제** (2026-04-27 1단계 직후):
- 단답 시 = 평문체 (~다 / 짧은 단어) ← **폐기**

**신규 박제** (2026-04-27 2단계 직후):
- 단답 시 = **일상 반말 (동료) 또는 해요체 (어른)** 사용
- *근엄한 ~다 종결* 자체 = 본작 나이트에 *부적합*

## 2. 무덤덤 + 너무 단답형 X (균형 박제)

### 2.1. 대표님 지시 두 축

> *"그냥 무덤덤할뿐 너무 단답형도아니고"*

**해석**:
- ✅ **무덤덤** = 감정 미세 표출 / 톤 차분 (보존)
- ❌ **너무 단답형** = "예", "......", "그렇다" 등 *한 음절·한 단어 답*만 반복 (개선)

### 2.2. 무덤덤 + 일반 길이 박제

**나쁜 예** (너무 단답):
> 「예.」
> 「······」
> 「그렇다.」

**좋은 예** (무덤덤 + 일반 길이):
> 「······예. 원래 많이 못 먹어요.」 (단답 + 짧은 부연)
> 「······고마워요. 잘 받았어요.」 (단답 + 인사)
> 「······맞아요. 그런 것 같아요.」 (단답 + 부드러운 부연)

**원칙**:
- 한 발화 = *최소 1 ~ 2 문장* (절대 단답 1 음절만 X)
- 부연은 *짧고 무덤덤* (감정 분출 X)
- "······" 침묵은 *발화 직전 망설임* 표시 (전체 침묵 X)

## 3. 1권 6 챕터 정정 대상 (구체)

### 3.1. 합쇼체 패턴 grep 결과 (2026-04-27)

| 패턴 | 1권 6 챕터 합산 |
|------|----------------|
| ~습니다 | 66건 |
| ~입니다 | 35건 |
| ~합니다 | 17건 |
| ~십시오 | 10건 |
| ~입니까 | 5건 |
| **합계** | **133건** |

이 중 *나이트 발화* 만 정정 대상. NPC (촌장·신관·검문관·점원·의뢰주 등) 합쇼체 = **보존** (그들의 캐릭터).

### 3.2. 정정 작업 순서 (2 단계)

#### 1단계 — 합쇼체 정정 (14건 / commit b5b6500)
1. 챕터별 합쇼체 line grep (`grep -B 3 -A 1 "(습니다|합니다|입니다|십시오|입니까)\."`)
2. 컨텍스트로 *발화자 식별*
3. 나이트 발화만 추출 → list 생성
4. Edit 으로 *합쇼체 → 해요체* 정정
5. 정정 후 grep 재검증 (나이트 발화 합쇼체 = 0건)

#### 2단계 — 격식 평문체 정정 (34건) ⭐
1. 챕터별 평문체 line grep (`grep "^「(알았다|그렇다|맞다|모른다|모르겠다|진짜다|아니다|있다|없다|한다)\.?」"`)
2. 컨텍스트로 *발화자 식별* (벨릭 발화는 보존 — 캐릭터 어체)
3. 나이트 발화만 추출 → list 생성
4. Edit 으로 *평문체 → 일상 반말 (동료) / 해요체 (어른)* 정정
5. 정정 후 grep 재검증 (나이트 발화 격식 평문체 = 0건)

**누적 정정 = 48건** (1단계 14 + 2단계 34)

#### 챕터별 정정 분포

| 챕터 | 1단계 (합쇼체) | 2단계 (평문체) | 합계 |
|------|---------------|---------------|------|
| 나이트1 | 8 | 0 | 8 |
| 나이트2 | 3 | 13 | 16 |
| 나이트3 | 0 | 7 | 7 |
| 나이트4 | 1 | 5 | 6 |
| 나이트5 | 1 | 1 | 2 |
| 나이트6 | 1 | 8 | 9 |
| **합계** | **14** | **34** | **48** |

### 3.3. 다른 인물 합쇼체 보존

| 인물 | 어체 | 비고 |
|------|------|------|
| 촌장 (1권 § 七) | 합쇼체 | *손님 대우* + 노인 격식 — 보존 |
| 신관 (1권 § 八·九) | 합쇼체 | 격식 의례적 — 보존 |
| 검문관 (1권 § 十二·1권 결말 광장) | 합쇼체 | 공무원 격식 — 보존 |
| 점원 / 길드 직원 (테사 등) | 합쇼체 | 사무 격식 — 보존 |
| 막다 (식당 주인) | 합쇼체 ~ 해요체 | 손님 대접 — 보존 |
| Bruiden / Cailean / Vaern 가문 | 합쇼체 | 귀족 격식 — 보존 |

→ NPC 합쇼체 = 1권 캐릭터 톤 핵심 / 보존 절대.

## 4. Vol.2~Vol.N 강제 적용

### 4.1. 매 본 집필 시 강제

- 나이트 발화 → 합쇼체 어미 grep 검증 = 0건 강제
- 동료 발화 = 반말 보존
- NPC 발화 = 캐릭터별 어체 (합쇼체 가능)

### 4.2. Vol.N 결말 1인칭 영구 소거 정합

`project_finale_canon_succession_cycle_2026-04-27.md` § 2.3-bis.2 = *1인칭 시점 영구 소거*.

→ 결말 시퀀스에서 나이트 발화 자체가 *극히 제한* 됨.
→ 그 제한된 발화도 **합쇼체 X** 강제.
→ 마왕 (100년 후) 의 단발 발화 = *마왕 = 평문체 짧은 한 마디* 만 (해요체조차 X — 인간성 박탈).

## 5. 작법 강제 정합 (Q23+Q24 박제)

### 5.1. Q23 처방 1 (단답 앞뒤 지문 + 내면 독백) 과 정합

- 합쇼체 → 해요체 정정 시 *단답 = 한 박자 부연* 도 함께 추가
- 예: 합쇼체 *"감사합니다"* → 해요체 *"고마워요"* + *"잘 받았어요"* (지문: 시선을 식탁 끝에 잠시 두고)

### 5.2. Q24 어리버리 유머 정합

- 핀트 엇갈림 / 문자 그대로 / 동문서답 등 5 패턴 = *단답형* 또는 *짧은 부연 형태*
- 어체 = 해요체 / 평문체 (합쇼체 X)
- 예: 「······힘들지 않았어?」 → 「······계속 걸어왔더니 다리가 좀 아파요.」 (해요체 + 핀트 엇갈림)
