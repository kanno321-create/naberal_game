# 나이트 말투 = 합쇼체 절대 X (대표님 critical 정정 지시)

> *"나이트의 말투가 합쇼체인데 절대 합쇼체 아니야,,,일반적인 말투야....그냥 무덤덤할뿐 너무 단답형도아니고....이건 반드시 수정하고 지나가야된다."*
>
> — 대표님 직접 paste · 2026-04-27 · Q23+Q24 자문 박제 직후

## 1. 어체 강제 박제 (합쇼체 X / 일반 말투 O)

### 1.1. 절대 금지 (합쇼체 어미)

**나이트 발화에서 일절 사용 금지**:

| 어미 | 예시 (금지) |
|------|------------|
| ~합니다 | "감사합니다" / "그렇게 합니다" |
| ~습니다 | "갔습니다" / "먹었습니다" / "모릅니다" |
| ~입니다 | "예입니다" / "사실입니다" |
| ~십시오 | "가십시오" / "들어주십시오" |
| ~입니까 | "맞습니까" / "그렇습니까" |
| ~합니까 | "어떡합니까" |
| ~지요 (격식) | "그렇지요" |

**판정 기준**: *공무원·하인·격식 사회 인사* 어조 = 합쇼체 = 나이트에 부적합.

### 1.2. 허용 어체 (일반 말투)

#### 어른 (모르는 / 어른) 대화 = **해요체** (~해요 / ~예요 / ~네요)

**원칙**: 정중하지만 *격식 X* / 부드러운 일반인 말투.

**예시**:

| 합쇼체 (금지) | 해요체 (허용) |
|---------------|--------------|
| 「감사합니다.」 | 「······고마워요.」 |
| 「죄송합니다.」 | 「······미안해요.」 |
| 「그렇습니다.」 | 「······그래요.」 |
| 「입니까?」 | 「······예요?」 |
| 「알겠습니다.」 | 「······알았어요.」 |
| 「먹지 못합니다.」 | 「······많이 못 먹어요.」 |
| 「가십시오.」 | 「······가요.」 |
| 「뭐라고 하셨습니까?」 | 「······뭐라고 했어요?」 |

#### 단답 시 = **평문체** (~다 / 짧은 단어)

**원칙**: 무덤덤·기계적 단답 시 (특히 자기 자신 / 자명한 사실).

**예시**:
- 「······맞다.」
- 「······모른다.」
- 「알았다.」
- 「······예.」 (단답 인정)
- 「······그래.」 (단답 동의)
- 「······」 (침묵)

**주의**: 단답체 너무 자주 X (대표님 지시 *"너무 단답형도 아님"*).

#### 동료 (프레이야·록솔·벨릭·안나·미리암 등) = **반말** (기존 유지)

**원칙**: 동료에게는 무덤덤 반말 (변경 X).

**예시**:
- 「가자.」
- 「너 어디 있었어.」
- 「······그래.」
- 「······모르겠어.」

## 2. 무덤덤 + 너무 단답형 X (균형 박제)

### 2.1. 대표님 지시 두 축

> *"그냥 무덤덤할뿐 너무 단답형도아니고"*

**해석**:
- ✅ **무덤덤** = 감정 미세 표출 / 톤 차분 (보존)
- ❌ **너무 단답형** = "예", "......", "그렇다" 등 *한 음절·한 단어 답*만 반복 (개선)

### 2.2. 무덤덤 + 일반 길이 박제

**나쁜 예** (너무 단답):
> 「예.」
> 「······」
> 「그렇다.」

**좋은 예** (무덤덤 + 일반 길이):
> 「······예. 원래 많이 못 먹어요.」 (단답 + 짧은 부연)
> 「······고마워요. 잘 받았어요.」 (단답 + 인사)
> 「······맞아요. 그런 것 같아요.」 (단답 + 부드러운 부연)

**원칙**:
- 한 발화 = *최소 1 ~ 2 문장* (절대 단답 1 음절만 X)
- 부연은 *짧고 무덤덤* (감정 분출 X)
- "······" 침묵은 *발화 직전 망설임* 표시 (전체 침묵 X)

## 3. 1권 6 챕터 정정 대상 (구체)

### 3.1. 합쇼체 패턴 grep 결과 (2026-04-27)

| 패턴 | 1권 6 챕터 합산 |
|------|----------------|
| ~습니다 | 66건 |
| ~입니다 | 35건 |
| ~합니다 | 17건 |
| ~십시오 | 10건 |
| ~입니까 | 5건 |
| **합계** | **133건** |

이 중 *나이트 발화* 만 정정 대상. NPC (촌장·신관·검문관·점원·의뢰주 등) 합쇼체 = **보존** (그들의 캐릭터).

### 3.2. 정정 작업 순서

1. 챕터별 합쇼체 line grep (`grep -B 3 -A 1 "(습니다|합니다|입니다|십시오|입니까)\."`)
2. 컨텍스트로 *발화자 식별* (앞뒤 지문에서 누가 말하는지)
3. 나이트 발화만 추출 → list 생성
4. Edit 으로 *합쇼체 → 해요체/평문체* 정정
5. 정정 후 grep 재검증 (나이트 발화 합쇼체 = 0건)

### 3.3. 다른 인물 합쇼체 보존

| 인물 | 어체 | 비고 |
|------|------|------|
| 촌장 (1권 § 七) | 합쇼체 | *손님 대우* + 노인 격식 — 보존 |
| 신관 (1권 § 八·九) | 합쇼체 | 격식 의례적 — 보존 |
| 검문관 (1권 § 十二·1권 결말 광장) | 합쇼체 | 공무원 격식 — 보존 |
| 점원 / 길드 직원 (테사 등) | 합쇼체 | 사무 격식 — 보존 |
| 막다 (식당 주인) | 합쇼체 ~ 해요체 | 손님 대접 — 보존 |
| Bruiden / Cailean / Vaern 가문 | 합쇼체 | 귀족 격식 — 보존 |

→ NPC 합쇼체 = 1권 캐릭터 톤 핵심 / 보존 절대.

## 4. Vol.2~Vol.N 강제 적용

### 4.1. 매 본 집필 시 강제

- 나이트 발화 → 합쇼체 어미 grep 검증 = 0건 강제
- 동료 발화 = 반말 보존
- NPC 발화 = 캐릭터별 어체 (합쇼체 가능)

### 4.2. Vol.N 결말 1인칭 영구 소거 정합

`project_finale_canon_succession_cycle_2026-04-27.md` § 2.3-bis.2 = *1인칭 시점 영구 소거*.

→ 결말 시퀀스에서 나이트 발화 자체가 *극히 제한* 됨.
→ 그 제한된 발화도 **합쇼체 X** 강제.
→ 마왕 (100년 후) 의 단발 발화 = *마왕 = 평문체 짧은 한 마디* 만 (해요체조차 X — 인간성 박탈).

## 5. 작법 강제 정합 (Q23+Q24 박제)

### 5.1. Q23 처방 1 (단답 앞뒤 지문 + 내면 독백) 과 정합

- 합쇼체 → 해요체 정정 시 *단답 = 한 박자 부연* 도 함께 추가
- 예: 합쇼체 *"감사합니다"* → 해요체 *"고마워요"* + *"잘 받았어요"* (지문: 시선을 식탁 끝에 잠시 두고)

### 5.2. Q24 어리버리 유머 정합

- 핀트 엇갈림 / 문자 그대로 / 동문서답 등 5 패턴 = *단답형* 또는 *짧은 부연 형태*
- 어체 = 해요체 / 평문체 (합쇼체 X)
- 예: 「······힘들지 않았어?」 → 「······계속 걸어왔더니 다리가 좀 아파요.」 (해요체 + 핀트 엇갈림)
