## 욜로모드 자체 해석 (세션 #5)

**채택 해석**: 세션 #5 설정 우선. 세션 #3 outline 은 **일부 수정**.

1. **주인공 = Ashenveil 출신 소년**. 이름 · 성별 · 나이 대표님 결정 대기 (공란 유지).
2. **"나이트" 는 호칭**. 각성 후 Act 2 부터 얻는 타칭·자칭. Act 1 에서는 그냥 소년.
3. **프롤로그 재해석**: 우주·수호자 설정은 **세계관 상위 레이어** 로 유지. 균형 수호자가 마왕군 첫 도래를 감시하며 개입하려 하는 장면으로 재작성 가능 (주인공 = 나이트 등식 폐기).
4. **Ch.01 재설계**:
   - 구 Ch.01 (숲 첫 전투) → Ch.02 후반 또는 Ch.03 배치
   - 새 Ch.01 = Ashenveil 평화 일상 → 서쪽 불길한 징조 → 마왕군 첫 도래 → 각성 암시 → 탈출 결심
5. **시점 전략**:
   - Act 1 전편: **1인칭 주인공** (LN 가독성 · 대표님 선호)
   - 프롤로그·각 챕터 시작 정경: 3인칭 전지 (짧게)
   - Act 2·3 동료 심리: 해당 인물 1인칭 단일 장면 허용
6. **톤**:
   - 세션 #3 "오버로드式 절제" → Act 1 에서는 **소년의 일상 감각** · 절제보다 생동감
   - 각성 후 (Act 2+) 점차 절제 톤으로 이행
