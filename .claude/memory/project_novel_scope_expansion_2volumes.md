# Episode 1 스코프 확장 · 2권 20~23만자

## 확정 사실 (대표님 2026-04-24 직접 지시)

### 분량
- **Episode 1 = 소설판 2권 분량**
- **총 20~23만자** (기존 outline.md 30~40만자 목표 대비 축소 조정 · 실전 집필 가능 범위)
- 권당 10~11.5만자 배분

### 확장 방향
- 기존 페이즈 1~5 골격 유지
- 각 페이즈 사이에 **새 방문지·던전·지방·지형·유적** 삽입
- 대표님 원문: *"추가로 방문하는 곳을 만들어야겠다 던전도 추가하고, 이런저런 지방이나 지형에도 방문하고 유적도"*

### Episode 2 분리 원칙 (상호 배타)
- Episode 2 카라 시퀄 = 본편 미방문 지역 9 왕국 + 2 자치구 + 5 신규 영역 활용
- Episode 1 확장은 **Ilaris·Elucia·Karzor 3 왕국 내부 심화** + Silvan Forest + 대륙 간 지형 + 4차 이하 봉인지 유적
- 두 에피소드의 지리 풀이 겹치지 않게 관리
