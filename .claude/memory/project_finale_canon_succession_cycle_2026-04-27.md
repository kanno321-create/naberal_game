# 시리즈 최종 결말 Canon — 순환의 호선

## 0. 결론 (대표님 직접 박제)

> **1. 마왕화 직후 = 신을 본보기로 찢어죽임 (카미코로시 1단계 실행) + 언데드·악마 부려 추종자·시스템 청소 (2단계) + 차별 반대자 1명을 새 신으로 임명 (시작지점 촌장 등) + 우주적 존재의 무기 전달·권능 하사·자격 부여로 신이 됨.**
>
> **2. 100년 이상 경과 → 인간+타종족 협동 문명 재건 → 모험가 길드 다종족 어우러짐 → 그러나 마왕(나이트)는 타락 — 본분 잊고 멍한 눈으로 100년 이상 반복, 몬스터·악마 계속 풀어 인간·타종족 모두 공격.**
>
> **3. 마지막 한 컷 — 모험가 길드에서 책을 읽던 마족 여성: *세리스의 생김새 + 나이트의 흰 머리*. 옆에서 들리는 이야기 = *마왕을 쓰러트릴 용사는 없는가*. = 다음 사이클 후크.**

이는 시리즈 *전체 호선의 종결 + 재진입 가능성* 박제. 1권~Vol.N 의 모든 회수 사슬이 마지막 한 컷으로 *순환* 으로 다시 열림.

## 1. 시퀀스 3 막 (대표님 박제 그대로)

### 막 1. 각성 직후 — 카미코로시 + 새 신 임명

#### 1.1. 신 처단 (카미코로시 1단계 실행)

> "마신/마왕으로 각성된 나이트가 신을 아주 본보기로 찢어죽이는 거지."

**행위**: 신을 *아주 본보기로 찢어죽임* — 단순 죽임 X / *공개 처단·시각적 박제* 형태

**의미**:
- 카미코로시 (神殺し) 1단계 = 시스템 정점 제거
- *본보기* = 다른 자들에게 *시스템 외부 존재가 신을 죽일 수 있음* 을 증명
- = 시스템의 *자가 정당화 불가능* 선언

**작법**:
- ✅ Show: 신의 시각적 처단 (찢어짐) + 권능의 박살 + 추종자들이 그 광경을 *직접 봄*
- ❌ 신의 형태 = 추상적·초자연적 X / *눈에 보이는 처단 가능한 형태* (육신·우상·왕좌 등)
- ❌ 단번의 처단 X / *느린·잔혹한 처단* (본보기 = 시간을 끌며 처단)

#### 1.2. 추종자·시스템 청소 (카미코로시 2단계 실행)

**대상 (대표님 박제 그대로)**:
- 교회 (성직자·추기경·신관)
- 추종자 (신실한 자들)
- *빌붙어 먹고 살던* 각종 귀족
- 평민 할 것 없이 (시스템에 협력하던 자들)

**수단**: 수많은 *언데드와 악마* 를 부림

**메커니즘**:
- 마왕의 권능 = 언데드·악마 소환·통제
- 시스템 측 = 수십만 / 마왕 측 = 무한대의 언데드·악마
- 결과 = 시스템 측 *완전 청소*

**의미**:
- 카미코로시 2단계 = 시스템 구조 청소
- 동시 = 카미코로시 3단계 *부분 실행* (시스템 충성 종족·평민 청소)
- → 종족 청소는 *완전 멸절* 이 아니라 *시스템 협력자만* 선별 청소

#### 1.3. 청소의 *예외* — 차별 반대자 보호

**박제 (대표님 직접)**:
> "타종족 차별을 반대하고 그들을 열심히 돌봐준 이들"

**의미**:
- 종족 청소 ≠ 인간 종족 *전체* 멸절
- *차별 반대자 = 보호*
- = 새 시스템의 *기반 인구* 가 됨
- = 카미코로시 3단계의 *진정한 본질*: 시스템에 협력한 자만 청소 / 시스템에 저항한 자는 보호

**작법**:
- ✅ 보호받는 자 = 1권부터 등장한 *기억 있는 자들* (회수 효과)
- ✅ 보호의 자세 = 마왕의 자비가 아니라 *예전 결단의 결과* (언데드·악마가 그들을 *알아봄* — 마왕이 미리 표시)

#### 1.4. 새로운 신 임명 (대표님 박제 핵심)

> "타종족 차별을 반대하고 그들을 열심히 돌봐준 이들 중 1명 (예를 들어 시작지점 촌장 등) 을 새로운 신으로 만들고."

**임명 후보** (대표님 박제 예시):
- **1순위**: Ch.01 *시작지점 촌장* (검은 머리 뾰족귀 아이를 보호하다 학살당한 마을의 촌장)
  - 1권 첫 등장 인물 → 시리즈 마지막 새 신 = *알파와 오메가 회수*
- **다른 후보**: 마리에 (약사) / 막다 (여관 주인) / 4년 전 학살 마을 어느 이름 없는 노인 / 기타

**임명 메커니즘** (대표님 박제 그대로):

| 단계 | 행위 | 의미 |
|------|------|------|
| 1 | **우주적 존재가 준 무기를 전달** | 우주적 존재 = 1권의 그 존재 (Vol.2 메인 플롯의 그 존재) / 무기 = 신적 권능의 매개체 |
| 2 | **권능을 하사받음** | 무기를 통한 권능 전이 |
| 3 | **권능을 사용할 자격을 얻음** | 자격 = *차별 반대자였다* 는 과거의 결단 |
| 4 | **신이 됨** | 권능 + 자격 = 신적 존재로 승격 |

**의미**:
- *신* 은 신실함이 아니라 *결단의 누적* 으로 만들어짐
- 새 신 = 과거 시스템의 *반대편* 에서 살아온 자
- = 5기둥 ② 신 대항의 궁극 형태 = *새 신을 누가 만드는가* 의 답: 신 대항을 한 자

**우주적 존재의 역할**:
- 1권부터 박제된 우주적 존재 (마왕이 아닌 다른 존재)
- *균형의 수호* 라는 본래 목적
- 마왕화 = 균형의 *시정* 행위
- 새 신 임명 = 균형의 *재구축*
- = 우주적 존재 = 시스템의 *외부 심판자*

**작법**:
- ✅ 임명 장면 = 1권 첫 등장 인물의 회상·재등장과 동시
- ✅ 무기 = 1권부터 박제된 시각적 모티프 (예: 손가락 마법진 / 흰 머리카락 한 가닥 / 부츠 흙 / 종소리 등) 와 연결
- ❌ 임명 = 단순 의식 X / *우주적 존재의 직접 출현* 또는 *마왕의 마지막 자비* 형태
- ✅ 새 신의 첫 권능 사용 = *과거 자기가 보호하지 못한 자들* 의 영혼 회복 또는 추모 (감정적 회수)

### 막 2. 100년 이상 경과 — 문명 재건 + 마왕의 타락

#### 2.1. 인간 + 타종족 협동 문명 재건

**시간 경과**: 수십 년 ~ 100년 이상

**재건의 모습**:
- 인간과 타종족이 *협동* 하여 문명을 재건
- 새 신 (시작지점 촌장 등) = 새 시스템의 정점
- 마족·엘프·오크·인간 등 모든 종족 = 차별 없이 공존
- = 1권 § 七 (촌장의 설명) 에서 박제된 *3년 전 신의 계시* 이전 상태로의 회복 + *진보*

**작법**:
- ✅ 재건 = 1권 박제 인물들의 후손이 활약
- ✅ 마리에 (약사) 의 후손이 의술을 잇거나 / 막다 (여관) 의 후손이 여관을 운영하는 등의 *작은 회수*
- ❌ 재건 = 완벽한 유토피아 X / *작은 갈등은 여전히 존재* 하지만 *해결 가능한* 형태

#### 2.2. 모험가 길드 — 다종족 어우러짐

**박제 (대표님 그대로)**:
> "수많은 종족의 모험가로 가득한 모험가 길드에서 여러 종족들이 이야기꽃을 피우고."

**모습**:
- 1권 § 五 *파르마 모험가 길드* 와 같은 형태
- 그러나 — 인간만이 아니라 *모든 종족* 이 등록
- 마족·엘프·오크·인간 등이 *같은 식탁* 에 앉아 *이야기꽃* 을 피움
- = 1권의 모험가 길드의 *진보된 모습*

**작법**:
- ✅ 길드 묘사 = 1권 모험가 길드와 *같은 구조* (게시판 / 접수대 / 안마당 / 식당)
- ✅ 그러나 분위기는 다름 — *환한 햇빛 + 다양한 옷색·향·언어* (1권의 회색·피냄새와 대비)
- ✅ 1권 박제된 모험가 길드 직원 (안경·깃펜) 의 후손 등장 가능

#### 2.3. 마왕 (나이트) 의 타락 — 멍한 눈의 반복

**박제 (대표님 그대로)**:
> "나이트 역시 수십 년 백년 이상 마왕/마신으로 지내오며 조금씩 타락하고 있었고. 본분을 잊은 채 멍한 눈으로 백년 이상 해오던 걸 반복만 할 뿐이다. 몬스터 및 악마들을 계속 풀어서 인간 및 타종족 모두를 공격하고 있었던 것."

**타락의 메커니즘**:
- 마왕화 = 인간성 외부 도약 = *시간 감각의 마비*
- 100년 이상 = *시간 자체가 의미 없어짐*
- 본분 (균형 수호 / 시스템 처단) → *잊음*
- 멍한 눈 = 1권 Ch.04 § 二 *Cailean 셈이 끝난 자의 눈* 과 같은 종류

**행동**:
- 몬스터·악마를 *계속* 풀어 *인간·타종족 모두* 를 공격
- = 본래 카미코로시 대상이 아니던 *차별 반대자의 후손들* 도 공격
- = 마왕 자신이 *과거 자기가 만든 시스템의 적* 이 됨

**아이러니**:
- 1권 § 七 신관의 *해야 할 일을 해내는 자의 표정* 과 같은 종류
- 마왕 (나이트) = 100년 후의 *그 신관* 과 같은 자세
- = *시스템에 길든 자* 의 형태가 됨
- = 시스템 외부 존재가 *새 시스템의 일부* 가 됨

**작법**:
- ❌ 타락 = 분노·광기 X / *무감각·반복· 의식 없음*
- ❌ 마왕의 자각 X / *자기가 무엇을 하는지조차 모름*
- ✅ Show: 마왕의 *멍한 눈* / *반복적 동작* / *시간 감각 마비* 의 묘사
- ✅ 5기둥 ① 절망 하강의 궁극: 정의의 영웅도 시간 앞에서 부패

### 2.3-bis. Q21 자문 작법 강제 박제 (다크판타지 수석 편집장 시점) ⭐⭐⭐

> **NotebookLM Q21 자문** (`reference_nlm_writing_advice_2026-04-27.md`) — *"한국 웹소설 역사에 남을 압도적 결말"* 평가의 작법 강제 5건. 결말 시퀀스 진입 순간부터 절대 적용 (이탈 금지).

#### 2.3-bis.1. 1권 § 七 신관 문장 *Anaphora Copy&Paste* ⭐⭐ (본작 문학적 정점)

**박제 강제**:

100년 후 마왕 묘사에 *1권 § 七 (나이트1.md line 229) 신관 문장* 을 **토씨 하나 틀리지 않게** 사용:

> **원문 (1권 § 七 신관 묘사 — Copy&Paste 대상)**:
>
> 「신관의 미소는 기쁨이 아니었다. 만족도 아니었다. 해야 할 일을 해내고 있는 자의 표정. 집 안에서 하루 일과 중 하나를 담담히 정리하고 있는 표정.」

**적용 위치**: 100년 후 마왕 (나이트) 의 *반복적 처형·청소 행위* 묘사 시 — 외부 시점 narrator 가 마왕의 표정을 묘사하는 자리.

**박제 방식**:
- 주어만 *"신관의 미소"* → *"마왕의 미소"* 또는 *"마왕의 표정"* 으로 교체
- 그 외 문장은 **토씨 하나 틀리지 않음** (자문 강제)
- 이탤릭·강조 표시도 1권과 동일 형식

**적용 예시**:
> 마왕의 표정은 기쁨이 아니었다. 만족도 아니었다. 해야 할 일을 해내고 있는 자의 표정. 집 안에서 하루 일과 중 하나를 담담히 정리하고 있는 표정.

**효과**:
- 독자가 *패턴 반복 발견* → 나이트가 *그토록 혐오한 시스템의 톱니바퀴(신관)와 완벽히 동기화* 깨달음
- = **압도적 허무주의적 카타르시스**
- = 본작 문학적 완성도의 *정점*

**금지**:
- ❌ 변형·축소·확장 금지 (anaphora = 정확한 반복)
- ❌ 다른 어휘로 대체 금지 (예: "해내는 자" 와 "해내고 있는 자" 는 다른 문장 — 원문은 *"해내고 있는 자"*)
- ❌ 마왕의 *내면 독백* 으로 박제 금지 (외부 시점만 — 2.3-bis.2 참조)

#### 2.3-bis.2. 1인칭 시점 *영구 소거* ⭐⭐⭐ (가장 잔혹한 작법)

**박제 강제**:

결말 시퀀스 *진입 순간* 부터 나이트의 *내면 독백 (1인칭) 자체* 를 서사에서 **영구적으로 소거**.

**구체 강제**:
- ❌ *"~라고 생각했다"* — 금지
- ❌ *"내면 자아 속삭임"* — 금지
- ❌ *"머리속에 ~가 떠올랐다"* — 금지
- ❌ *"외면 / 내면 두 자아"* — 금지 (Q-CORE 4 융화 포함)
- ❌ 1인칭 *나는 / 내가 / 내 ~* 어미 — 금지

**허용 시점**:
- ✅ *외부 관찰자 시점*: 새 신 (시작지점 촌장 등) 의 시점
- ✅ *외부 관찰자 시점*: 마족 여성 (마지막 한 컷) 의 시점
- ✅ *전지적 작가 시점*: narrator 의 건조한 관찰 서술
- ✅ *후대 모험가 시점*: 마왕 재앙을 안줏거리로 삼는 잡담

**효과**:
- 시점의 박탈 = 마왕이 *인간성·자아 잃고 서사 밖으로 추락* 한 잔혹한 선고
- = 나이트의 *서사 밖 추락 자체가 서사의 한 형태*
- = 독자가 마왕을 *주인공* 이 아닌 *재앙·자연재해* 로 인식하기 시작

**작법 강제**:
- 결말 시퀀스 *진입점* 명확화 — *각성 클라이맥스 § 1.8 (이마 키스 + 심연 사슬 + 신전 진입)* 직후부터 1인칭 영구 소거 발효
- 결말 진입 전 (각성 클라이맥스 진행 중) = 1인칭 유지 가능 (마지막 1인칭의 무게)
- 결말 진입 후 = *마왕의 머릿속* 묘사 *완전 부재*

**5기둥 정합**:
- 5기둥 ① 절망 하강 = 시점 자체의 박탈
- 5기둥 ② 신 대항 = 시스템 외부였던 자가 시점도 잃음
- 5기둥 ③ 사망 = 인간성 (시점) 의 사망

#### 2.3-bis.3. 100년 시간 압축 = *시적 압축 + 외부 시점* (몽타주 X)

**박제 강제**:

100년 = 라이트노벨 호흡에서 *물리적 요약 (몽타주) X* / 단 한 번의 *시점 전환 + 공간 도약* 으로 박제.

**금지 (몽타주식)**:
- ❌ "그리고 100년이 지났다."
- ❌ 시간 경과를 직접 서술하는 narrator voice
- ❌ 마왕 시점에서의 시간 흐름 회상
- ❌ 연도·날짜·계절 언급으로 시간 경과 표시

**허용 (시적 압축)**:
- ✅ **장면 완전 전환** → 후대 모험가 시점 직접 진입
- ✅ 후대 모험가들이 *마왕 재앙을 안줏거리로 삼는 묘사*
- ✅ 전설의 영웅 (나이트) = 한낱 *퇴치할 자연재해* 로 불림
- ✅ 후대 모험가의 *지친 한숨* + *씁쓸한 잡담*
- ✅ 후대 모험가의 회상에서 *과거의 영웅* 이 *오래된 전설* 로 자리잡음

**효과**:
- 100년 시간 무게·부식 *단번에 체감*
- 독자가 *시간이 얼마나 지났는지* 정확히 모르게 만듦 = 시간의 *무한성* 박제
- = 시간의 *부패력* 이 시각적으로 압도

**적용 위치**:
- § 막 2 진입점 = 1인칭 소거 직후 = 후대 모험가 시점 직접 진입
- § 막 2 종료점 = 마지막 한 컷 (모험가 길드 마족 여성) 진입

#### 2.3-bis.4. 흰 머리카락 *흙처럼 바스라짐*

**박제 강제**:

마왕의 *외투 안주머니* 에 보존된 *세리스의 흰 머리카락 한 가닥* — 100년 후 묘사 시:

**구체 박제**:
> 마왕은 옥좌에 앉아 멍한 눈을 허공에 두고 있었다. 그의 손이 습관처럼 외투 안주머니를 더듬었다.
>
> 그곳에 보존되어 있던 세리스의 흰 머리카락 한 가닥이 — 손가락 끝에 닿자 **흙처럼 바스라져 손가락 틈으로 빠져나갔다**.
>
> 자신의 가장 소중한 징표가 부서지는 자리에서도, 마왕의 시선은 허공에 그대로였다. 감정 동요는 *전혀 없었다*.

**효과**:
- *극도의 비극성 + 소름* 동시 박제
- 마왕이 가장 소중한 것 (세리스의 마지막 흔적) 에도 *반응 없음* = 인간성 사망의 시각적 증거
- 1권 § 八 (또는 그 이후) 박제된 *흰 머리카락 보존* 모티프의 *최종 회수*

**금지**:
- ❌ 마왕의 *후회* 묘사 X
- ❌ 마왕의 *인지* 묘사 X (손가락 틈으로 빠져나감을 *인지하지도 못함*)
- ❌ 흰 머리카락의 *극적 빛남* X / *흙처럼 바스라짐* (조용한 부서짐)
- ❌ 1인칭 시점 사용 X (외부 narrator 만)

**작법**:
- ✅ 동작은 *습관* — 의식적 행위 X
- ✅ 부서짐은 *자연스러운* 진행 — 누가 부순 게 아님
- ✅ 마왕의 시선·표정 = 변화 *전혀 없음*
- ✅ Q22 마지막 한 컷 박제 (마족 여성의 *흰 머리카락 어깨로 흘러내림*) 와 *대비 구도*

#### 2.3-bis.5. 레퍼런스 — *고장 난 시계침* 의 자세

**박제 (자문 인용)**:

본작 마왕 = 다음 작품의 마왕상 *반대*:
- ❌ **분노·광기 휩싸인 괴물** X
- ✅ **망각 속에서 영원히 같은 자리만 맴도는 *고장 난 시계침***

**레퍼런스 정합**:

| 작품 | 인물 | 본작 마왕과의 일치점 |
|------|------|---------------------|
| **보석의 나라** | 포스포필라이트 | 1만 년 홀로 견디며 자아 잃고 신적 존재 부식 → *감정 결여 + 멍한 반복* |
| **베르세르크** | 그리피스 (페무토) | 인간 시절 감정 상실 + *무감각한 절대 고요* |

**작법 강제**:
- 마왕의 *반복 행위* = 시계침의 무의식적 회전과 같은 결
- 마왕의 *감정 결여* = 포스포필라이트의 *부식된 자아*
- 마왕의 *무감각* = 페무토의 *절대 고요*
- 마왕의 *멍한 눈* = 다크판타지 *부식의 정점* 시각화

**대비 (1권 나이트와의 진화)**:

| 측면 | 1권 나이트 | 100년 후 마왕 |
|------|-----------|--------------|
| 시점 | 1인칭 (외면+내면 자아) | **시점 박탈** |
| 표정 | 백지 + 점진 융화 | **신관과 동일 anaphora 표정** |
| 행위 | 의식적 결단 | **습관적 반복** |
| 감정 | 미세한 누수 | **완전 부재** |
| 정체 | 균형의 수호자 | **고장 난 시계침** |

→ 1권 → 결말 = *시간의 부식이 영웅을 시스템과 동일자로 만듦* 의 박제

#### 2.3-bis.6. 5기둥 톤 ② 강화 — *신 대항조차 결국 신이 됨*

Q21 작법 강제는 5기둥 ② (신 대항 / 배신된 신) 의 *궁극 형태* 박제:

- 1권: 신을 의심하는 자
- Vol.2~Vol.N: 신을 부정하는 자
- Vol.N 클라이맥스: 신을 죽이는 자
- **결말 100년 후**: **신과 같은 자세를 가진 자**
- = *신 대항이 새 신을 만들고, 그 신 대항자조차 부패해 새 신과 같이 됨*
- = 시스템의 *무한 자가 재생산*

**독자에게 주는 효과**:
- *모든 저항은 결국 시스템에 흡수된다* 의 절망적 메시지
- 그러나 *마지막 한 컷* (마족 여성) 에서 *희망의 사이클* 도 동시 박제
- = 다크판타지 *비극의 영원성 + 희망의 영원성* 의 균형

### 막 3. 마지막 한 컷 — 다음 사이클 후크

#### 3.1. 모험가 길드의 마지막 장면

**박제 (대표님 그대로)**:
> "그중 어떤 이들의 이야기를 그 옆에서 책을 읽던 마족 여성이 듣게 되는데, 최근 마신/마왕의 폐단과, 무분별한 악마와 언데드를 풀음으로서 피해가 상당하니 그를 쓰러트릴 용사는 없는 건가 과연…… 이런 이야기를 듣게 되고, 끝맺음 하면 된다."

**장면 구조**:

| 요소 | 박제 |
|------|------|
| 위치 | 모험가 길드 (1권의 그 길드와 같은 구조 / 또는 다른 도시의 다종족 길드) |
| 분위기 | 환한 낮 / 다양한 종족 어우러짐 / 이야기꽃 |
| 주변 인물 | 모험가들 (다종족) — 마왕 폐단을 이야기 |
| **중심 인물** | **마족 여성 — 옆에서 책을 읽음** |
| 이야기 내용 | "최근 마왕의 폐단" / "무분별한 언데드·악마" / "피해가 상당" / **"그를 쓰러트릴 용사는 없는가"** |
| 마족 여성의 반응 | (대표님 박제 X — 작가 결정 영역) |

#### 3.2. 마족 여성의 정체 — 나이트 + 세리스의 *마력 융합 딸* (확정)

**박제 (대표님 직접 — 2026-04-27 추가 확정)**:
> "세리스와 나이트는 인간이 아님. 마족은 인간처럼 번식 안 하고 마력을 통해서 그들을 닮은 자를 자식으로 만들게 되고, 나이트 또한 인간이 아니므로, 일반적인 번식이 안 됨. 그러므로 나이트의 강력한 마력과, 세리스의 그 마력을 받아들임을 암시하는 장면을 넣어주고. 그 결과물이 새로운 시대의 용사다. 한마디로 직접 생식활동을 해서 낳은 자식이 아닌, 나이트의 마력, 세리스의 마력이 융합되어 만들어진 존재로 마족에선 이를 자식으로 여긴다. 한마디로 나이트와 세리스의 딸이다."

**시각적 박제**:
- 생김새 = *세리스* (마족 / 뾰족 귀 / 어두운 눈 / 마법사적 분위기)
- 머리색 = *나이트의 흰 머리*
- = 두 부모의 시각적 융합 = *마력 융합의 결과물*

**정체 확정**: **나이트와 세리스의 *딸* (마력 융합 자식)**

**메커니즘**:
- 나이트 = 인간 아님 (균형의 수호자 / 우주적 존재의 분신)
- 세리스 = 인간 아님 (마족)
- 두 종족 모두 = *생식 번식* X
- 마족 종족의 자손 = *마력 융합* 으로 만들어짐
- 결과물 = 두 마력의 *닮은 자* = 자식

**작법 강제 (이전 박제와의 변경)**:
- ❌ 정체 *명시* X → **부분적 명시 OK** (마지막 한 컷의 *암시* 형태로 박제)
- ✅ 시각적 디테일 = 두 부모의 융합 (세리스 생김새 + 나이트 흰 머리) 으로 *유전적 자식* 임을 시각적 박제
- ✅ 마지막 박자 = *책 그대로 읽음* + *흰 머리카락 한 가닥이 어깨에 흘러내림* 등 미세한 디테일로 *나이트의 자식* 임을 암시
- ❌ 직접 *"이는 나이트와 세리스의 딸이다"* 서술 X / 시각적·암시적 박제만

#### 3.3. 독자에게 주는 효과

**박제 (대표님 그대로)**:
> "독자들은 아마도 그 여자가 마왕을 물리칠 거라고 생각하게 만드는 거지."

**작가 의도**:
- 독자가 *그 여성이 마왕을 물리칠 것* 이라 *추측* 하게 만듦
- 추측 = 확신 X / *희망 + 두려움 + 호기심* 의 복합
- = 시리즈 다음 시즌·시리즈·게임 진입의 강력한 후크

**독자가 동시 인지하는 것**:
- 마왕 = 한때의 영웅 (나이트)
- 마왕을 물리칠 용사 = 세리스의 모습 + 나이트의 머리색
- = *과거의 영웅이 적이 되고, 그를 물리칠 자는 그의 사랑·후손·환생*
- = 비극의 *완전한 순환*

#### 3.4. 끝맺음

**박제 (대표님 그대로)**: "끝맺음 하면 된다."

= 시리즈 종결 = 다음 사이클 진입의 *시작점*
