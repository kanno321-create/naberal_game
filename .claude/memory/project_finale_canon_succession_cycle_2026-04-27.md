# 시리즈 최종 결말 Canon — 순환의 호선

## 0. 결론 (대표님 직접 박제)

> **1. 마왕화 직후 = 신을 본보기로 찢어죽임 (카미코로시 1단계 실행) + 언데드·악마 부려 추종자·시스템 청소 (2단계) + 차별 반대자 1명을 새 신으로 임명 (시작지점 촌장 등) + 우주적 존재의 무기 전달·권능 하사·자격 부여로 신이 됨.**
>
> **2. 100년 이상 경과 → 인간+타종족 협동 문명 재건 → 모험가 길드 다종족 어우러짐 → 그러나 마왕(나이트)는 타락 — 본분 잊고 멍한 눈으로 100년 이상 반복, 몬스터·악마 계속 풀어 인간·타종족 모두 공격.**
>
> **3. 마지막 한 컷 — 모험가 길드에서 책을 읽던 마족 여성: *세리스의 생김새 + 나이트의 흰 머리*. 옆에서 들리는 이야기 = *마왕을 쓰러트릴 용사는 없는가*. = 다음 사이클 후크.**

이는 시리즈 *전체 호선의 종결 + 재진입 가능성* 박제. 1권~Vol.N 의 모든 회수 사슬이 마지막 한 컷으로 *순환* 으로 다시 열림.

## 1. 시퀀스 3 막 (대표님 박제 그대로)

### 막 1. 각성 직후 — 카미코로시 + 새 신 임명

#### 1.1. 신 처단 (카미코로시 1단계 실행)

> "마신/마왕으로 각성된 나이트가 신을 아주 본보기로 찢어죽이는 거지."

**행위**: 신을 *아주 본보기로 찢어죽임* — 단순 죽임 X / *공개 처단·시각적 박제* 형태

**의미**:
- 카미코로시 (神殺し) 1단계 = 시스템 정점 제거
- *본보기* = 다른 자들에게 *시스템 외부 존재가 신을 죽일 수 있음* 을 증명
- = 시스템의 *자가 정당화 불가능* 선언

**작법**:
- ✅ Show: 신의 시각적 처단 (찢어짐) + 권능의 박살 + 추종자들이 그 광경을 *직접 봄*
- ❌ 신의 형태 = 추상적·초자연적 X / *눈에 보이는 처단 가능한 형태* (육신·우상·왕좌 등)
- ❌ 단번의 처단 X / *느린·잔혹한 처단* (본보기 = 시간을 끌며 처단)

#### 1.2. 추종자·시스템 청소 (카미코로시 2단계 실행)

**대상 (대표님 박제 그대로)**:
- 교회 (성직자·추기경·신관)
- 추종자 (신실한 자들)
- *빌붙어 먹고 살던* 각종 귀족
- 평민 할 것 없이 (시스템에 협력하던 자들)

**수단**: 수많은 *언데드와 악마* 를 부림

**메커니즘**:
- 마왕의 권능 = 언데드·악마 소환·통제
- 시스템 측 = 수십만 / 마왕 측 = 무한대의 언데드·악마
- 결과 = 시스템 측 *완전 청소*

**의미**:
- 카미코로시 2단계 = 시스템 구조 청소
- 동시 = 카미코로시 3단계 *부분 실행* (시스템 충성 종족·평민 청소)
- → 종족 청소는 *완전 멸절* 이 아니라 *시스템 협력자만* 선별 청소

#### 1.3. 청소의 *예외* — 차별 반대자 보호

**박제 (대표님 직접)**:
> "타종족 차별을 반대하고 그들을 열심히 돌봐준 이들"

**의미**:
- 종족 청소 ≠ 인간 종족 *전체* 멸절
- *차별 반대자 = 보호*
- = 새 시스템의 *기반 인구* 가 됨
- = 카미코로시 3단계의 *진정한 본질*: 시스템에 협력한 자만 청소 / 시스템에 저항한 자는 보호

**작법**:
- ✅ 보호받는 자 = 1권부터 등장한 *기억 있는 자들* (회수 효과)
- ✅ 보호의 자세 = 마왕의 자비가 아니라 *예전 결단의 결과* (언데드·악마가 그들을 *알아봄* — 마왕이 미리 표시)

#### 1.4. 새로운 신 임명 (대표님 박제 핵심)

> "타종족 차별을 반대하고 그들을 열심히 돌봐준 이들 중 1명 (예를 들어 시작지점 촌장 등) 을 새로운 신으로 만들고."

**임명 후보** (대표님 박제 예시):
- **1순위**: Ch.01 *시작지점 촌장* (검은 머리 뾰족귀 아이를 보호하다 학살당한 마을의 촌장)
  - 1권 첫 등장 인물 → 시리즈 마지막 새 신 = *알파와 오메가 회수*
- **다른 후보**: 마리에 (약사) / 막다 (여관 주인) / 4년 전 학살 마을 어느 이름 없는 노인 / 기타

**임명 메커니즘** (대표님 박제 그대로):

| 단계 | 행위 | 의미 |
|------|------|------|
| 1 | **우주적 존재가 준 무기를 전달** | 우주적 존재 = 1권의 그 존재 (Vol.2 메인 플롯의 그 존재) / 무기 = 신적 권능의 매개체 |
| 2 | **권능을 하사받음** | 무기를 통한 권능 전이 |
| 3 | **권능을 사용할 자격을 얻음** | 자격 = *차별 반대자였다* 는 과거의 결단 |
| 4 | **신이 됨** | 권능 + 자격 = 신적 존재로 승격 |

**의미**:
- *신* 은 신실함이 아니라 *결단의 누적* 으로 만들어짐
- 새 신 = 과거 시스템의 *반대편* 에서 살아온 자
- = 5기둥 ② 신 대항의 궁극 형태 = *새 신을 누가 만드는가* 의 답: 신 대항을 한 자

**우주적 존재의 역할**:
- 1권부터 박제된 우주적 존재 (마왕이 아닌 다른 존재)
- *균형의 수호* 라는 본래 목적
- 마왕화 = 균형의 *시정* 행위
- 새 신 임명 = 균형의 *재구축*
- = 우주적 존재 = 시스템의 *외부 심판자*

**작법**:
- ✅ 임명 장면 = 1권 첫 등장 인물의 회상·재등장과 동시
- ✅ 무기 = 1권부터 박제된 시각적 모티프 (예: 손가락 마법진 / 흰 머리카락 한 가닥 / 부츠 흙 / 종소리 등) 와 연결
- ❌ 임명 = 단순 의식 X / *우주적 존재의 직접 출현* 또는 *마왕의 마지막 자비* 형태
- ✅ 새 신의 첫 권능 사용 = *과거 자기가 보호하지 못한 자들* 의 영혼 회복 또는 추모 (감정적 회수)

### 막 2. 100년 이상 경과 — 문명 재건 + 마왕의 타락

#### 2.1. 인간 + 타종족 협동 문명 재건

**시간 경과**: 수십 년 ~ 100년 이상

**재건의 모습**:
- 인간과 타종족이 *협동* 하여 문명을 재건
- 새 신 (시작지점 촌장 등) = 새 시스템의 정점
- 마족·엘프·오크·인간 등 모든 종족 = 차별 없이 공존
- = 1권 § 七 (촌장의 설명) 에서 박제된 *3년 전 신의 계시* 이전 상태로의 회복 + *진보*

**작법**:
- ✅ 재건 = 1권 박제 인물들의 후손이 활약
- ✅ 마리에 (약사) 의 후손이 의술을 잇거나 / 막다 (여관) 의 후손이 여관을 운영하는 등의 *작은 회수*
- ❌ 재건 = 완벽한 유토피아 X / *작은 갈등은 여전히 존재* 하지만 *해결 가능한* 형태

#### 2.2. 모험가 길드 — 다종족 어우러짐

**박제 (대표님 그대로)**:
> "수많은 종족의 모험가로 가득한 모험가 길드에서 여러 종족들이 이야기꽃을 피우고."

**모습**:
- 1권 § 五 *파르마 모험가 길드* 와 같은 형태
- 그러나 — 인간만이 아니라 *모든 종족* 이 등록
- 마족·엘프·오크·인간 등이 *같은 식탁* 에 앉아 *이야기꽃* 을 피움
- = 1권의 모험가 길드의 *진보된 모습*

**작법**:
- ✅ 길드 묘사 = 1권 모험가 길드와 *같은 구조* (게시판 / 접수대 / 안마당 / 식당)
- ✅ 그러나 분위기는 다름 — *환한 햇빛 + 다양한 옷색·향·언어* (1권의 회색·피냄새와 대비)
- ✅ 1권 박제된 모험가 길드 직원 (안경·깃펜) 의 후손 등장 가능

#### 2.3. 마왕 (나이트) 의 타락 — 멍한 눈의 반복

**박제 (대표님 그대로)**:
> "나이트 역시 수십 년 백년 이상 마왕/마신으로 지내오며 조금씩 타락하고 있었고. 본분을 잊은 채 멍한 눈으로 백년 이상 해오던 걸 반복만 할 뿐이다. 몬스터 및 악마들을 계속 풀어서 인간 및 타종족 모두를 공격하고 있었던 것."

**타락의 메커니즘**:
- 마왕화 = 인간성 외부 도약 = *시간 감각의 마비*
- 100년 이상 = *시간 자체가 의미 없어짐*
- 본분 (균형 수호 / 시스템 처단) → *잊음*
- 멍한 눈 = 1권 Ch.04 § 二 *Cailean 셈이 끝난 자의 눈* 과 같은 종류

**행동**:
- 몬스터·악마를 *계속* 풀어 *인간·타종족 모두* 를 공격
- = 본래 카미코로시 대상이 아니던 *차별 반대자의 후손들* 도 공격
- = 마왕 자신이 *과거 자기가 만든 시스템의 적* 이 됨

**아이러니**:
- 1권 § 七 신관의 *해야 할 일을 해내는 자의 표정* 과 같은 종류
- 마왕 (나이트) = 100년 후의 *그 신관* 과 같은 자세
- = *시스템에 길든 자* 의 형태가 됨
- = 시스템 외부 존재가 *새 시스템의 일부* 가 됨

**작법**:
- ❌ 타락 = 분노·광기 X / *무감각·반복· 의식 없음*
- ❌ 마왕의 자각 X / *자기가 무엇을 하는지조차 모름*
- ✅ Show: 마왕의 *멍한 눈* / *반복적 동작* / *시간 감각 마비* 의 묘사
- ✅ 5기둥 ① 절망 하강의 궁극: 정의의 영웅도 시간 앞에서 부패

#### 2.4. 새 신의 부재 또는 부패 (옵션)

**상태**: [대표님 결정 대기]

**가능성**:
- (a) 새 신 (시작지점 촌장) 도 100년 후 죽음 → 후계자 부재 또는 부패
- (b) 새 신은 영생 (권능으로) → 그러나 마왕의 타락을 제어 못 함
- (c) 새 신은 죽고 새 시스템이 자가 운영 → 시스템 자체가 부패 시작

본 캐논은 (b) 또는 (c) 노선을 *암시* 만 박제. 구체는 대표님 결정 영역.

### 막 3. 마지막 한 컷 — 다음 사이클 후크

#### 3.1. 모험가 길드의 마지막 장면

**박제 (대표님 그대로)**:
> "그중 어떤 이들의 이야기를 그 옆에서 책을 읽던 마족 여성이 듣게 되는데, 최근 마신/마왕의 폐단과, 무분별한 악마와 언데드를 풀음으로서 피해가 상당하니 그를 쓰러트릴 용사는 없는 건가 과연…… 이런 이야기를 듣게 되고, 끝맺음 하면 된다."

**장면 구조**:

| 요소 | 박제 |
|------|------|
| 위치 | 모험가 길드 (1권의 그 길드와 같은 구조 / 또는 다른 도시의 다종족 길드) |
| 분위기 | 환한 낮 / 다양한 종족 어우러짐 / 이야기꽃 |
| 주변 인물 | 모험가들 (다종족) — 마왕 폐단을 이야기 |
| **중심 인물** | **마족 여성 — 옆에서 책을 읽음** |
| 이야기 내용 | "최근 마왕의 폐단" / "무분별한 언데드·악마" / "피해가 상당" / **"그를 쓰러트릴 용사는 없는가"** |
| 마족 여성의 반응 | (대표님 박제 X — 작가 결정 영역) |

#### 3.2. 마족 여성의 정체 — 나이트 + 세리스의 *마력 융합 딸* (확정)

**박제 (대표님 직접 — 2026-04-27 추가 확정)**:
> "세리스와 나이트는 인간이 아님. 마족은 인간처럼 번식 안 하고 마력을 통해서 그들을 닮은 자를 자식으로 만들게 되고, 나이트 또한 인간이 아니므로, 일반적인 번식이 안 됨. 그러므로 나이트의 강력한 마력과, 세리스의 그 마력을 받아들임을 암시하는 장면을 넣어주고. 그 결과물이 새로운 시대의 용사다. 한마디로 직접 생식활동을 해서 낳은 자식이 아닌, 나이트의 마력, 세리스의 마력이 융합되어 만들어진 존재로 마족에선 이를 자식으로 여긴다. 한마디로 나이트와 세리스의 딸이다."

**시각적 박제**:
- 생김새 = *세리스* (마족 / 뾰족 귀 / 어두운 눈 / 마법사적 분위기)
- 머리색 = *나이트의 흰 머리*
- = 두 부모의 시각적 융합 = *마력 융합의 결과물*

- 마족 (세리스) 와의 마력 결합 = 가능
- = 나이트 + 세리스 = 마력 융합 자식 가능

## 6. 5기둥 톤 정합 (궁극 형태)

### 6.1. 5기둥 ① 절망 하강 — 정의의 영웅도 시간 앞에 부패

이 시퀀스 = 5기둥 ① 의 *궁극 형태*:
- 1권: 마을 학살 (절망의 시작)
- Vol.2: 자기 손 학살 자각 (절망의 깊어짐)
- Vol.N 클라이맥스: 세리스 죽임 (절망의 정점)
- **결말 100년 후**: 마왕 자신의 *부패* (절망의 영원성)
- = 절망은 *영원히 끝나지 않음* / 영웅도 부패함

### 6.2. 5기둥 ② 신 대항 — 신의 *대체* 와 *부패*

5기둥 ② 의 궁극:
- 신을 죽임
- 새 신을 만듦
- 마왕 자신이 *신과 같은 존재* 가 됨
- 100년 후 마왕 = 부패 = 시스템에 길듦
- = *신 대항조차 결국 새 신 만들기* 의 아이러니
- = 시스템의 *영원한 자가 재생산*

### 6.3. 5기둥 ③ 오버로드식 사망 — 마왕 본인의 잠재적 사망

5기둥 ③ 의 다음 사이클:
- 1권 ~ Vol.N: 동료들의 누적 사망 (세리스 정점)
- 결말: 마왕 자신이 *언젠가 죽을 것* 의 암시
- = 다음 사이클에서 *마왕 = 죽을 차례*
- = 오버로드식 사망의 *새 주인공* 후보

### 6.4. 5기둥 ④ 쉼표·블랙유머 — 모험가 길드의 *가벼운 호흡*

마지막 한 컷 = 5기둥 ④ 의 *균형 호흡*:
- 무거운 회고 (마왕의 폐단) + *모험가들의 가벼운 이야기꽃*
- 100년 후의 *환한 낮* + 마왕의 *어두운 그림자*
- = 다크 판타지 *비극과 일상의 공존* 정점

### 6.5. 5기둥 ⑤ 비례적 대가 — 마왕의 *시간의 대가*

5기둥 ⑤ 의 궁극:
- 마왕화 = 인간성의 *영구한 대가*
- 100년 = *시간 자체가 대가* 가 됨
- 멍한 눈 = 본분의 *영혼적 부식*
- = 곁의 자 부식의 마지막 형태 = *자기 자신의 부식*

## 8. 작법 강제 (결말 시퀀스)

### 8.1. 막 1 (각성 직후)

- ❌ 신의 형태 = 추상적 X / *눈에 보이는 처단 가능한 형태*
- ❌ 단번의 처단 X / *시간을 끌며 처단* (본보기)
- ❌ 청소 = *완전 멸절* X / *시스템 협력자 선별 청소*
- ✅ 보호받는 자 = 1권 박제 인물 (회수 효과)
- ✅ 새 신 임명 = 1권 첫 등장 인물 (시작지점 촌장 등)
- ✅ 우주적 존재의 직접 출현 또는 무기 전달의 시각적 박제

### 8.2. 막 2 (100년 후)

- ❌ 재건 = 완벽한 유토피아 X / *작은 갈등 존재*
- ❌ 마왕의 후회·자각 묘사 X (자각 X = 비극)
- ❌ 마왕의 분노 묘사 X (멍함이 정답)
- ✅ 1권 박제 인물의 후손이 활약 (작은 회수)
- ✅ 모험가 길드 = 1권의 그 길드와 같은 구조 + 다른 분위기 (대비)
- ✅ 마왕의 *멍한 눈* / *반복적 동작* / *시간 감각 마비*
- ✅ 1권 모티프의 *부서진 형태* (손가락 마법진 흐려짐 / 흰 머리카락 한 가닥 부서짐)

### 8.3. 막 3 (마지막 한 컷)

- ❌ 마족 여성의 정체 *명시* X (의도적 모호)
- ❌ 마족 여성의 행동 *큰 묘사* X (책을 읽음 + 듣는 자세만)
- ❌ 마지막 박자 = 결정적 액션 X / *조용한 종결*
- ✅ 시각적 디테일: 책 / 옷자락 / 머리카락 한 가닥 / 어두운 눈
- ✅ 옆 테이블 모험가들의 이야기 = 자연스러운 잡담 형태
- ✅ 핵심 대사 *"그를 쓰러트릴 용사는 없는가 과연……"* 그대로 박제
- ✅ 마지막 박자 = 마족 여성의 *미세한 변화* 또는 *그대로 책 읽음* (둘 다 가능)

### 8.4. 호흡 가이드

- 막 1 = 강렬한 호흡 (신 처단·청소·임명)
- 막 2 = *시간이 흐른 호흡* (재건 + 100년 + 마왕의 멍함)
- 막 3 = **조용한 호흡** (모험가 길드 + 잡담 + 마족 여성 + 책 + 마지막 박자)
