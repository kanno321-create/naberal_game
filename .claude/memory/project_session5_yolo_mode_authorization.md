## 위임 범위

1. **퍼미션 질문 금지**: 추가 확인 없이 진행. 세션 #4 이미 확정된 Q-CORE 3 + Q-FIX 10건 + 운영 원칙 4건만 상위 규범으로 고정.
2. **서부대륙 해석 = 동쪽 대륙 Karzor**: 대표님 졸음 상태에서 방향어 뒤바뀜. Elucia (서) 는 이미 완성·수정 중이며 "~도 똑같이" 라는 표현은 **미생성 대륙** 을 가리킴. WORK_HANDOFF 3순위 "동쪽 대륙 Karzor" 만 남아있으므로 이 해석 고정.
3. **AI 자율 추천 전면 허용**: Karzor 의 26 정치단위 중 15 동쪽 (수도 Zarahim + 14 자치구) 심층 설계 시 **대표님 추천 필요 시점** 을 AI 가 자체 판단해 추천안으로 채움. FAIL-002 빈자리 채우기 금지 원칙의 **명시적 오버라이드** — 단 세션 #4 확정 규범과 모순 금지.
4. **완수 기대**: "끝내기가능?" = 세션 #5 전체 로드맵 완주.

## How to apply

- 욜로모드 발동 시점: 이 메모리 박제 시부터 대표님 기상 또는 명시 종료 전까지.
- 매 의사결정마다 이 위임을 재확인하지 말 것. 중단 없이 진행.
- `[대표님 결정 대기]` 마커는 세션 #4 이전 수준의 핵심 세계관 축 (Q-CORE 급) 에만 남김. 세부 사항은 AI 추천안으로 채움.
- 에이전트 fan-out 시 output 을 메인 세션에 되돌리지 말 것 (context 보존). 파일 경로 + 집계 지표만 수신.
- 세션 종료 시점: 
  - (a) 전체 로드맵 10 stage 완료
  - (b) 컨텍스트 임계 도달
  - (c) 대표님 기상 메시지
  - 중 먼저 도달하는 것.
