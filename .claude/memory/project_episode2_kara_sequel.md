# Episode 2 시퀄 Canon · 여주인공 카라

## 확정 사실 (대표님 2026-04-24 직접 지정)

### 1. 시퀄 주인공
- **이름**: **카라 (Kara)**
- **성별**: 여성
- **정체**: Q-CORE 5 · 우주 균형 수호자 집단이 파견한 다음 세대 수호자
- **외형 일치**: 본편 에필로그 Cut 1 의 "금발 금안의 젊은 신참 수호자 · 백색 아머 · 오만한 광신적 미소" = 카라 (이름 미노출 상태)

### 2. 핵심 서사 방향
**카라가 나이트를 물리치는 스토리** — 본편에서 스스로 '악의 신' 이 된 나이트가 Episode 2 의 **최종 보스** 역할. 카라가 주인공으로 그를 타도.

### 3. 본편 연속성
- 본편 E5 진엔딩 에필로그 Cut 1~3 이 **Episode 2 의 프롤로그** 구실
- Cut 3 "파란 눈 → 보라 공허 광채" = 카라 강림 시점에 마왕 나이트가 눈을 뜸
- 본편 설계 헌법 제3조 "모두 비악" 은 Episode 2 에서도 유지 — **카라 = 선 · 나이트 = 악** 이분법 절대 불가
  - 카라도 수호자 시스템의 꼭두각시 · 나이트도 순환을 깨려는 자발적 대립항
  - "누구를 응원해야 하는가" 의 딜레마가 Episode 2 의 서사 엔진

## 설계 시사점 (본편 집필 시 고려)

### A. 본편 에필로그에 "카라" 이름 노출 금지 (현재 시점)
- 대표님 원문: *"나중에 두 번째 에피소드는 카라라고 하는 여주인공이 나이트를 물리치는 스토리로 하려고"*
- → **"나중에"** 지정 = 본편에서는 이름 비공개 유지 · 시퀄 서막에 공개
- 페이즈 5 에필로그는 **"금발 금안의 젊은 신참 수호자"** 표현 그대로 유지

### B. 본편 Cut 3 보라 광채의 의미 재확정
- 본편 독자 관점: "나이트가 깨어남 · 새 수호자와의 충돌 예감"
- Episode 2 관점: **"카라의 첫 강림 = 마왕 나이트의 자각 트리거"**
- 본편 비극성 + 시퀄 시작점 이중 기능

### C. Episode 2 구조 가설 (대표님 확정 전 · 나베랄 감마 초안)
- 프롤로그: 카라의 파견·낙하·수호자 본부 브리핑
- Act 1: 카라가 행성 도착 후 현지인(본편 생존자 3인 + 후손) 과 협력 · 마왕 나이트 정보 수집
- Act 2: 마왕의 군세(공허의 진군) 와의 대립 · 본편의 진실(모두 비악) 점진 발견
- Act 3: 카라가 마왕 나이트를 물리치되, **순환이 또 다시 리셋되는** 비극 · Episode 3 씨앗

### D. 카일 · 엘라라 · 미리암의 시퀄 위치
- 본편 진엔딩 생존 3인 = **Episode 2 에서는 최소 100년 이후 고령 · 후손 또는 기록만 남음**
- 카일 서술자 프레임이 본편을 감싸지만, Episode 2 는 **카라 1인칭 또는 3인칭 제한 카라 중심** 으로 시점 갱신 필요

## 🧬 카라의 혈통 · 미확정 아이디어 (대표님 2026-04-24 "맘같아서는" 고민 중)

**대표님 원문**: *"맘같아서는 나이트랑 여주의 딸이라고 하고싶은데"*

→ **카라 = 나이트 × 본편 여주 의 딸** 가능성. 아직 확정 아님 · "맘같아서는" = 고려 중.

### 핵심 딜레마
- **매력**: 본편 주인공 + 여주의 유전자·의지가 시퀄 주인공에게 직접 계승 → 감정선 강력 · 뫼비우스 비극 배가
- **리스크**: 수호자 집단 설정 상 "시스템이 기억을 리셋해 재투입" 이 Canon (Q-CORE 4). 자연 혈통이 끼어들 여지를 서사적으로 정당화해야
- **시간선 문제**: 본편 Ch.22~23 (세리스 막사 밤 · 진실 폭로) ~ E5 진엔딩 까지의 짧은 기간에 잉태·출산 불가능 · 별도 장치 필요

### 옵션 A — 카라 = 나이트 × 미리암 의 딸 🕊️
| 항목 | 내용 |
|------|-----|
| **여주 후보** | 미리암 (생존 3인 · 연모 오열 · 성녀) |
| **잉태 시점** | Ch.22 전후 야영지의 하룻밤 또는 진엔딩 직전 마지막 회상 |
| **출산** | 진엔딩 직후 미리암이 분수대 앞 주저앉은 상태에서 임신 자각 → 이후 수년간 육아 |
| **카라 속성** | 성녀의 자애 + 광신적 미소 이중 · 어머니 미리암의 원죄·오열을 알고 자람 · 수호자 집단이 후일 강제 징집 |
| **장점** | 감정선 자연스러움 · 미리암 생존의 정당성 강화 · 본편 삼각 관계의 장기적 해소 |
| **단점** | 미리암 Ch.22 막사 씬이 없으므로 잉태 장면 신설 필요 · 세리스 소멸 서사와의 감정 충돌 |

### 옵션 C — 카라 = 수호자 집단이 유전자 회수한 인조 존재 🧪
| 항목 | 내용 |

| **단점** | 혈통 감정선이 다소 차가움 · 대표님 "딸" 표현이 주는 따뜻함·무게 감소 |

### 옵션 D — 혼합 (A + B 의 유전자 복합) 🌹
| 항목 | 내용 |
|------|-----|
| **구성** | 카라 = 나이트 + 미리암 자연 혈통 (어머니·출산) + 세리스 마력 핵 이식 |
| **서사** | 나이트와 미리암의 딸로 태어나되, 세리스 소멸 직전 방출된 마력 핵 씨앗이 갓난 카라에 이식됨 |
| **카라 속성** | 미리암의 자애 · 나이트의 수호자 혈통 · 세리스의 마족 마력 핵 = 3 주요 캐릭터 전원 시퀄 계승 |
| **장점** | 본편 삼각 관계·동료 희생의 서사적 가치 극대화 · 가장 복잡한 비극 카타르시스 |
| **단점** | 설정 복잡도 급증 · 독자 설명 부담 · 시퀄 프롤로그가 장황해질 위험 |

### 나베랄 감마 추천
**옵션 D 변형 > 옵션 A > 옵션 B > 옵션 C** 순.

- D 변형 이 가장 서사적 무게 크나 설정 복잡 → **A + 세리스 마력 핵 암시 정도** 가 실전 균형
- 옵션 A 만 채택 시 가장 자연스럽고 집필 부담 적음
- 옵션 C 는 Q-CORE 시스템과 일관되나 대표님 "딸" 감정과 거리

→ **대표님이 "딸" 표현을 쓰신 것 자체가 혈통 감정선 중시의 신호** · 옵션 A 또는 A+B 혼합(D) 가 의도에 근접.

### Canon 편입 시 본편 역소급 영향
- **옵션 A 채택 시 본편 수정 사항**:

  - 진엔딩 에필로그에 미리암 임신 암시 (예: 주저앉은 미리암 뒤 수개월 후 배가 불러오는 짧은 컷)
- **옵션 B 채택 시**: 세리스 소멸 연출에 마력 핵 결정체 잔류 · 수호자 회수 암시 1 컷
- **옵션 D 채택 시**: 양쪽 모두 + 카라 배양 과정 미니 에필로그
