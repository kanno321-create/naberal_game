# 이중자아 명명 통일 + 메타 어휘 빈도 압축

## 0. 5 에이전트 검수 지적 (라운드 6 외부)

> **검사관 지적**: "이중자아 명명 표류 — 머릿속 남자 → 내면 자아 → 외면 자아 → 내면 → 외면 → 두 자아 → 6 챕터 내 *3번 변경* = 독자 학습 부하"
>
> **편집장 지적**: "*외면 자아 / 내면 자아* 라는 용어가 Ch.02부터 *공식 어휘*로 등장. 매번 두 자아를 *분리·이름표·작동* 의 3단계로 추적하게 만듦"

## 1. 통일 명명 박제 (Canon)

### 1.1. Full Form (공식 명명)

| 명명 | 사용 영역 |
|------|----------|
| **외면 자아** | 1인칭 화자 = 의식 = 몸 지배자 (현재 시점) |
| **내면 자아** | 무의식 = 감정 누수 = 수천 년 누적자 |

### 1.2. Short Form (자주 사용 시)

| 명명 | 사용 |
|------|------|
| **외면** | 외면 자아의 짧은 호칭 |
| **내면** | 내면 자아의 짧은 호칭 |

### 1.3. 금지 명명

| 금지 | 이유 |
|------|------|
| 머릿속 남자 | Ch.01 도입부에서만 *최초 인지* 표현 사용 가능 / Ch.02 이후 사용 금지 |
| 두 자아 | 모호 (*어느 두 자아?*) — 사용 금지 |
| 두 목소리 | 신체적 비유 — *외면/내면* 통일 위해 사용 금지 |
| 그 친구 | Ch.02 line 124 *친구* 호칭 = 이중자아 명명 일관성 깨짐 |

### 1.4. 신체 비유 (선택적 사용 가능)

다음 비유는 *외면/내면* 명명 대신 사용 가능 (라노벨 호흡 회복):

| 신체 비유 | 의미 | 사용 영역 |
|----------|------|----------|
| 두 호흡 | 외면+내면 호흡 차이 | 전투·긴장 시 |
| 두 손 | 외면+내면 행동 차이 | 행동 비교 시 |
| 두 박자 | 외면+내면 시간 인식 차이 | 의식 흐름 시 |

→ 신체 비유는 *외면/내면 명명을 대체* 하는 자세 / 정착되지 않게 빈도 ≤ 챕터당 2회

## 2. 메타 어휘 빈도 압축 (검사관 지적)

### 2.1. 측정 결과 (라운드 5 후)

| 챕터 | 외면 자아 | 내면 자아 | 합계 |
|------|----------|----------|------|
| 나이트1 | 1 | 0 | 1 |
| 나이트2 | 4 | 7 | 11 |
| 나이트3 | 2 | 1 | 3 |
| 나이트4 | 1 | 0 | 1 |
| 나이트5 | 4 | 4 | 8 |
| 나이트6 | 1 | 2 | 3 |
| **합계** | **13** | **14** | **27** |

### 2.2. 강제 빈도 (Vol.2~Vol.N)

- **챕터당 ≤ 5회** (외면 + 내면 합산)
- 같은 단락 내 *외면이 ~ / 내면이 ~* 패턴 반복 = 1회로 압축
- *외면이 동의했다 / 내면이 침묵했다* 류 매 단락 1회 = 금지

### 2.3. 변주 동의어 (메타 어휘 대체)

| 메타 어휘 (제한) | 변주 |
|----------------|------|
| 외면 자아가 ~ | 의식이 ~ / 손이 먼저 ~ / 외면이 ~ |
| 내면 자아가 ~ | 무의식이 ~ / 속삭임이 ~ / 내면이 ~ |
| 두 자아가 동의했다 | 합의가 한 자리에서 ~ |
| 두 자아가 침묵했다 | 안과 밖이 같은 침묵에 ~ |

## 3. Ch.01 *머릿속 남자* → *내면 자아* 정착 박제

### 3.1. 1권 명명 진화 정합

- Ch.01 도입 §二: *머릿속 남자* (인지 안 됨 / 처음 인지)
- Ch.01 §五~§九: *그 친구* / *속삭임* (인지 후 호칭 모호)
- Ch.02~Ch.06: *외면 자아 / 내면 자아* (공식 명명)

### 3.2. Ch.01 명명 정착 자연스러움 보존

Ch.01 의 *머릿속 남자* / *속삭임* 은 캐릭터의 *처음 인지* 표현이므로 보존. Ch.02 부터 *외면/내면* 공식 명명 사용. **명명 진화 자체가 캐릭터 발전 곡선의 일부**.

## 4. Vol.2~Vol.N 강제

- 챕터당 *외면 자아 / 내면 자아* 합산 ≤ 5회
- *머릿속 남자* / *그 친구* / *두 자아* / *두 목소리* 사용 금지
- *외면이 ~ / 내면이 ~* 같은 단락 반복 = 압축
- 신체 비유 (두 호흡 / 두 손 / 두 박자) 사용 가능 단 빈도 ≤ 2회
