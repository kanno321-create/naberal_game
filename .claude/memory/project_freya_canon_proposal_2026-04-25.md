## 2. Canon 이미 확정된 요소 (세션 #11 이전)
- 이름 **Freya** — 북유럽 전쟁·사랑 여신
- 16세 · 성인식 직전
- **훈련받은 몸** — 섬세하지만 근육질은 아님
- **숏소드** 소지 + 덫 제작 도구
- **이중 눈빛**: 반짝이는 밝은 눈 → 어둡고 무거운 눈 (복선)
- 성인식 직전인데 훈련병처럼 보이는 위화감
- 전쟁·죽음 여신 이름 = 신비로운 톤의 복선

## 3. 세션 #11 신규 설계 (표지 박제)

### 3.1 얼굴·머리
- **머리 색**: 금발 (Norse 여신 이름 · 복선 대비)
- **머리 스타일**: 땋음 + 흩날림 혼합
- **눈 색**: 녹색-푸른 (bright green-blue)
- **얼굴형**: 섬세한 anime bishoujo · 16세 적정 (어린이도 성숙녀도 아님)

### 3.2 복장
- **상의**: 가죽 조끼 (sleeveless leather vest) + 긴소매 earth-tone 튜닉
- **하의**: fitted 갈색 바지
- **신발**: knee-high traveling boots
- **방어구**: 양팔에 가죽 bracers
- **액세서리**: 벨트 + 매달린 pouch
- **무기**: 숏소드 좌측 허리

### 3.3 자세 (표지)
- 나이트 우측에 서서 한 손은 숏소드 손잡이 위
- 표정: alert but soft (경계하지만 부드러움)
- 키: 나이트 어깨 높이 (나이트 178cm → 프레이아 약 165cm 추정)
