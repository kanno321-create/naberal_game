# Q-CORE 4 — 나이트 이중 자아 → 융화 구조

## 확정 (2026-04-22 세션 #6 · 대표님 직접 5축 결정)

### 1) 호칭
- **균형 수호자 = 창조물** (창조되는 존재, 원래 이름 없음)
- Act 1 에서 누군가가 이름을 물어 → 백지 소년이 즉흥으로 **"나이트"** 선택
- Act 2~3 동료들과의 모험·시간 공유 → 주인공이 **스스로 "나이트"를 본인 이름으로 확정**
- **이름 자체가 자아 융화의 상징** (즉흥 호명 → 정체성 확정)

### 2) 융화 트리거 (하이브리드)
- **레벨업 시 (10레벨당 1회)** 자동 융화 이벤트
- **"기억의 조각" 획득 시마다** 새 스킬·능력치·패시브 추가
- 게임 메카닉 + 내러티브 이벤트 동시 작동 → 미디어믹스 완벽 연동
- LN 독자 = 기억 조각 서사로 성장 체감 / 게임 플레이어 = 레벨업으로 성장 체감

### 3) 속도 커브
- Act 1 말부터 **점진 (완만)**
- **그라인딩 보상**: 미친 듯이 노가다하면 지역 레벨보다 훨씬 강력한 스킬 획득 가능 (숙련 플레이어 우대)
- 일본 RPG 전통 (Dragon Quest/Final Fantasy) + 소울라이크 (Elden Ring) 계승

### 4) 발전된 자아 주력
- **백지 기반 + 수호자 지혜 흡수 (감정 중심)**
- 최종 나이트 = **감정을 가진 수호자** (인간적 공감 + 수호자 지식)
- 원형 수호자 (냉정·규칙) 보다 발전 = **선택할 줄 아는 수호자**
- 장르 차별화: 오버로드 아인즈(냉정) / Nier 2B(규칙) 와 정반대 방향

### 5) Bad Ending
- **없음** (실패 시나리오 미존재)
- 엔딩은 "실패/성공" 이 아니라 **"어떻게 성장했는가"** 의 차이
- 성장·융화는 확정. 속도·완성도·방향만 엔딩별로 차이.
- 기존 엔딩 5구조에서 Bad 자리 → **차후 재정비 필요**

## 두 자아의 관계 · Act 별 구조

### Act 1 (Ch.01~11) — 1인칭 · 백지(A) 지배
- A = 백지 나이트 (인간 감정·학습 능력·순수성)
- B = 균형 수호자 (봉인된 잠재 자아, 규칙·지식·의무감)
- **B 는 괄호·지문·"몸이 먼저 안다" 감각으로만 현현**
- 이름 "나이트" = 즉흥 호명 단계
- A 가 B 의 존재를 **알지 못함** — 단지 이상하게 몸이 아는 것들이 있을 뿐

### Act 2 (Ch.12~) — 3인칭 제한 · 두 자아 공존·대화
- B 의 기억이 조각조각 회복되기 시작
- A·B 대화·갈등 시작 (적대 X · 협상 중심)
- 괄호 발화 빈도 증가 — B 가 점점 명료해짐
- 기억의 조각 수집 · 레벨업 이벤트 집중 구간

### Act 3 — 장면별 혼용 · 점진 융화 → 통합
- A + B 융합 과정 실시간 서술
- 괄호 발화 감소 → 통합 단일 자아로 수렴
- 이름 "나이트" = 본인이 **선택한 정체성**으로 승격
- 엔딩 유형별 융화 완성도 차등

## 문체적 표현 (style_bible v2 연동)

| 요소 | 표기 규칙 | Act 별 밀도 |
|------|---------|------------|
| A 발화 | 본문 1인칭 서술 기본 | Act 1: 주력 / Act 3: 감소 |
| B 발화 | **(괄호)** 또는 *이탤릭* | Act 1: 씬당 0~2회 / Act 2: 2~5회 / Act 3: 감소→통합 |
| 신체 반응 | 지문 "몸이 안다" | 전 Act 공통 |
| 이름 표현 | Act 1: 즉흥 / Act 2: 균열 / Act 3: 확정 | 이름 자체가 서사 장치 |

**융화 진도 = 괄호 발화 빈도 변화**로 측정.

## Q-CORE 1/2/3 과의 정합성 검증

- **Q-CORE 1** (마왕≠할배, 수호자가 마족 신 승격) — 수호자 개념 계승 ✅
- **Q-CORE 2** (할배 속죄) — 별개 서사, 충돌 없음 ✅
- **Q-CORE 3** (수정 1/2 이중 관리) — 이중 자아와 **주제 호응** (세계 이중 구조 ↔ 자아 이중 구조) ✅

→ Q-CORE 4 는 기존 3개와 **상호 보강** 관계.
