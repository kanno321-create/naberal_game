# 확정 사실 (대표님 2026-04-22 세션 #4)

## 1. Q-CORE 3 해소
- **수정 2 = 마왕의 마족 증식 장치** (신규 가설 5 확정)
- 기존 4 가설 (마족 백업·초고대·수호자·우주 포식자) 전부 폐기
- 나베랄 감마 직전 추정 "초고대 가설 자연 부합" = 과해석 오류 · 자기 정정 (2회 연속 FAIL-002 조짐)

## 2. 제작 메커니즘
- **제작자**: 태초의 마족 마왕 (= 최초의 마족 단독 존재)
- **제작 목적**: 마족을 늘리기 위해
- **당시 세계 상태**: 마력 분포 불안정 · 여러 곳에서 마력 붕괴 발현
- **제작 공정**: 여러 마력 붕괴 에너지를 수정에 담아 완성
- **결과물**: 제2 마력 수정 (= 수정 2)

## 3. 시간축 최종 정합 (Q-CORE 1+2+3 통합)
```
태초 — 마왕 단독 존재 (= 최초의 마족)
    ↓ 수정 2 제작 (마력 붕괴를 담아)
마족 증식 → 마족 집단 형성
    ↓ 마족 시대 · 용족·엘프 공존 · 황금기
    ↓ 수정 1 제작 (지식·계산 네트워크 · 발언 18)
마족 붕괴 · 수호자 개입 · 마족 1명 승격 → 첫 번째 신(할배)
    ↓ 신 시대 — 할배 판단 미스
인간 시대 — 할배 속죄 · 서민 마법 배포 · Rev.3 본편
```

## 4-2. 인간 사회의 할배 인식 (2026-04-22 확정)
- 실체 할배 = 마법 잘 아는 착한 노인으로만 인식
- "첫 번째 신" 교리 = 별도의 추상적 신학 대상 (할배와 무관한 것으로 인식)
- 두 개념이 동일 존재임은 Rev.3 Ch.23B 공개까지 아무도 모름
- 할배는 숭배받지 않는 **완벽히 숨은 존재** — 속죄 삶의 구체 양식

## 6. 마왕의 종족 문제 해소
- 이전 미정 질문 "마왕의 종족" = **마왕 자신이 마족 시초** · 별도 외계 종족 아님
- "마왕은 마족 시대 이전부터 존재" = 마족 집단이 형성되기 전 마왕 단독 시기
- 즉 마왕의 종족적 정체 = 마족 (그 중 최초·단독)
