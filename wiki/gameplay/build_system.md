# 빌드 시스템 — 스피어 보드 × 정복자 보드 × 스킬트리

> 대표님 브레인스토밍 (2026-04-21) 2차 박제 · Revision 2.
> 설계 헌법 제2조 (빌드 다양성) 준수.

## 설계 원칙 — 헌법 제2조

> **모든 극단 빌드가 엔드게임을 클리어 가능해야 한다.**

**참조 성공 사례**:
- Path of Exile — 10년 빌드 다양성
- Last Epoch — 1인 개발 대작 (Ailment·DoT·Minion·Crit 모두 유효)
- FF10 — 스피어 보드 자유도
- Diablo 4 — 정복자 보드 + 스킬트리 확장

---

## 주인공 직업 시스템 — 고유 직업 + 다직업 느낌

### 확정 구조 (대표님)

> **주인공 = 고유 직업 "균형의 수호자" (서사적 고정).**
> **단, 스킬트리 경로에 따라 여러 직업 같은 느낌으로 전투 가능.**

이는 FF10 티다 · 아론 · 루루 등이 자신의 길을 가되, 스피어 보드로 다른 캐릭터 경로를 배울 수 있는 구조와 유사. **클래스 시스템 없이 빌드 다양성** 확보.

### 5대 분기 (스킬트리 경로)

| 분기 | 직업 느낌 | 핵심 |
|---|---|---|
| **진노 (Rage)** | 광전사 · 바바리안 | AP 폭발 · 콤보 극대화 |
| **수호 (Bulwark)** | 팔라딘 · 기사 | HP 극대 · 극딜 누적 |
| **속박 (Affliction)** | 강령술사 · 네크로만서 | 상태이상 곱연산 |
| **균형 (Harmony)** | 룬마스터 · 만능 전사 | 모든 상황 적응 |
| **광기 (Madness)** | 광신도 · 살수 | HP 낮을수록 강함 |

**혼합 빌드 가능**: 진노 60% + 속박 40% 등 자유 분배. PoE 수준 깊이.

## 스킬 시스템 3층 구조 — 대표님 확정

대표님이 정확한 공식을 제시하셨습니다:
**FF10 스피어 보드 + Diablo 4 정복자 보드 + Diablo 4 스킬트리**

### 1층 — 스킬트리 (Diablo 4 방식)

**내용**: 액티브 스킬 · 패시브 선택.
**역할**: 메인 전투 스킬 선택 (4~6개 슬롯에 할당).
**작동**: 레벨업 시 스킬 포인트 → 스킬트리 노드 해금.

**예시 구조 (진노 분기)**:
```
진노 스킬트리
├─ 기본
│   ├─ [액티브] 회전베기 (광역)
│   ├─ [액티브] 관통찌르기 (단일)
│   └─ [패시브] 기본 공격 +10%
├─ 중급 (기본 트리 완료 후 해금)
│   ├─ [액티브] 광전사 모드 (버프)
│   ├─ [액티브] 연속베기 (광역 다중)
│   └─ [패시브] 크리티컬 확률 +15%
└─ 상급 (중급 일부 해금 후)
    ├─ [액티브] 혈우의 춤 (궁극기)
    └─ [패시브] 처형 임계치 +10%
```

**스킬 포인트 비용**:
- 레벨 1당 1포인트 획득
- 기본 노드: 1포인트
- 중급 노드: 2포인트
- 상급 노드: 3포인트
- 궁극기: 5포인트

### 2층 — 스피어 보드 (FF10 방식)

**내용**: 특수 스피어를 사용해 큰 격자 보드의 노드 해금.
**역할**: **스탯 확장 + 범용 스킬 학습** (스킬트리 외부의 보조).
**작동**: 전용 자원(스피어) 으로 노드 활성화.

**스피어 종류**:
- **보통 스피어**: 현재 위치 주변 노드 해금 (점진적 확장)
- **고속 스피어**: 먼 노드까지 이동 (시간 절약)
- **특수 스피어**: 잠긴 노드 해제 (보스 드랍)

**노드 종류**:
- 스탯 노드 (HP +100, 공격력 +5, AP +1 등)
- 범용 스킬 노드 (분기 무관하게 습득 가능)
- 크로스오버 노드 (다른 분기의 기본 스킬)
- 비밀 노드 (숨겨진 고유 권능)

**크로스오버 원리**:
진노 빌드를 메인으로 하되, 스피어 보드로 수호 분기의 "얼음벽" 스킬을 배울 수 있음. **빌드 하이브리드** 가능.

**참조**: FF10 스피어 보드의 "캐릭터 간 경로 교차" 메커니즘 → 이 게임에서는 "분기 간 교차".

### 3층 — 정복자 보드 (Diablo 4 Paragon Board 방식)

**내용**: 레벨 100 이후 해금되는 확장 보드. 스탯·범용 기술의 **후기 확장**.
**역할**: 엔드게임 스펙 확장 · 미세 튜닝.
**작동**: 레벨 100 이후 매 레벨당 정복자 포인트 획득.

**구조**:
```
[중앙 보드 — 시작점]
    ↓ (Gate: 게이트 노드로 다음 보드 진입)
[보드 2 — 특화 영역]
    ↓ (Gate)
[보드 3 — 전문화 영역]
    ...
[최대 4~6개 보드 연결 가능]
```

**게이트 노드**:
- 보드 간 연결 지점
- 도달 시 새 보드 진입
- 보드마다 다른 특화 (공격 · 방어 · 유틸)

**레전더리 노드**:
- 각 보드에 1~2개 고유 노드
- 강력한 유일 효과 (예: *"상태이상 스택 × 2"*)
- 빌드 정체성 결정

### 3층 통합 효과

```
[플레이어 스펙]
 = 스킬트리 (전투 스킬)
 + 스피어 보드 (스탯 + 범용 + 크로스오버)
 + 정복자 보드 (엔드게임 확장)
```

이 3층 구조가 **PoE 수준의 빌드 깊이** 를 확보하면서도 유저가 **자기 빌드의 모든 결정을 시각적으로 볼 수 있음**. 게임 수명 영구화 전략.

## 동료 구성 — 인간 4 + 타종족 4 (대표님 확정)

### ⚠️ 중요 수정 (Revision 2)

이전 박제본의 "동료 5명" 구성을 **폐기**. 대표님 확정:

> **동료 = 인간족 4명 + 타종족 4명 = 총 8명.**

이로 인해 파티 구성 복잡성 대폭 증가. 상세 설정은 [[../design/main_character]] 참조.

### 동료 8인 직업 구성 (제안)

**타종족 4명**:
- 엘프: 궁수 / 드루이드 / 성직자 중 고정 + 보조
- 마족: 흑마법사 / 그림자 암살자 / 혈마검사 중 고정 + 보조
- 오크: 광전사 / 샤먼 / 기사 중 고정 + 보조
- 용족: 용기사 / 용마법사 / 고룡술사 중 고정 + 보조

**인간족 4명** (다양한 사회 배경):
- 인간 기사: 방패 전사 고정 + 성직자 보조
- 인간 신관: 힐러 고정 + 마법사 보조
- 인간 마법사: 마법사 고정 + 학자 보조
- 인간 도적: 도적 고정 + 전사 보조

### 동료 직업 = 고정 + 보조 (대표님 확정)

각 동료는 **메인 직업 1개 + 보조 직업 1개** 구조:

```
[예: 엘프 동료 — 궁수 (메인) + 드루이드 (보조)]
    ↓
[메인 직업 스킬트리 — 깊은 전문화]
    궁수: 관통 사격 · 연사 · 저격
    ↓
[보조 직업 스킬트리 — 얕은 서브 옵션]
    드루이드: 자연 치유 · 엔탱글먼트
```

**장점**:
- 각 동료의 **전문성** 보장 (메인)
- **유틸리티** 확장 (보조)
- 유저 빌드 자유도 증가
- 파티 내 역할 중복 방지

### 전투 출전 파티 구성

총 8명 중 **3~4명 출전** (주인공 + 동료 3~4 = 4~5인 파티).

**편성 자유도**:
- 이야기 전개상 필수 동료 배치 (챕터별)
- 자유 슬롯에 유저 선택 동료
- 전투 외에는 모든 8명 관리 가능
- 관계·레벨은 출전 여부와 무관하게 주인공 성장에 연동

## 직업별 방어 시스템 — 8직업으로 확장

### 1. 방패 전사 (Shield Warrior) — 물리 탱커

| 방어 행동 | 효과 |
|---|---|
| 완전방어 | 방패 막기 (100%) |
| 데미지 감소 | 40% 감소 |
| 카운터 | 방패 휘두르기 → 스턴 2턴 |
| 특수 시너지 | 스턴된 적 전체 공격 +30% |

**담당 동료**: 인간 기사 · 오크 (기사 보조) · 용족 (용기사 일부)

### 2. 힐러 / 성직자 (Healer) — 마법 탱커

| 방어 행동 | 효과 |
|---|---|
| 프로텍션 돔 | 파티 전체 원형 완전방어 |
| 디바인 실드 | 단일 + 공격력 +20% |
| 피해 반감 + 힐 | 피격 50% 감소 + 힐 |
| 특수 | 언데드 · 타락체 특효 |

**담당 동료**: 인간 신관 · 엘프 (성직자) · 인간 기사 (보조)

### 3. 도적 (Rogue) — 상태이상 전문

| 방어 행동 | 효과 |
|---|---|
| 트랩 설치 | 역공격 + 상태이상 |
| 피격 증가 | 받는 피해 +20% |
| 카운터 | 암살 치명타 |
| 특수 | 상태이상 × 카테고리 곱연산 |

**담당 동료**: 인간 도적 · 마족 (그림자 암살자)

### 4. 양손 전사 (Berserker) — 공격형 탱커

| 방어 행동 | 효과 |
|---|---|
| 칼날막기 | 완전방어 (100%) |
| 광전사 진노 | 파티 공격 +40%, 자신 방어 -20% |
| 카운터 | 고데미지 반격 |
| 특수 | 주변 아군 공격 +10% |

**담당 동료**: 오크 (광전사) · 인간 도적 (전사 보조)

### 5. 마법사 / 원소술사 (Mage) — 원소 탱커

| 방어 행동 | 효과 |
|---|---|
| 얼음벽 | 완전방어 (마법 강함) |
| 전기 방어 | 경직 (1턴 스킵) |
| 불 방어 | 연소 + 혼란 |
| 특수 | 원소 약점 x2 |

**담당 동료**: 인간 마법사 · 마족 (흑마법사) · 용족 (용마법사)

### 6. 소환사 (Summoner) — 분산형 탱커

| 방어 행동 | 효과 |
|---|---|
| 소환수 피격 대체 | 소환수가 피격 |
| 다중 소환 | 여러 마리 동시 |
| 카운터 | 소환수 반격 |
| 특수 | 어그로 분산 |

**담당 동료**: 용족 (고룡술사) · 인간 학자 (보조)

### 7. 궁수 (Archer) — 원거리 공격형 · NEW

| 방어 행동 | 효과 |
|---|---|
| 회피 사격 | 뒤로 물러나며 원거리 공격 |
| 저격 | 카운터 치명타 |
| 덫 | 행동 지연 |
| 특수 | 원거리 약점 보정 |

**담당 동료**: 엘프 (궁수)

### 8. 드루이드 / 샤먼 (Druid/Shaman) — 하이브리드 · NEW

| 방어 행동 | 효과 |
|---|---|
| 자연 보호막 | 50% 감소 + 힐 |
| 엔탱글먼트 | 적 이동 제한 |
| 원소 변환 | 피해 속성 변환 |
| 특수 | 자연·정령 시너지 |

**담당 동료**: 엘프 (드루이드) · 오크 (샤먼)

## 상태이상 시스템 — 카테고리 × 곱연산

### 4대 카테고리

| 카테고리 | 상태이상 | 효과 |
|---|---|---|
| **A. 물리 약화계** | 출혈 · 독 · 약화 | DoT + 공격력 감소 |
| **B. 제어계** | 스턴 · 기절 · 마비 | 행동 불가 |
| **C. 인지계** | 눈멈 · 혼란 · 공포 | 정확도 · 자체 공격 |
| **D. 원소계** | 연소 · 빙결 · 감전 | 원소 DoT + 약점 |

### 데미지 연산 (Path of Exile 방식)

```
최종 데미지 =
  기본 데미지
  × (1 + 속성 증폭)              ← 합연산
  × (1 + 카테고리 A 스택)         ← 합연산
  × (1 + 카테고리 B 스택)         ← 합연산
  × (1 + 카테고리 C 스택)         ← 합연산
  × (1 + 카테고리 D 스택)         ← 합연산
  × 치명타 배율                    ← 독립 곱연산
```

```
[전투 중 콤보 구간 유지]
    ↓
[콤보 구간별 재료 드랍]
    콤보 31~60 → 희귀 재료 (희귀 장비 제작)
    콤보 61~100 → 에픽 재료 (에픽 장비 제작)
    콤보 101~200 → 전설 재료 (전설 장비 제작)
    콤보 200+ → 신화 재료 (리본 · 최강 방어구)
    ↓
[제작소에서 재료 조합 → 장비 제작]
```

**의미**: 제작 = **전투 숙련도 + 노가다**. 대표님의 "노가다 좋아함" 개인 취향이 전략 게임 깊이로 승화.
