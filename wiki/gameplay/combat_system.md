# 전투 시스템 — 웨이브 턴제 핵앤슬래시 with 패링

> 대표님 브레인스토밍 (2026-04-21) 2차 박제 · Revision 2.
> 설계 헌법 [[../design/world_lore#설계-헌법-design-constitution]] 준수.

## 장르 정체성

**HD-2D 다크 판타지 웨이브 턴제 핵앤슬래시 with 직업별 패링 시스템**

4장르 융합:
- 턴제 JRPG (사고 시간 보장)
- 액션 RPG (단축키 입력 — **반사신경이 아닌 전략적 순서**)
- 소울라이크 (방어 턴 패링)
- 로그라이트 (콤보 배율 · 재료 드랍)

**시장 증명작**: Clair Obscur: Expedition 33 (2025 · Metacritic 90+).

---

## 전투 루프 구조

```
[전투 시작]
    ↓
[웨이브 1 소환 — 잡몹 다수]
    ↓
┌── [주인공 턴 (공격)]
│     전략적 순서로 4~6개 기술 슬롯 입력
│     AP 소진까지 다중 행동
│     → 콤보 카운터 누적 (이 턴 동안만)
│     → 동료 AI 프리셋 자동 연계
│     턴 종료 시 콤보 리셋, 이 턴의 콤보 기록 저장
└────────────────────────
    ↓
┌── [적 턴 (방어)]
│     적 공격 예고 → 방어 메뉴 등장
│     유저: [완전방어 / 완전회피 / 반격] 선택
│     타이밍 입력 → 성공/실패 판정
│     성공: 보상 (턴+/콤보 보너스/데미지+)
│     실패: 피격 + 페널티
└────────────────────────
    ↓
[웨이브 1 전멸 → 웨이브 2 소환]
    ↓
(웨이브 2~3 반복)
    ↓
[전투 종료 — 턴별 최대 콤보 기록 × 보상 배율]
```

## 웨이브 인카운터 시스템

### 1 전투 = 2~3 웨이브 (핵앤슬래시 원칙: 몹 다수 밀도 보장)

| Wave | 구성 표준 | 목적 |
|---|---|---|
| **Wave 1** | 잡몹 10~15마리 | 몸풀기 · 콤보 시동 |
| **Wave 2** | 잡몹 15~25마리 + 강화몹 1~2 | 중간 긴장 · 콤보 유지 |
| **Wave 3** | 잡몹 10~15마리 + 엘리트 1 | 클리어 · 최종 콤보 배율 |

**핵심 원칙** (대표님 확정): 콤보가 턴 종료 시 리셋이므로 **한 턴 안에 쌓을 수 있도록 몹이 충분히 많아야 함**. 핵앤슬래시의 본질.

### 웨이브 전환 연출 (2초 이내)
- BGM 긴장 고조
- 화면 진동
- *"놈들이 더 온다!"* 대사 (동료)

## 주인공 공격 턴 — AP 시스템

### Action Points (AP) 기본
주인공 공격 턴의 행동 자원. 스킬마다 AP 비용 다름. AP 소진 시 턴 종료.

### AP 성장 곡선

| 구간 | AP 풀 | 1턴 처치 가능 몹 수 | 처리 웨이브 수 |
|---|---|---|---|
| 초반 (Lv 1~15) | 3~5 | 3~5마리 | 0.5웨이브 |
| 중반 (Lv 25~40) | 10~15 | 15~25마리 | 1웨이브 |
| 후반 (Lv 50~70) | 25~40 | 50~80마리 | 2~3웨이브 |
| 엔드게임 (Lv 100+) | 80+ | 100+ | 4웨이브 일소 |

### AP 회복 — 다양한 소스 (대표님 확정)

**기본**: 매 턴 시작 시 완전 회복.

**추가 기믹 (콤보 증가 기믹과 통합)**:
- 적 처치 시 AP +1
- 서포터 동료의 서포팅 시 AP +2
- 주인공 자체 기술 발동 시 AP 일부 환급
- 아이템 효과로 공격 성공 시 AP 회복
- 상태이상 부여 성공 시 AP +1

### 장비 희귀도 → AP 확장

| 등급 | AP 최대치 보너스 | 추가 효과 |
|---|---|---|
| 일반 | +0 | - |
| 희귀 (Rare) | +2 | 특수 스킬 슬롯 1 |
| 에픽 | +5 | 특수 스킬 슬롯 2 |
| 전설 | +10 | 고유 스킬 슬롯 1 |
| **역대 신 무기** | +20 | 신의 고유 권능 |

## 스킬 단축키 시스템 — 전략적 순서 (대표님 확정)

### ⚠️ 중요 정정 (Revision 2)

이전 박제본의 "타이밍 리듬 게임화·반사신경 압박" 설계를 **폐기**. 대표님 확정 방향:

> **입력 속도의 물리적 압박은 거의 없음.**
> **콤보 난사 시 효과 낮음.**
> **4~6개 기술 슬롯을 상황에 맞게 순서 배분하는 것이 핵심.**

이건 **반사신경 게임이 아니라 판단력 게임**. 리듬 게임이 아니라 **실시간 전술 게임**.

### 기술 슬롯 시스템

**배정**: 스킬트리에서 전투 스킬 선택 → **4~6번 키에 슬롯 할당**

```
[전투 중 유저 입력]
    1 키: 기술 A (광역 약)
    2 키: 기술 B (단일 강)
    3 키: 기술 C (상태이상)
    4 키: 기술 D (버프)
    5 키: 기술 E (처형)
    6 키: 기술 F (회복·AP 순환)
    ↓
  상황에 맞는 순서로 연속 입력
  예: 1,2,3,4,5 (광역 → 단일 → 디버프 → 버프 → 처형)
  또는: 2,4,5,1,3 (단일 → 버프 → 처형 → 광역 → 디버프)
  ↓
  AP 소진 시 턴 자동 종료 or 유저 수동 종료
```

### 최적 순서는 상황 의존적

콤보 성공 = "몹의 상태" × "동료 기술" × "주인공 기술 간 시너지" 조합.

**상황 변수**:
- 적의 상태이상 현재 스택
- 적의 HP 백분율 (처형 임계치 근접 여부)
- 아군 버프 상태
- 다음 웨이브 예상 구성
- 동료 기술 등록 상태 (현재 프리셋)

**판단력의 깊이**: Chess 의 오프닝 · Last Epoch 의 빌드 시퀀스 · Path of Exile 의 링크 순서. 반사신경이 아닌 **전술적 판단**.

### 스킬 분류

| 종류 | AP 비용 | 용도 | 예시 |
|---|---|---|---|
| **학살 스킬** (광역) | 낮음 (2~3) | 잡몹 웨이브 정리 | 회전베기·화염폭풍 |
| **단일 스킬** (집중) | 중간 (5~8) | 엘리트·강적 집중타 | 처형베기·관통사격 |
| **버프 스킬** | 낮음 (1~3) | 자기 강화 | 광전사 모드 |
| **상태이상 스킬** | 낮음 (2~4) | 디버프 부여 | 독살·눈멈 |
| **궁극기** | 극대 (15+) | 보스·결정타 | 극딜 모드 전용 |

### 처형 임계치 (Execute)
적 HP 20% 이하 시 일반 공격도 즉사. 처형 성공 시 콤보 +2 보너스.

## 콤보 시스템 — 턴 단위 리셋 (대표님 확정)

### 핵심 구조 (Revision 2)

**콤보는 매 턴 0에서 시작**. 턴 종료 시 즉시 초기화.

```
[턴 시작] → 콤보 0
    ↓
[기술 연속 입력]
콤보 1 → 콤보 2 → ... → 콤보 N (해당 턴 최대)
    ↓
[턴 종료]
콤보 N 기록 → 보상 배율 계산 → 리셋 (다음 턴 0부터)
```

**의미**: "한 턴 안에서 얼마나 많이 쌓는가" 가 핵심. 핵앤슬래시의 본질 — 몹 밀도가 곧 콤보 가능성. 몹이 많을수록 콤보도 커짐.

### 콤보 정의 (미확정 — 대표님 결정 대기)

**옵션 A**: 그 턴 가한 공격 횟수 (AP 소비 스킬 수)
**옵션 B**: 그 턴 처치한 몹 수

양자 특성:
- A안: 스킬 선택의 전략성 강조 (다중 공격 스킬 유리)
- B안: 학살 쾌감 강조 (처형 중요)
- **하이브리드도 가능**: 공격 성공 = 콤보 +1, 처치 = 콤보 +2 같은 복합 규칙

### 콤보 증가 기믹 다양화 (대표님 확정)

| 트리거 | 콤보 증가 | AP 회복 |
|---|---|---|
| 일반 공격 성공 | +1 | - |
| 적 처치 | +2 | +1 |
| 상태이상 부여 성공 | +1 | +1 |
| 처형 (HP 20% 이하 즉사) | +2 | +2 |
| 동료 서포트 기술 발동 | +1 | +2 |
| 동료 연계 공격 성공 | +1 | - |
| 특수 아이템 효과 | 변동 | 변동 |
| 방어 턴 완전방어 성공 | +1 | - |
| 방어 턴 반격 성공 | +3~5 | +1 |

**설계 원칙**: 콤보가 단일 수단이 아니라 **다양한 상호작용의 결과물**. 전투의 모든 요소가 콤보 상승에 기여.

### 콤보 구간 보상 배율

| 콤보 (한 턴 내) | 시각 피드백 | 데미지 배율 | 보상 배율 (이 턴분) |
|---|---|---|---|
| 1~10 | 흰색 카운터 | x1.0 | x1.0 |
| 11~30 | 노란색 + 진동 | x1.3 | x1.5 |
| 31~60 | 주황색 + BGM 가속 | x1.7 | x2.0 |
| 61~100 | 붉은색 + 필터 | x2.2 | x3.0 |
| 101~200 | 카메라 흔들림 | x2.8 | x5.0 |
| 200+ | **MASSACRE** 화면 전체 효과 | x3.5 | x10.0 |

### 최종 보상 계산

전투 종료 시 **각 턴의 콤보 기록이 누적** 되어 최종 보상 결정.

```
전투 최종 보상 =
    Σ (각 턴 콤보 × 해당 턴 배율) × 전투 종료 보너스
```

**전략**: 매 턴 최대한 콤보를 쌓되, 여러 턴 연속 고콤보 유지가 최고 보상.

### 콤보 × 희귀 재료 드랍 (대표님 신규 확정)

**제작 시스템과의 결정적 연결**:

| 콤보 구간 | 드랍 가능 재료 |
|---|---|
| 1~30 | 일반 재료 (제작 불가능) |
| 31~60 | 희귀 재료 (매직 장비 제작) |
| 61~100 | 에픽 재료 (에픽 장비 제작) |
| 101~200 | 전설 재료 (전설 장비 제작) |
| 200+ | 신화 재료 (**리본** · 최강 방어구 제작) |

**의미**: 최강 방어구 · 리본 제작 = **고콤보 유지 필수**. 핵앤슬래시의 파밍 동기가 콤보 숙련도와 결합됨. 대표님이 선호하시는 "노가다" 가 전략 게임화됨.

## 동료 AI 연계 시스템 — FF12 간빗(Gambit) 결

### 대표님 확정 구조

**전투 중**: 동료는 **완전 자동**. 단 유저는 **미리 등록한 패턴 중 선택** 가능.
**전투 외**: 동료의 여러 패턴을 **커스터마이징** 가능 (수십 가지 스킬 조합).
**연계 발동**: 주인공 스킬에 따라 동료가 **차례대로 기술 사용**.

### 구조화

```
[전투 외 — 파티 편성 화면]
    ↓
[동료별 패턴 커스터마이징]
    각 동료 3~5개 프리셋 제작 가능:
        - 공격 모드 (딜링 우선)
        - 서포트 모드 (버프 유지)
        - 치유 모드 (회복 우선)
        - 디버프 모드 (상태이상 유지)
        - 커스텀 A/B (유저 자유 조합)
    ↓
[전투 진입 시 프리셋 선택]
    ↓
[전투 중: 유저가 현재 프리셋 전환 가능]
    "공격 모드 → 치유 모드" 토글 (1 AP 소모 또는 자유)
```

### 연계 태그 시스템

주인공 스킬 발동 시 태그 방출:
- `[화염]` `[얼음]` `[전기]` `[관통]` `[광역]` `[근접]` `[원거리]` `[단일]` `[디버프]` `[처형]`

**동료 AI 반응**:
- 주인공 기술 A → 동료 1 반응 기술 (0.3초 후)
- 동료 1 기술 → 동료 2 반응 기술 (0.3초 후)
- **차례대로 연쇄 발동** → 콤보 자연 증가

### 긴급 자동 우선순위 (상시 유효)

유저 프리셋과 무관하게 항상 동작:
- 주인공 HP < 30% → 힐러 자동 치유
- 주인공 디버프 심각 → 성직자 자동 해제
- 동료 HP < 20% → 힐러 해당 동료 응급 치유

## 방어 턴 시스템 — 동료 직업 기반

### 방어 메뉴 3선택지

| 선택지 | 성공 보상 | 실패 결과 | 권장 상황·빌드 |
|---|---|---|---|
| **완전방어** | 무피해 + **아군 턴 +1** | 담당 동료 피격 | 보스·엘리트·안정형 |
| **완전회피** | 무피해 + **콤보 유지** | 콤보 감소 + 피해 | 콤보 빌드·학살 유지 |
| **반격** | 피해 일부 + **강력 카운터 + 콤보 +3~5** | 큰 피해 + 콤보 초기화 가속 | 공격형·저레벨 구간 |

**주의**: 콤보는 턴 종료 시 리셋되므로, 방어 턴에서 받는 콤보 보너스는 **다음 주인공 턴에 승계될지 결정 필요 (대표님 Q 추가)**.

### 패링 타이밍 윈도우

```
적 공격 애니메이션 (총 1.2초)
├─ 예비 동작 (0.6초) — 예측 단서
├─ ⚡ 패링 윈도우
│    ├─ Perfect (0.15초)
│    ├─ Good (0.35초)
├─ 타격 순간 (0.1초)
└─ 후속 동작 (0.35초)
```

### 난이도별 윈도우

| 난이도 | Perfect | Good | 보조 |
|---|---|---|---|
| Easy | 0.25초 | 0.6초 | 화면 가장자리 플래시 (접근성) |
| Normal | 0.15초 | 0.35초 | 적 애니메이션 단서만 |
| Hard | 0.1초 | 0.2초 | 일부 단서 제거 |
| Nightmare | 0.05초 | 0.1초 | 페이크 동작 혼입 |

## 직업별 방어 시스템 — 동료 담당

각 동료의 직업이 방어 방식 결정. 상세는 [[build_system#직업별-방어-시스템]].

### 6대 직업군 방어 요약

| 직업 | 완전방어 | 카운터 | 특수 효과 |
|---|---|---|---|
| **방패 전사** | 방패 막기 (100%) | 스턴 2턴 | 스턴 시 데미지 +30% |
| **힐러 / 성직자** | 프로텍션 돔 (파티) | 언데드 약점 | 피격량 50% 감소 + 힐 |
| **도적** | 트랩 설치 (다중 디버프) | 암살 치명타 | 상태이상 카테고리 × 곱연산 |
| **양손 전사** | 칼날막기 (100%) | 고데미지 반격 | 광전사 진노 (파티 공격 버프) |
| **마법사** | 얼음벽 (100%) | 원소 상성 | 전기 경직 / 불 혼란 |
| **소환사** | 소환수 피격 대체 | 소환수 반격 | 어그로 분산 |

## 반격 2차 선택 시스템

### 반격 성공 → 즉시 기술 선택

```
[패링 성공 판정]
    ↓
[화면 일시 정지 + 주인공 주변 4개 아이콘]
    ↓
[유저 1.5초 내 버튼 선택]
    ↓
[선택 기술 발동]
```

### 반격 기술 4슬롯 (Q/E/R/F)

| 슬롯 | 기술 예시 | 효과 | 최적 상황 |
|---|---|---|---|
| **Q** | 참격 반격 | 단일 고뎀 + 출혈 3턴 | 강적 1:1 |
| **E** | 회선 반격 | 광역 + 콤보 +5 | 다수 잡몹 |
| **R** | 처형 반격 | HP 30% 이하 즉사 확정 | 피니시 |
| **F** | 유선 반격 | 낮은 뎀 + 다음 턴 AP +5 | 자원 회전 |

### 반격 시기적 가치 (레벨별)

| 레벨 구간 | 반격 효율 | 추천 |
|---|---|---|
| 저레벨 (유저 >> 적) | ⭐⭐⭐⭐⭐ | 반격 (무쌍) |
| 동레벨 (유저 ≈ 적) | ⭐⭐⭐ | 완전방어·회피 |
| 고레벨 (유저 << 적) | ⭐ | 완전방어 극단 (극딜 누적) |

## 극딜 게이지 — 주인공 버전

### 구조

완전방어/회피 성공 시 극딜 게이지 +1 누적. 누적은 **턴 단위 콤보와 별개로 전투 내내 유지**.

| 누적 게이지 | 배율 | 활용 |
|---|---|---|
| 3~5 | x2 | 엘리트 몹 |
| 6~10 | x5 | 미니 보스 |
| 11~20 | x10 | 보스 체력 큰 덩어리 |
| 21+ | x20 | 진보스 (엔드게임) |

### 극딜 모드 발동 연출
- 화면 전체 효과 (필터 · 슬로우모션 · BGM 정지)
- 주인공 궁극기 발동
- 궁극기 × 게이지 배율 = 최종 데미지
- 콤보 + 누적 게이지 수치

## 동료 극딜 모드 — 엔드게임 전용 (대표님 신규 확정)

엔드게임 컨텐츠 전용 고급 전술. 상세는 [[endgame#동료-극딜-모드]].

### 컨셉

> *"주인공만 키우면 동료는 알아서 크지만, 턴 몰아주기로 좋아하는 동료를 주인공만큼 강하게 만들 수 있다. 수십 번 공격으로 숨겨진 몹 사살 가능."*

### 시스템

```
[전투 중 특수 전술]
    ↓
[턴 양도 스킬로 모든 동료 AP → 특정 동료 1명에게 집중]
    ↓
[파티 전원이 해당 동료에게 버프 · 적 디버프 · 상태이상 스택]
    ↓
[해당 동료가 한 턴에 수십 번 공격]
    ↓
[숨겨진 몹도 사살 가능]
```

### 전략적 의미

- 플레이어가 좋아하는 동료에게 **감정적 투자**
- 주인공 육성에 집중했으면서도 **각 동료의 극적 순간** 창조
- 엔드게임의 **"내 동료가 빛나는 순간"** 생성
- 동료별 관계 MAX 퀘스트와 연동 (해당 동료 고유 궁극기 해금 = 극딜 모드 정점)

## FF7 스타일 "한 방 최종 보스" — 주인공 버전

본편 최종 보스 재도전 고인물 메타. 상세는 [[endgame#한-방-최종-보스-엔드게임-메타]].

### 준비 페이즈 (5~8턴)

```
[Round 1~6: 준비]
    ├─ 힐러: 주인공 공격력 +200%
    ├─ 양손 전사: 광전사 진노 치명타 +50%
    ├─ 마법사: 적 방어력 -80%
    ├─ 도적: 상태이상 5~7스택
    ├─ 소환사: 동료 전원 → 주인공 턴 양도
    └─ 주인공: 완전방어 대기 (극딜 게이지 누적)
    ↓
[Round 7: 극딜 페이즈]
    AP: 80~100, 버프: x8~15, 상태이상 곱: x5~10, 콤보: x5~10
    ↓
[최종 데미지 = 기본 × 수백 배 → 보스 처단]
```

## 전투 UI 원칙

### 미니멀 — 스피드 우선

- 커맨드 창 작게, 필드 크게
- 애니메이션 중첩 재생
- 속도 설정 (일반/고속/초고속) 유저 토글

- 화면 우상단 큰 숫자
- 구간별 색상 변화
- 턴 종료 시 *"COMBO ENDED: N"* 기록 표시

### 기술 슬롯 UI
- 화면 하단 4~6개 키 아이콘 + 쿨다운 표시
- AP 현재치 · 최대치 큰 숫자
- 상태이상 적용 몹 위에 아이콘 표시
