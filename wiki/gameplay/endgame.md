# 엔드게임 — 히든 보스 · 최강 무기 · 동료 극딜 · 진엔딩

> 대표님 브레인스토밍 (2026-04-21) 2차 박제 · Revision 2.
> 세계관 [[../design/world_lore]] · 스토리 [[../design/story_outline]] · 전투 [[combat_system]] 연계.

## 엔드게임 철학

**목표**: 메인 스토리 엔딩 후의 **수명 확장**. Steam 플레이타임 100시간+ 팬덤 형성.

**참조 성공작**:
- Hades — 로그라이크 + 지속 업데이트
- Stardew Valley — 출시 후 1.5·1.6 확장으로 거대 팬덤
- Terraria — 10년 지속 업데이트
- FF7 / Chrono Trigger — 최강 무기 파밍 + 히든 보스 전통

## 엔드게임 4단계 구조

```
[Phase 3 종결 시점 · Lv 70~85]
     ↓
┌── A. 최종 보스 직행 루트 ─ 정석 엔딩
│     적정 레벨 · 엔딩 A/B/C/D 분기
│     (빌드 기반 칭호)
│
└── B. 엔드게임 파밍 루트 ─ 선택
      ├─ 숨겨진 던전 다수 해금
      ├─ 히든 보스 도전
      ├─ 역대 신 무기 획득
      └─ 목표 Lv 100~120
           ↓
      C. 강화 상태로 최종 보스 재도전
           ├─ "한 방 처단" 가능 (FF7 Knights of Round 결)
           ├─ 진엔딩 해금 (조건 만족 시)
           └─ 보너스 컷신
                ↓
           D. 진엔드게임 (True Endgame)
                ├─ True 히든 보스 (우주 포식자 여왕)
                ├─ Lv 150+ 도전
                └─ 최고 업적 · 칭호
```

## 히든 보스 체계 — 3계층

대표님 설계 기반:

### 최상위 히든 보스 (Tier S · 초난도)

#### 1. 행성의 핵 수호자 (쌍둥이, 2마리)

| 보스 | 전투 스타일 | 컨셉 |
|---|---|---|
| **핵 수호자 α** | 물리 · 속도 · 연속 타격 | 행성의 능동적 자위 기제 |
| **핵 수호자 β** | 마법 · 지연 · 광역 · 환경 활용 | 행성의 반응적 자위 기제 |

**설정**:
- 행성 자체가 파멸 방지 위해 **자가 생성한 방어 기제**
- 의식 있는 존재가 아닌 **반응적 자위 시스템**
- 본편에서 등장하지 않음 — 엔드게임에서만 조우
- 주인공을 "행성을 교란하는 침입자" 로 인식

**서사 의미**: 우주의 생명체가 자기 보존 의지를 가짐. 주인공이 "행성의 의사" 라 주장하지만, 행성 자체는 그를 원치 않을 수도 있다는 철학적 레이어.

**위치**: 차원 균열 깊은 곳 (Phase 3 이후 해금).

#### 2. 우주 포식자 진짜 에이전트 (1마리)

**설정**:
- 본편에서는 **이벤트성 전투만** (실제 치열한 교전 없음)
- Phase 2 에서 주인공 정체 밀고한 에이전트의 **진짜 모습**
- 엔드게임에서 드디어 실제 전투 발생
- 인간 형태 → 이형으로 변신
- 우주 포식자 세력의 **행성 담당 최고위 에이전트**

**전투 스타일**: 하이브리드 · 변칙적 · 행성 밖 존재의 이질성.

**서사 의미**: 본편의 진짜 적의 해결. 단 여왕은 건재 — True Endgame 암시.

### 중간 히든 보스 (Tier A · 중난도)

#### 역대 신의 무기 봉인체 (여러 마리)

**컨셉 (대표님 설계)**:
- 지난 수천 년간 타락했던 **역대 신들**
- 각자 죽어가며 자신의 무기로 **타락한 추종자들을 정화** 하고 함께 봉인됨
- 봉인 지역이 여러 곳에 존재 — 첫 번째 신의 증언
- **무기 = 신의 후회와 속죄의 결정체**

**보스 특성**:
- 위의 Tier S 보스보다는 약함
- 본편 최종 보스보다는 훨씬 강함
- 각자 **서로 다른 무기 · 전투 패턴**
- **최종 무기 드롭 담당**

**서사적 정당화 — 제가 감탄한 부분**:
- 플레이어가 사용하는 최종 무기 = **과거 신의 유언 계승**
- 각 무기마다 고유 역사 · 저주 · 권능
- Dark Souls "보스 무기 = 보스 영혼" 문법 + 감정 레이어 추가

#### 제안 역대 신 무기 보스 5~7기

| 보스 | 무기 | 추정 패턴 | 드롭 |
|---|---|---|---|
| **두 번째 신 — 불꽃의 봉인체** | 화염 대검 | 광역 불 · DoT | 처형자 대검 |
| **세 번째 신 — 서리의 봉인체** | 얼음 지팡이 | 빙결 · 방어 | 파수꾼 지팡이 |
| **네 번째 신 — 탐식의 봉인체** | 독 단검 | 상태이상 극한 | 속박자 단검 |
| **다섯 번째 신 — 광기의 봉인체** | 쌍검 | 하이리스크 패턴 | 광전의 쌍검 |
| **여섯 번째 신 — 균형의 봉인체** | 장검 | 중성 · 적응형 | 조율자 장검 |

(세부 설정은 대표님 결정 대기)

### 특수 히든 보스 (Tier S+ · True Endgame 전용)

#### 우주 포식자 여왕 (1마리)

**설정**:
- Phase 3 에서 처단한 것은 부하 · 에이전트일 뿐
- **진짜 우주적 위협**
- 여러 행성 포식한 전적
- True Endgame 해금 시에만 조우 가능

**조건**:
- 모든 히든 보스 처단
- 진엔딩 분기 해금
- 특정 조건 (모든 동료 관계 MAX + 역대 신 무기 5종+)

**서사 의미**: *"너는 이 행성을 구했다. 하지만 우주는 여전히 내 것이다"* — 시퀄 가능성 암시.

## 역대 신 무기 — 최종 무기 세트

### 컨셉 원칙

**"무기는 배신당한 신의 후회이자 자기 정화의 유언"**

- 각 무기가 한 신의 역사를 담음
- 플레이어 사용 = 과거 신의 뜻을 이어받음
- 사용 시 고유 권능 발현 (AP 보너스 · 특수 스킬 슬롯)

### 빌드별 대응 최강 무기

| 무기명 | 빌드 | 핵심 스탯 | 고유 권능 |
|---|---|---|---|
| **조율자** (장검) | 균형 | 모든 스탯 +25% | 반격 4기술 전부 강화 |
| **처형자** (낫/대검) | 진노 | 공격력 극한 · AP 최대치 +10 | 콤보 보상 배율 +50% |
| **파수꾼** (대방패/지팡이) | 수호 | HP 2배 · 방어 +100% | 극딜 게이지 효율 +100% |
| **속박자** (송곳/단검) | 속박 | 상태이상 확률 +50% | 카테고리 간 곱연산 추가 배율 |
| **광전의 인도자** (쌍검) | 광기 | HP 낮을수록 데미지 폭증 | HP 10% 이하 1턴 무적 |

### 획득 조건 — 빌드 검증 시스템

각 무기는 **해당 빌드의 숙련도 증명** 으로 획득:

| 무기 | 조건 |
|---|---|
| 조율자 | 무명의 방랑자 보스 처단 (균형 빌드 전용) |
| 처형자 | 진노 빌드로 역대 신 A 보스 처단 |
| 파수꾼 | 수호 빌드로 역대 신 B 보스 처단 (무피해) |
| 속박자 | 상태이상 7스택 동시 유지 상태로 역대 신 C 처단 |
| 광전의 인도자 | HP 30% 이하 상태로 역대 신 D 처단 |

**의미**: 각 빌드가 **자신에게 맞는 보스에서 빛남을 증명** 해야 최강 무기 획득. 빌드 정체성 강화.

## 동료 극딜 모드 — 대표님 확정 (Revision 2)

### 컨셉

> *"주인공만 키우면 동료는 알아서 크지만, 턴 몰아주기로 좋아하는 동료를 주인공만큼 강하게 만들 수 있다. 수십 번 공격으로 숨겨진 몹 사살 가능."*

엔드게임 전용 고급 전술. 주인공 중심 육성 원칙을 유지하면서도 **특정 동료를 스포트라이트** 로 끌어올리는 시스템.

### 시스템 구조

```
[엔드게임 전투 특수 전술 — 동료 극딜 빌드]
    ↓
[턴 양도 스킬로 모든 동료 AP → 특정 동료 1명에게 집중]
    ├─ 힐러: "축복의 위임" → 선택 동료에게 AP 양도 + 공격력 +200% 버프
    ├─ 방패 전사: "수호의 희생" → 자신 턴 포기 + 선택 동료 완전방어 1턴 보장
    ├─ 마법사: 적에게 방어력 -80% 디버프
    ├─ 도적: 적에게 상태이상 7스택 도배
    └─ 나머지 동료: 선택 동료에게 턴 몰아주기
    ↓
[선택된 동료: 한 턴에 수십 번 공격]
    AP 극대치 + 버프 + 디버프 + 상태이상 곱연산 + 콤보 배율
    ↓
[숨겨진 몹 · 히든 보스 사살 가능]
```

### 타겟 동료 선택 자유도

플레이어가 좋아하는 동료 누구나 가능:
- 엘프를 궁수 극딜 빌드로 → 원거리 일제 사격
- 마족을 흑마법사 극딜 빌드로 → 대규모 DoT 폭발
- 오크를 광전사 극딜 빌드로 → 근접 무쌍
- 인간 기사를 방패 전사 극딜 빌드로 → 카운터 기반 학살
- 모든 동료가 주연이 될 수 있음

### 선행 조건 — 동료 관계 MAX

각 동료의 극딜 모드 발동 가능 조건:
- 해당 동료 관계 MAX 달성 → 고유 궁극기 해금
- 해당 직업 스킬트리 완성
- 해당 동료 전용 최강 무기 획득

### 서사·정서적 의미

- 플레이어가 **각 동료에게 감정 투자** 유도
- 주인공 중심 육성을 유지하면서도 **각 동료의 극적 순간** 창조
- 엔드게임의 *"내 동료가 주인공만큼 빛나는 순간"* 생성
- **9인 파티 전원의 서사 중요도** 보장 (주인공 독주 방지)

### 전투 예시 — 동료 극딜 빌드

```
[Round 1~5: 준비 페이즈]
    ├─ 주인공: 대기 (완전방어) · AP 쌓기
    ├─ 인간 신관 (힐러): "축복의 위임" → 엘프에게 AP 양도, 공격 +200%
    ├─ 인간 도적 (도적): 적에게 독·출혈·약화 도배
    ├─ 용족 (용마법사): 적 방어력 -80%
    └─ 마족 (흑마법사): 적에게 눈멈·혼란
    ↓
[Round 6: 엘프 극딜]
    엘프 AP 누적 = 40
    공격력 배율 x15 (버프 + 상태이상 + 디버프 + 콤보)
    연속 40회 관통 사격 → 500+ 콤보 달성
    히든 보스 일격
```

**결과**: 엘프 한 명이 주인공급 딜 폭발. 플레이어는 *"내가 사랑한 엘프가 보스를 처단했다"* 는 감정.

### 1인 개발 구현 범위 (Post-Launch 가능)

- 출시 시점: 주인공 극딜 우선 구현
- 출시 + 3개월 패치: 동료 극딜 2~3명 구현
- 출시 + 6개월 패치: 동료 극딜 전 8명 완성
- 출시 + 1년: 동료 간 연계 극딜 (2명 동시 폭발)

## 콤보 × 희귀 재료 드랍 — 제작·노가다 연동 (Revision 2)

### 대표님 확정 구조

**콤보 구간 유지 = 고등급 재료 드랍**. 제작 시스템 ([[build_system#장비-제작-시스템-콤보-드랍-기반]]) 과 결정적 연결.

### 드랍 테이블

| 콤보 구간 (한 턴 내) | 드랍 재료 | 제작 가능 장비 |
|---|---|---|
| 1~30 | 일반 재료 | 제작 불가 |
| 31~60 | 희귀 재료 | 희귀 장비 |
| 61~100 | 에픽 재료 | 에픽 장비 |
| 101~200 | 전설 재료 | 전설 장비 (역대 신 무기 소재 일부) |
| 200+ | 신화 재료 | **리본** · **최강 방어구** |

### 파밍 루프 — 대표님 노가다 취향 반영

```
[엔드게임 던전 진입]
    ↓
[웨이브 전투 — 콤보 고점 유지]
    ↓
[콤보 200+ 달성 → 신화 재료 드랍]
    ↓
[제작소 복귀 → 리본·최강 방어구 제작]
    ↓
[더 강해진 스펙으로 더 높은 던전 진입]
    ↓
[반복 — 파밍 루프]
```

### 핵심 의미

대표님이 선호하시는 **"노가다"** 가 단순 반복이 아니라:
- **콤보 숙련도** = 전략적 순서 + 타이밍
- **빌드 최적화** = 스킬트리 + 스피어 보드 + 장비
- **파티 조합** = 동료 8명 프리셋 튜닝
- 이 모든 것의 통합 노력이 **신화 재료 드랍** 으로 보상됨

파밍이 전략 게임의 최상위 층위가 됨. Path of Exile · Last Epoch 팬층을 조준.

### 재료 종류별 특수 효과

| 재료명 | 등급 | 특수 효과 | 획득 조건 |
|---|---|---|---|
| 타락의 파편 | 희귀 | 상태이상 확률 +5% 부여 | 콤보 31+ |
| 조율의 결정 | 에픽 | 균형 스탯 부여 | 콤보 61+ |
| 봉인의 잔재 | 전설 | 고유 속성 부여 | 콤보 101+ |
| 균형자의 눈물 | 신화 | 리본 제작 전용 | 콤보 200+ |
| **역대 신의 유언** | 신화+ | 역대 신 무기 강화 | 콤보 200+ (봉인지 전용) |

## 진엔딩 해금 조건

### 트리거 조건 (대표님 결정 반영)

진엔딩 루트 해금 필수 조건:
1. **모든 동료 관계 MAX** (5명 전원)
2. **히든 보스 2마리 이상 처단** (Tier A 또는 S)
3. **첫 번째 신(늙은 방랑 학자) 5차 만남 트리거**

### 엔드게임 시점 분기 (대표님 Q5 결정: 엔딩 직전)

본편 최종 보스 처단 직후 새 신 임명 의식 준비 단계에서:
- 일반 엔딩 조건만 충족 → 일반 엔딩 A/B/C/D
- 진엔딩 조건 충족 → 추가 선택지 등장 → 진엔딩 루트

상세 컷신 구성: [[../design/story_outline#진엔딩-악의-신의-탄생-비극의-정점]].

### 중요 원칙 — 설계 헌법 제3조 재확인

진엔딩의 비극성:
- 나이트는 **피해자인 인간** 을 적으로 삼음
- 인간은 신·교회의 피해자 (악이 아님)
- 나이트도 피해자 (균형자로서의 순환에 갇힘)
- 모두가 피해자인 비극 → 진엔딩의 무게

**"눈물 없는 슬픈 눈"** 의 이유 = 모든 감정이 소진된 상태에서의 결단.
