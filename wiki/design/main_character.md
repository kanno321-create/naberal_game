# 주요 캐릭터

> 주인공·동료·중요 NPC 설계. 대표님 브레인스토밍 (2026-04-21) 2차 박제 · Revision 2.
> 세계관 [[world_lore]] · 스토리 [[story_outline]] 연계.

## 캐릭터 구성 원칙

### 설계 헌법 반영
- **제1조**: 주인공 = 유저가 100% 육성·컨트롤 / 동료 = 자동 성장, 직업·프리셋만 커스텀
- **제3조**: 인간도 피해자 → **인간 동료 4명 배치** (인간이 파티의 주요 동맹)

### 파티 구성 확정 (대표님)

> **주인공 1명 + 동료 8명 (인간족 4 + 타종족 4) = 총 9인**
> **전투 출전: 주인공 + 동료 3~4명 = 4~5인 파티**

### 다직업 느낌 구현 (대표님 확정)

> **주인공은 고유 직업이지만, 스킬트리 경로에 따라 여러 직업 같은 느낌으로 전투 가능**

5대 스킬트리 분기 (상세는 [[../gameplay/build_system]]):
- 진노 → 광전사 · 바바리안 느낌
- 수호 → 팔라딘 · 기사 느낌
- 속박 → 강령술사 · 네크로만서 느낌
- 균형 → 룬마스터 · 만능 전사 느낌
- 광기 → 광신도 · 살수 느낌

혼합 빌드도 자유.

### 3중 자아 구조 (Phase 2 전환의 핵심)

Act 2 에서 기억 회복 후 한 몸 안에 3개의 자아가 충돌:

1. **균형자 자아 (본래 자아)** — 수천 년 수행한 냉정한 행성 의사. 임무 지향.
2. **영웅 모험가 자아 (Phase 1 에서 형성)** — 동료와의 우정·사랑. 감정적 존재.
3. **세뇌 잔재 (Phase 1 중 무의식 내면화)** — 교회 교리 파편. 타종족 미묘한 거부감.

**Phase 2 내면 대화 시퀀스** = 세 자아의 말싸움. 유저 선택이 엔딩 분기에 영향.

### 성격 — Phase 별 변화

| Phase | 성격 |
|---|---|
| **프롤로그** | 원래 나이트 — 냉정·초연·판단 지향 |
| **Phase 1 전반** | 기억 상실 직후 — 순진한 호기심형 |
| **Phase 1 후반** | 영웅 모험가 — 동료애, 정의감 |
| **Phase 2 전환** | 3자아 붕괴 · 혼란 · 고독 |
| **Phase 2 후반** | 타종족 연대 · 죄책감 감당 |
| **Phase 3** | 권능 해방 · 모든 종족의 중재자 |
| **일반 엔딩** | 다음 행성으로 떠날 준비 |
| **진엔딩** | 모든 감정 소진된 악의 신 |

### 결정 대기 질문

1. **외형**: 인간형 / 이종족 / 나이 / 성별
2. **무기**: 검 / 쌍검 / 대검 / 창 / 낫 / 특수
3. **복장 변화**: Phase 별 어떻게 변하는가 (평민 → 영웅 → 수배자 → 수호자 → 마왕)

## 동료 8인 — 인간 4 + 타종족 4

### ⚠️ Revision 2 — 동료 확장

이전 박제본의 "동료 5명" 구성을 폐기. 대표님 확정 = **인간 4 + 타종족 4 = 총 8명**.

### 직업 구조 (대표님 확정)

각 동료는 **고정 메인 직업 + 보조 직업** 구조. 메인은 깊게, 보조는 얕게.

### 타종족 동료 4명

#### 동료 1 — 엘프

| 항목 | 설정 |
|---|---|
| 이름 | 미확정 |
| 메인 직업 | 궁수 (Archer) |
| 보조 직업 | 드루이드 또는 성직자 |
| 전투 포지션 | 원거리 관통 공격 · 회피 사격 · 자연 지원 |
| 합류 시점 | Phase 1 초반~중반 |
| 서사 톤 | 예언자 · 마지막 왕실 수호자 |

**컨셉**: 인간 왕국의 타종족 학살을 목격한 증인. Phase 2 반전을 가장 아프게 받아들임. 예언 능력으로 첫 번째 신과의 교류 가능성.

#### 동료 2 — 마족

| 항목 | 설정 |
|---|---|
| 이름 | 미확정 |
| 메인 직업 | 흑마법사 (Warlock) |
| 보조 직업 | 그림자 암살자 (Shadow Assassin) |
| 전투 포지션 | 광역 DoT · 상태이상 · 그림자 |
| 합류 시점 | Phase 1 중반 |
| 서사 톤 | 고립된 마족 부족의 지혜 계승자 |

**컨셉**: 교회 *"마족 = 악"* 교리 때문에 가장 먼저 박해받은 생존자. 인간에 대한 복잡한 감정 (혐오가 아닌 슬픔).

#### 동료 3 — 오크

| 항목 | 설정 |
|---|---|
| 이름 | 미확정 |
| 메인 직업 | 광전사 (Berserker) |
| 보조 직업 | 샤먼 (Shaman) |
| 전투 포지션 | 전방 돌진 · 파티 공격 버프 · 자연 정령 |
| 합류 시점 | Phase 1 후반 |
| 서사 톤 | 초원 전사 부족 족장 후계자 |

**컨셉**: 오크는 실은 명예 중심 문화. *"야만인"* 프레이밍을 뒤집는 캐릭터. 주인공과 직설적·신뢰 관계.

#### 동료 4 — 용족

| 항목 | 설정 |
|---|---|
| 이름 | 미확정 |
| 메인 직업 | 용기사 (Dragon Knight) |
| 보조 직업 | 고룡술사 (Draconic Summoner) |
| 전투 포지션 | 대각선 브레스 · 장거리 · 소환 |
| 합류 시점 | Phase 1 말기 또는 Phase 2 초기 |
| 서사 톤 | 수백 년 산 고룡의 인간화 / 용의 피 계승자 |

**컨셉**: 가장 늙은 종족. **과거 균형 순환을 어렴풋이 기억**. 첫 번째 신의 존재를 어렴풋이 감지.

### 인간족 동료 4명 (설계 헌법 제3조의 살아있는 증명)

인간이 파티의 주요 동맹. 각자 다른 사회 배경으로 **"인간 = 악이 아님"** 입증.

#### 동료 5 — 인간 기사

| 항목 | 설정 |
|---|---|
| 이름 | 미확정 |
| 메인 직업 | 방패 전사 (Shield Warrior) |
| 보조 직업 | 성직자 (Priest) |
| 전투 포지션 | 탱킹 · 스턴 유발 · 성스러운 보호 |
| 합류 시점 | Phase 1 중반 |
| 서사 톤 | 왕국 기사단 출신 · 교회 교리 의심 |

**컨셉**: 명예로운 기사지만 왕실의 타종족 학살 작전을 목격하고 의문 품기 시작. Phase 2 에서 탈영해 주인공 합류. 명예의 대가 = 조국 배반의 고통.

|---|---|

| 전투 포지션 | 상태이상 · 암살 · 근접 반격 |
| 합류 시점 | Phase 1 중~후반 |
| 서사 톤 | 도시 하층민 · 왕국 저항 조직 |

**컨셉**: 사회 밑바닥에서 진실을 가장 먼저 체감한 인물. 왕국의 억압 · 교회의 선전에 맞선 저항 조직 소속. Phase 2 이전부터 타종족에 호의적.

### 동료 관계도 시스템

**진엔딩 트리거 조건 중 하나 = 모든 동료 관계 MAX (8명 전원)**.

- 챕터 진행 중 동료와의 사이드 퀘스트
- 캠프에서 대화 이벤트
- 동료 개인 사연 해결 (복수·구출·치유 등)
- 관계 MAX 시 **동료 고유 궁극기** 해금 → [[../gameplay/endgame#동료-극딜-모드]] 에 필수

### 동료 AI 프리셋 커스터마이징 (대표님 확정)

**전투 외**: 각 동료 3~5개 프리셋 제작 가능 (수십 가지 스킬 조합).
- 공격 모드 / 서포트 모드 / 치유 모드 / 디버프 모드 / 커스텀 A·B
- 유저 자유 스킬 조합
- 각 프리셋당 기술 등록 순서 결정

**전투 중**: 미리 등록한 프리셋 중 선택 (실시간 개조 불가, 프리셋 전환만 가능).

## 중요 NPC

### 첫 번째 신 — 늙은 방랑 학자 (The First One)

> *"이 행성은 너무 오래되었다네. 나는 여섯 번째 균형을 보았지. 이번이 일곱 번째일까, 여덟 번째일까. 역사는 반복된다. 봉인된 것은 사라지지 않아. 어딘가에서 그들은 기다리지."*

상세 설정: [[world_lore#첫-번째-신-the-first-one]].

**요약**:
- 이 행성의 **최초의 신** · 유일하게 타락하지 않은 신
- 타락 초기 증상을 자각 → **스스로 권좌 포기** → 인간계로 하강
- 수천 년 평범한 방랑자로 살며 약자 보조
- **인간계 마법 체계 발전에 공헌** (인간 마법사 동료의 스승 계보)
- 역대 신들의 모든 타락·처단·봉인을 눈으로 지켜봄
- 주인공의 능력·인성을 보고 행성 미래를 맡기고 싶어함

### 등장 시점

- **1차 (Phase 1 초반)**: 시작 마을 술집 · 잡담 수준
- **2차 (Phase 2 중반)**: 재회 · 주인공 정체 암시
- **3차 (Phase 3 진입)**: 역대 신 봉인지 첫 힌트
- **4차 (엔딩 직전)**: 진엔딩 분기 트리거 · **정체 완전 공개**
- **5차 (엔드게임)**: 역대 신 봉인지로 안내

### 외형
흰 수염의 노인 · 낡은 로브 · 지팡이 · 평범해 보임. 정체 공개 직전 드러나는 미세한 신성.

### 타락한 현재 신 — 최종 보스 (Phase 3)

- **정체**: 이 행성의 일곱 번째 신 (또는 여덟 번째)
- **타락 양상**: 권능 중독 극기 · 인간 번식 과잉 유도 · 타종족 학살 묵인
- **처단 직전 대사 (제안)**: *"드디어… 쉴 수 있겠군…"* — 악이 아닌 피로감
- **무기**: 처단 시 플레이어에게 획득 (역대 신 무기 세트)

### 교회 수뇌부 (Phase 2 중간 보스)

- **교황**: 권능 중독 극기 · 신의 의지 빙자, 실제로는 권력 집착
- **광신도 사제단**: 세뇌된 신도 (공포가 아닌 광기)
- **이단 심문관**: 교리 조작 실행자
- **양심적 사제 (NPC)**: 비밀 저항 · 인간 신관 동료 합류 매개

### 우주 포식자 에이전트 (Phase 2 전환 이벤트)

- **외형**: 인간 형태 잠복 → 정체 드러날 때 이형 변신
- **역할**: Phase 2 주인공 정체 밀고자
- **본편 전투**: 이벤트성 전투만 (실제 교전은 엔드게임)

### 왕국 핵심 인물들

- **왕 / 왕비 / 왕자 / 왕녀**: Phase 1 지배층 · Phase 2 수배령 선포
- **왕실 기사단장**: Phase 2 추격 책임자. 인간 기사 동료의 과거 전우일 가능성
- **학자·마법사 협회장**: 인간 마법사 동료의 스승 (첫 번째 신 간접 계승)

## 캐릭터 디자인 우선순위 (1인 개발 공수)
