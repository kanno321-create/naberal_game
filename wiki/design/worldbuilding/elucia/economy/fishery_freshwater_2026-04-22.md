## 2. 핵심 거점 상세

### 2-1. Lonwyn 호수군 (Aldric 왕국) — 담수 어업의 중심

Lowen 강이 Lonwyn 분지로 유입되며 형성된 대호수군. Aldric 왕국의 주요 경제 기반으로, 어업·수운·호변 염전이 복합적으로 발달한다.

| 항목 | 내용 |
|------|------|
| 주요 호수 | Lonwyn Lake (최대 ~8,000 km²) + 4~7개 중소호 (추정) |
| 어업 방식 | 그물 어업·낚시·어살 (추정) |
| 산출물 | 잉어·뱀장어·수조류(오리·거위) |
| 가공 | 훈제·소금 절임·건조 |
| 교역 | 내륙 성좌국·Sylren 방면으로 운반 |

**Aldric 왕국 경제 구조**:
- 담수 어업 + 호변 목축(양·거위) + 내수 수운이 3기둥
- 해안이 없는 내륙 왕국임에도 수운 네트워크로 교역 활발

### 2-2. Loravel 습지 (Ceren 왕국) — 뱀장어와 소금의 복합 경제

Loravel 습지는 뱀장어 서식지로 유명하며(추정), Ceren 왕국 어부들이 계절별로 이동하며 채취한다. 조석 영향을 받는 구역에서 소금 채취도 병행된다.

| 항목 | 내용 |
|------|------|
| 주요 어종 | 뱀장어·붕어·갈대 사이 소형 어류 |
| 특이점 | 습지 내부 섬 깊숙이 들어가야 대어(大魚) 포획 |
| 채취 방식 | 평저선(平底船)·대나무 어살·그물 |
| 연계 산업 | 갈대 채취(지붕재)·약초·이탄 채취 복합 |

### 2-3. 대하천 어업 — 내륙 도시 단백질 라인

## 4. 이름 모를 학자 마법과 담수 어업 (Q-CORE 2 연동)

서민 강변 어업 마을에 "이름 모를 늙은 학자" 의 흔적이 간간이 전해진다:

- 물고기 보존에 쓰이는 **식품 냉각 주문** 이 어느 강변 마을에서 구전으로 퍼져 있다 (간접 단서)
- 어부들은 이것이 어디서 왔는지 모르고 "옛날 어부 어른이 가르쳐줬다" 고만 알고 있다
- 교회 신관은 이 주문이 공식 신학에 없음을 알아채나 규모가 작아 방치하는 곳도 있다 (추정)
