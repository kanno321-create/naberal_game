# 주인공 출신 마을 후보 — Silvan 서부 변경림

> **대표님 최종 선택 필수**: 20개 후보 중 1개 확정. 평가 기준: 이름 느낌 · 분위기 · 스토리 포텐셜.
> 확정 후 Wave 3 에서 상세 마을 파일 생성.

## 선정 기준 (Wave2 자체 기준)

| 기준 | 설명 |
|------|------|
| **고립감** | 세상으로부터 충분히 멀어야 함 (주인공 출발 동기) |
| **이름 음감** | Set B 원칙: Latin+Germanic+Celtic 혼합, 발음 쉬움 |
| **분위기 독창성** | 기존 JRPG 출신 마을 클리셰 탈피 |
| **스토리 포텐셜** | 마왕 군대가 처음 나타나거나, 비밀이 있을 것 같은 느낌 |
| **Silvan 생태 적합성** | Silvan Forest 또는 Silvan 접경 서부 지대 |

#### 1. **Ashenveil** (애신베일)
| 항목 | 내용 |
|------|------|
| **어원** | *ashen* (잿빛) + *-veil* (안개 장막) |
| **위치** | Silvan 서쪽 끝 · 안개 잦은 저지대 |
| **인구** | 약 150~200 |
| **산업** | 버섯 채취·목탄·소규모 수렵 |
| **분위기** | 항상 안개. 낮에도 어두운 숲 그림자. 마을 전체가 회색 빛. |
| **스토리 포텐셜** | "안개 속에서 온 것들" — 마왕 군대의 첫 출현 지점으로 최적. 안개가 실제 설정과 연결 가능 |
| **추천 이유** | 이름 음감이 독특하고 음울하면서도 아름다움. FF7 분위기 |

### ★★ 차선 추천 (8개)

---

#### 6. **Elmcross** (엘름크로스)
*elm* (느릅나무) + *-cross* (교차로) · 느릅나무 아래 교차로 마을 · 행상인들이 지나치는 곳 · 인구 300~450 · 평범하지만 다양한 정보가 모이는 마을

#### 7. **Oakveil** (오크베일)
*oak* (참나무) + *-veil* (장막) · 거대한 참나무들이 하늘을 가리는 마을 · 빛이 부족하지만 여름엔 초록 빛 천국 · 인구 200~350

#### 8. **Thornwick** (쏜위크)
*thorn* (가시) + *-wick* (취락) · 가시덤불 방어선으로 둘러싸인 마을 · 자기 방어가 강한 마을 문화 · 인구 180~280

#### 9. **Greenveil** (그린베일)
*green* (초록) + *-veil* (장막) · Silvan 외곽 초원과 숲이 만나는 경계 · 두 세계가 공존 · 인구 350~500

#### 10. **Brookhollow** (브룩할로우)
*brook* (시냇물) + *-hollow* (골짜기) · 시냇물 소리가 끊이지 않는 골짜기 마을 · 고요하고 평화로운 분위기 · 인구 150~250

#### 11. **Willowmere** (윌로우미어)
*willow* (버드나무) + *-mere* (연못) · 버드나무 연못가 마을 · 슬픔과 아름다움이 공존 · 인구 200~300

#### 12. **Darkfen** (다크펜)
*dark* (어두운) + *-fen* (늪) · Silvan 남부 늪지대 · 외부인이 자발적으로 오지 않는 마을 · 인구 100~150 · 극한 고립감

#### 13. **Silverbrook** (실버브룩)
*silver* (은빛) + *-brook* (시냇물) · 달빛에 은빛으로 빛나는 시냇물의 마을 · 낮보다 밤이 아름다운 역설적 마을 · 인구 250~380

*mire* (진흙 수렁) + *-wick* · 진흙 습지 취락 · 가장 어두운 분위기 · 인구 80~150

#### 15. **Coldwick** (콜드위크)
*cold* (차가운) + *-wick* · 서늘한 Silvan 고지 취락 · 주민들이 침묵에 익숙 · 인구 150~250

#### 16. **Ashwick** (애시위크)
*ash* (재) + *-wick* · 화재 후 재건된 마을 · 과거의 상처를 안고 사는 공동체 · 인구 120~200

#### 17. **Fernwick** (펀위크)
*fern* (고사리) + *-wick* · 단순하고 소박한 고사리 마을 · 평범함의 극치 · 인구 200~350

#### 18. **Mosswick** (모스위크)
*moss* (이끼) + *-wick* · 이끼 취락 · Mossford(Oryn)와 유사하지만 숲 내부 · 인구 150~250

#### 19. **Gloomveil** (글룸베일)
*gloom* (어둠·우울) + *-veil* (장막) · 이름 자체가 직접적으로 어두움 · 과감한 네이밍 · 인구 100~180

#### 20. **Dawncroft** (던크로프트)
*dawn* (새벽) + *-croft* (경작지) · Dawnwick의 변형 · 더 소박하고 농촌적 · 인구 200~300

- **발음하기 쉬운**: Graymoss · Dawnwick · Elmcross
- **독특한 음감**: Ashenveil · Misthollow · Fernhollow
