## 요약

성좌국 음식 문화는 교회 달력과 불가분하게 연결된다. 금식일·성찬일·축제일에 따라 먹는 음식이 달라지며, 제단 빵(성체빵)은 종교적 의미와 실용적 의미를 동시에 갖는다. 교황청은 금식일 규정을 통해 식생활까지 통제하는 종교 권력을 행사한다.

---

## 음식 체계

### 1. 제단 빵 (Communion Bread / Panis Sacer)

| 항목 | 내용 |
|------|------|
| 용도 | 미사 성찬 의식 + 일상 식량 보조 |
| 재료 | 순수한 밀가루 + 소금 + 정화수 (성수 의무) |
| 제조 | 수도원 전담 제빵 (평민 자체 제조 금지) |
| 배급 | 교구 성당에서 매주 일요일 신도에게 소량 배분 |
| 상징 | "신의 몸을 나눈다" — 교회 통제의 음식판 발현 |
| Q-CORE 단서 | 일부 마을에서 "어떤 노인이 제단 빵 반죽을 가르쳐줬는데 더 오래 보관된다"는 소문 산발적 (간접) |

---

### 2. 금식일 음식 (Fasting Days)

교황청 달력 기준 연간 약 40~60일 금식 규정 (추정).

| 금식 유형 | 허용 음식 |
|---------|---------|
| 완전 금식 (대금식) | 물만 허용 (연간 수 회 · 중요 절기 전날) |
| 부분 금식 | 생선·채소·과일 · 육류·유제품 금지 |
| 절제일 | 육류 금지 · 생선·빵·소금 허용 |

금식일 주요 음식:
- **소금 절인 생선**: Irondelta·Mirevane 해안 어획물
- **채소 스튜**: 무·당근·양배추 + 허브 (교회 수도원 정원 재배)
- **렌즈콩 죽**: 가난한 순례자 금식일 주식
- **발효 채소**: 소금 절임 무·배추류

---

### 3. 성찬·축제 음식 (Feast Days)

| 축제 | 대표 음식 |
|------|---------|
| 수호 성인 Aurel 축일 | 황금색 꿀빵 + 구운 돼지 + 포도주 |
| 승천절 | 흰 밀가루 케이크 + 훈제 고기 + 향신료 포도주 |
| 수확 감사제 | 첫 수확 밀 빵 + 사과주 + 지역 특산 요리 |
| 광명 축제 | 오트밀 크레페 + 꿀 + 구운 밤 |

---

### 4. 계층별 일상 식사

| 계층 | 일상 식사 |
|------|---------|
| 교황·추기경 | 다코스(고급 고기)·향신료 와인·정제 밀빵·해외 수입 과일 |
| 공작·귀족 | 구운 육류·밀빵·포도주·채소 요리 (향신료 사용) |
| 신관 | 수도원 공동식 — 간소한 채소 스튜·빵·물 (절제 원칙) |
| 상인·장인 | 밀빵·보리죽·소금 생선·계절 채소·맥주 |
| 평민·농민 | 보리죽·순무·소금·가끔 소금 생선·빵 (금식일엔 더 제한) |
| 순례자 | 렌즈콩·딱딱한 빵·건조 과일·수도원 배급품 |

| **향신료 마늘 빵** | 이단 퇴치 미신 (마늘 = 악마 쫓기) 관련 가정 상비 음식 |

## 음식과 권력

- **금식일 통제** = 교황청이 서민의 식사 방식까지 규율
- **제단 빵 독점** = 종교 공급망 독점으로 신앙 통제
- **포도주 과세** = 교황청 포도주 인가세 → 양조장 통제
