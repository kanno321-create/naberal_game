## 요약

성좌국 의상은 계층에 따라 극명하게 구분된다. 교황의 흰색·황금·자주색 삼색 법의에서 추기경 자주→신관 검정→평민 거친 삼베 순으로 내려갈수록 단순해진다. "신에 가까울수록 화려하다"는 신학적 의상 원리가 성좌국 의상 문화의 기반.

## 계층별 의상

### 교황 (Pontifex)

| 장면 | 의상 |
|------|------|
| 공식 의전 | 흰색 긴 법의(Cassock) + 황금 수놓은 자주색 외투(Cope) + 3층 교황관(Tiara) + 황금 십자가 목걸이 |
| 미사 집전 | 흰색 Alb + 황금 Stole + 자주색 Chasuble |
| 일상 | 흰색 Cassock + 황금 패딩 + 보라 두건(Zucchetto) |
| 발 | 붉은 슬리퍼 (교황 특권 상징) |

### 추기경 (Cardinal)

| 장면 | 의상 |
|------|------|
| 공식 | 자주색 긴 법의 + 보라 망토 + 추기경 넓은 모자(Galero 자주색) |
| 일상 | 자주색 Cassock + 자주색 Zucchetto |
| 행렬 | 자주색 예복 + 황금 십자가 패용 |

### 대주교·주교 (Archbishop / Bishop)

| 직위 | 의상 |
|------|------|
| 대주교 | 보라색 법의 + 대주교관(Mitre) + 보라 망토 |
| 주교 | 보라+검정 법의 + 주교관 + 주교 지팡이(Crosier) |

### 일반 신관 (Priest)

- 기본: 검정 긴 Cassock + 흰색 Surplice (미사 시)
- 겨울: 검정 Cassock 위 검정 두꺼운 망토
- 머리: 삭발 또는 짧은 머리 (신관 표시)
- 금속 십자가 목걸이 (교황청 공인 양식만 허용)

### 성기사 (Knight)

- 백색 풀 플레이트 아머 (Sacred Choir)
- 황금·보라 하프 플레이트 (Aurelian Light)
- 은색 체인메일 (Veil Warden)
- 모두 보라 망토 착용 (성좌국 소속 표시)

### 귀족 (공작·백작)

- 교황청 의전: 보라 테두리 귀족 코트하디 + 가문 문장 브로치
- 일상: 고급 모직 긴 가운 (보라·금·적색 사용 제한 — 교황청 허가 가문만 가능)
- 공작: 보라 외투 착용 가능 (교황 직접 하사 시)

### 서민·농민 (Commoner)

- 삼베·양모 거친 천 단순 튜닉 + 허리띠
- 색: 갈색·회색·자연색 (보라·황금·자주 착용 금지)
- 겨울: 양가죽 외투
- 모든 성인은 교회 방문 시 두건 착용 관습

### 수도원 수사·수녀 (Monk / Nun)

- 수사: 검정 튜닉 + 흰색 두건(Cowl) + 노끈 허리띠 (청빈 상징)
- 수녀: 흰색 습의 + 검정 두건 + 검정 스카풀라

## 색 위계 규칙

| 색 | 허용 계층 |
|----|---------|
| 흰색(법의) | 교황 독점 |
| 자주·보라 | 추기경 이상 · 교황 하사 공작 |
| 황금 자수 | 교황·추기경 공식 의전 |
| 은색 | 주교·기사 일부 |
| 검정 | 신관 기본 |
| 갈색·회색 | 평민·수도원 |

---

## 의상의 집필 활용 묘사 포인트

- 보라색 법의 가두리 → 즉각 고위 성직자 인식
- 검정 긴 법의 + 머리 삭발 → 일반 신관
- 백색 풀 플레이트 → Sacred Choir 교황 근위 기사
- 거친 삼베 튜닉 + 두건 → 성지 순례자 (평민)
