# Thornhallow (손할로우)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Orinthal 동부 · Oryn 왕국 접경 숲 경계 |
| 인구 | 약 200~350 |
| 기능 | 삼림 벌목·수렵 거점 마을 |
| 특징 | 가시나무 울타리로 마을 경계. Oryn 왕국 숲으로의 접근을 통제 |
| 어원 | *thorn* (가시) + *hollow* (속이 빈 골짜기) = "가시 골짜기" |
| 교회 | 목조 소교회 · 격주 방문 신부 |
| 특기 | 타종족 목격 신고가 Orinthal 공작령에서 가장 많은 마을. 이단심문청 정기 순찰 대상 |
