# Varemoor (바레무어)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Veldenmere · Norvend 기슭 광산 지구 |
| 인구 | 약 600~900 |
| 기능 | 광산 인부 거주 + 광석 1차 가공 |
| 특징 | 성좌국 내 최대 광산 인부 집결 마을. 분진·소음·험한 환경. 신관 상주가 어려워 격주 방문 |
| 어원 | *vare* (광석 원석 · 내부 조어) + *-moor* (황무지) = "황무지 광석 마을" |

| 특기 | Uldenmass 공작 직할 광산 노동자 주거지. 광산 사고 은폐 의혹 (추정) |
