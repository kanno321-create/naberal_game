# Millcross (밀크로스) — 방앗간·교차로 취락

| 항목 | 내용 |
|------|------|
| **위치** | Auravel 강 중류 · 성좌국 남부 대로 교차점 |
| **인구** | 약 600~900 |
| **산업** | 수력 방앗간 5기 운영 · 인근 농장 곡물 도정 |
| **분위기** | 장날이면 북적이나 평소는 조용. 방앗간 소리가 끊이지 않음 |
| **어원** | *mill* (방앗간) + *-cross* (교차로) |
| **특징** | 순례자·상인 행렬이 지나는 교통 요충 소촌 |
