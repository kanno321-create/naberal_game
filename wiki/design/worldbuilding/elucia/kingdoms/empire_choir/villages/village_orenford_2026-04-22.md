# Orenford (오렌포드)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Orinthal · Aurion Divide 서쪽 기슭 |
| 인구 | 약 500~700 |
| 기능 | Aurion 평원 → Orinthal 삼림 진입 관문 |

| 어원 | *oren* (황금빛 나무 · 내부 조어) + *-ford* (여울 건너는 곳) |

| 특기 | Veloris 아카데미 학자들이 표본 채집을 위해 방문하는 마을 |
