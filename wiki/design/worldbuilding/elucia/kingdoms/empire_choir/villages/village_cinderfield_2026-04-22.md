# Cinderfield (신더필드)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Aurionmere 남부 · 교황군 훈련 공터 인근 |
| 인구 | 약 250~400 |
| 기능 | 교황군 보급 농촌 (곡물·채소 근거리 납품) |
| 특징 | 교황군 훈련장 인근이라 신관·기사 방문 잦음. 마을 주민 대부분이 교황군 보급 계약자 가족 |
| 어원 | *cinder* (재) + *field* (들판) = 과거 화전 개간 들판 |
| 교회 | 교황군 전용 예배당 별도 + 마을 교구 성당 분리 운영 |
| 특기 | 이단 혐의자 신고율 성좌국 내 상위권 (교황군 영향) |
