# Saltwick (솔트윅)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Mirevane · Eloryn 강 하류 삼각주 인근 |
| 인구 | 약 600~900 |
| 기능 | 소규모 염전·소금 생산 |
| 특징 | Ceren 왕국의 소금 독점에 대항하는 성좌국 자체 소금 생산 거점 (소량이나 상징적 의미). 교황청 소금 공급 일부 담당 |
| 어원 | *salt* + *-wick* (집락·작은 마을) = "소금 마을" |
| 교회 | 염전 인부 거주 구역 내 소교회 |
| 경제 | Ceren 소금 대비 품질 낮으나 성좌국 내수 공급 |
