# Hollowthorn (할로우손)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Orinthal · Oryn 왕국 접경 삼림 깊숙이 |
| 인구 | 약 150~250 |
| 기능 | 수렵·채집·약초 채취 |
| 특징 | 성좌국에서 가장 외딴 교구 마을 중 하나. 신관이 1년에 2~3회만 방문. 주민들이 교황청과 사실상 격리됨 |
| 어원 | *hollow* (속이 빈) + *thorn* (가시나무) = "텅 빈 가시나무 마을" |
| 교회 | 오래된 목조 소교회 (보수 방치) |
| 특기 | 양심파 잠복 비율 상대적 높음 가능 (추정 · 대표님 미확정). 타종족 목격 신고 없음 — 은폐 or 미접촉 불명 |
| Q-CORE 단서 | 마을 어귀 오래된 나무에 "불꽃 없이 불을 피우는 법"이라는 긁힌 글씨가 있다. 언제, 누가 새겼는지 모름. (Q-CORE 2 간접) |
