# Aulmere (올메어)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Solanthen · Auravel 강 상류 농경지 |
| 인구 | 약 300~500 |
| 기능 | 밀·보리 농업 마을 |
| 특징 | Soltharr 공작 직할 대농장 인근. 소작농 공동체. 수확 감사제 전통 가장 성실히 지키는 마을 |
| 어원 | *aul* (라틴 aulis = 마당·안마당) + *-mere* (연못·강) = "마당가 강변 마을" |
| 교회 | 석조 교구 성당 · 수확 봉헌 제단 정성스러움 |
| Q-CORE 단서 | "작년 가뭄 때 이름 모를 노인이 우물 정화 주문을 가르쳐줘서 수확이 가능했다"는 주민 증언. 대교구 보고서 미기재 (Archbishop Vorn 관할 지역). (Q-CORE 2 간접) |
