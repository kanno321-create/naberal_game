# Pelborne (펠보른)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Loranthas · Via Sacra (순례 대로) 상 Lumstow 남쪽 20km |
| 인구 | 약 400~700 |
| 기능 | 순례자 중간 숙박·음식 공급 |
| 특징 | 마을 전체가 순례자 서비스 경제로 운영. 여관 5개·빵집·약초 상인 밀집 |
| 어원 | *pel* (라틴 peregrinus 파생 · 순례자) + *-borne* (운반·흐름) = "순례자의 흐름 마을" |
| 교회 | Via Sacra 변 대형 교구 성당 · 항상 열려있는 "열린 성당" |
| 특기 | 성인 Aurel 축일 때 Solaris 입성 전날 대규모 순례자 야영지 |
