# Duskbell (더스크벨)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Loranthas · Lowen 강변 소구릉 |
| 인구 | 약 300~450 |
| 기능 | 호수 어업·수도원 보급 마을 |
| 특징 | 마을 언덕 위 종탑이 매일 해 질 때 종을 15분 울리는 전통. "황혼의 종소리" 는 Loranthas 공작령 상징 중 하나 |
| 어원 | *dusk* (황혼) + *bell* (종) = "황혼 종 마을" |
| 교회 | 구릉 위 종탑 소교회 · 수도원 부속 농원 인근 |
| 특기 | 인근 수도원 (베일 수호 기사단 파견 소대 주둔)으로 가는 보급 마을 |
