# Caldenmere (칼덴메어)

| 항목 | 내용 |
|------|------|
| 위치 | Duchy of Mirevane · Eloryn 강 중류 서안 |
| 인구 | 약 400~600 |
| 기능 | 강 어업·나룻배 도하 서비스 |
| 특징 | 강이 항상 차갑고 맑음. 연어·청어 주산지. 수확 감사제 때 생선 봉헌 관습 |
| 어원 | *cald* (게르만어 kalt = 차가운) + *-mere* (연못·강) = "냉수 강변 마을" |
| 교회 | 강변 석조 교구 성당 · 어부 수호 성인 봉헌 |
| 특기 | Irondelta 항구로 가는 강 수운 중계 거점. 나룻배 선주 가문 Caldver 가 대대로 운영 |
