# Aurewatch (오레워치) — 성좌국 북부 감시 요새

## 요약

Aurion 평원 북단, Greygate Pass 남방 도로와 Eloryn 강 상류 사이의 요충지에 건설된 감시·방어 도시. Thaloss 왕국과의 경계를 통제한다.

---

| 항목 | 내용 |
|------|------|
| **위치** | Aurion 평원 북단 · Greygate Pass 남쪽 도로 30km 지점 |
| **규모** | 인구 약 15,000~20,000 |
| **기능** | 북방 감시·관세 징수·교황군 주둔 |
| **어원** | *aure* (황금빛 감시) + *-watch* (감시·경계) = "황금 감시탑" |
| **지형** | 평원 위 인공 구릉에 건립 · 사방 시야 확보 |
| **분위기** | 군기 넘치는 군사 도시. 상업보다 안보가 우선 |
