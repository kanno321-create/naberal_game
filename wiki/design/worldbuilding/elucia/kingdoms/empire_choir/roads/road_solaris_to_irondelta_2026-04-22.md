# Solaris → Irondelta 서부 교역 가도

| 항목 | 내용 |
|------|------|
| 정식명 | Via Commercialis Occidentis (서부 교역 가도) |
| 구간 | Solaris → Ashford → Caldenmere → Irondelta |
| 총 거리 | 약 220km (추정) |
| 포장 여부 | 전 구간 석회암 포장 (교역량 많아 유지보수 우선) |
| 주요 기능 | 수도 ↔ 서해 항구 물자 수송·곡물 수출 |
| 통행 | 상인 통행 활발 · 관세소 2개소 (Ashford·Caldenmere) |
| 특기 | Via Imperialis 대륙 간선도로와 Solaris 에서 연결 |
| 서사 기능 | 주인공 Irondelta 경유 시 사용 가능 루트 |
