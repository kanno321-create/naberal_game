## 요약

탈로스 요리는 극한 환경에서의 생존 식문화를 기반으로 한다. 고칼로리·보존성·열생성이 3대 원칙. 정교한 조리보다 양과 기능을 우선시하며, 맥주는 물보다 안전한 음료로 일상 필수품이다.

---

## 주요 음식

### 육류
| 요리 | 설명 |
|------|------|
| **노르벤드 훈제 멧돼지** | 사냥 후 냉훈법으로 6개월 보존 · 탈로스 최고급 식재료 |
| **산양 염장 고기** | 소금 (수입) 사용 · 겨울 주식 |
| **철냄비 스튜** | 염장육 + 수입 보리 + 산악 뿌리채소 혼합 · 광부 일상식 |
| **매복 구이 (Ambush Roast)** | 전투 후 야전에서 직화 구이 · 기사단 전통 요리 |

### 유제품
| 요리 | 설명 |
|------|------|
| **Norvend 숙성 치즈** | 산양유 기반 · 6~12개월 동굴 숙성 · 독특한 광물 맛 |
| **발효 버터** | 겨울 열량 공급원 |

### 빵·곡물
| 요리 | 설명 |
|------|------|
| **호밀 검은빵** | 탈로스 일상 빵 · 밀빵보다 거칠고 단단 · "탈로스 돌빵"이라 불림 |
| **보리죽** | 겨울 아침식사 · 광부들의 기본 에너지원 |

### 음료
| 음료 | 설명 |
|------|------|
| **탈로스 흑맥주** | 호밀 기반 진한 흑맥주 · 광부들이 갱도 입장 전 한 잔 마시는 전통 |
| **노르벤드 허브차** | 고산 허브 혼합 · 체온 유지 효과 있다는 민간 신앙 |
| **Karzor 향신료 차** | 희귀·고가 · 왕족·상인만 접근 가능 |

## 식문화 특성

| 특성 | 내용 |
|------|------|
| 식사 속도 | 빠름 — 업무 중 식사 관습 |
| 식사 대화 | 최소 · 식사 = 연료 보충 개념 |
| 만찬 예절 | 없음에 가까움 — 외지인 경악 포인트 |
| 술자리 | 유일하게 말이 많아짐 — 맥주 3잔 이후 |
| 금기 | 음식 남기기 = 심각한 결례 (배고픔이 미덕) |

## Karzor 향신료 연결

- 소량이지만 Nomen 섬 경유 교역로를 통해 Karzor 향신료 유입
- 탈로스 왕실 요리에만 등장 — 평민은 구경도 못함
- 탈로스인들은 "향신료 범벅 음식"을 경멸하면서도 비밀리에 선물로 선호
