# Icehelm → Irongate 왕도 서쪽 주도로 (Western King's Road)

| 항목 | 내용 |
|------|------|
| **출발** | Icehelm 남문 |
| **도착** | Irongate (Greygate Pass 입구) |
| **거리** | 약 70km |
| **여행 시간** | 도보 3일 · 마차 2일 · 말 1일 |
| **경유지** | Ironvale (40km 지점) · Greymantle (Pass 입구 전) |
| **도로 상태** | 철 강화 석판 포장 · 탈로스 내 최고 수준 |
| **계절 제한** | 겨울 Pass 폐쇄 시 Ironvale 에서 통행 중단 |
| **중요도** | ★★★★★ — 탈로스 철 수출의 메인 동맥 |

## 도로 의의

이 도로는 탈로스 경제의 대동맥. Norvend 광산 철광석이 Embervane 에서 가공된 후 이 도로를 통해 Greygate Pass 를 넘어 Vaelin·성좌국으로 수출된다. Varn 공작이 이 도로 유지보수 비용을 통행세로 징수하는 것이 왕실과의 갈등 원인 중 하나.
