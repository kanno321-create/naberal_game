# Icehelm → Stonecrown 고원 도로 (Highland Road)

| 항목 | 내용 |
|------|------|
| **출발** | Icehelm 동문 |
| **도착** | Stonecrown 시 (Maerith 접경) |
| **거리** | 약 120km |
| **여행 시간** | 도보 5일 · 마차 3일 |
| **경유지** | Haldurmere · Stormbarrow |
| **도로 상태** | 고원 노면 · 포장 미흡 · 여름만 전면 통행 |
| **계절 제한** | 겨울 3~4개월 통행 불가 (눈 폭설) |
| **중요도** | ★★★ — Maerith 교역 경로 |

## 도로 의의

탈로스에서 Maerith 왕국으로 향하는 주요 교역로. Whitecrest Saddle 고개를 넘는 마지막 경로. Solden 백작 관할.
