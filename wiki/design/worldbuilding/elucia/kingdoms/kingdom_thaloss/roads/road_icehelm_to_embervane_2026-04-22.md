# Icehelm → Embervane 산업도로 (Smelter's Way)

| 항목 | 내용 |
|------|------|
| **출발** | Icehelm 동문 |
| **도착** | Embervane 제련 도시 |
| **거리** | 약 60km |
| **여행 시간** | 도보 2.5일 · 마차 1.5일 |
| **경유지** | Deepcoombe (산악 우회로) · Slagburn |
| **도로 상태** | 광석 운반 마차용 넓은 석판 도로 |
| **계절 제한** | 연중 통행 (산맥 남사면) |
| **중요도** | ★★★★ — 광석→제련 공급 라인 |

## 도로 의의

광산 채굴 원석이 왕도를 경유하지 않고 직접 Embervane 제련소로 이동하는 산업 전용 도로. 광석 운반 마차가 24시간 오간다. Brennar 공작가 관할.
