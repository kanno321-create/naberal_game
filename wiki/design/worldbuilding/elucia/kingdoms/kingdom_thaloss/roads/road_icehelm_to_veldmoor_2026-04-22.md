# Icehelm → Veldmoor 식량 수송 도로 (Grain Road)

| 항목 | 내용 |
|------|------|
| **출발** | Icehelm 남문 |
| **도착** | Veldmoor 마을 (Southveld 공작령) |
| **거리** | 약 80km |
| **여행 시간** | 마차 2일 (식량 운반 마차) |
| **경유지** | Grudenmoor |
| **도로 상태** | 산맥 남기슭 완만 — 마차 통행 가능 |
| **계절 제한** | 연중 통행 (최남부 경로) |
| **중요도** | ★★★★ — 왕도 식량 공급 라인 |

## 도로 의의

산악 왕국의 식량 공급 생명선. Southveld 의 보리·호밀이 이 도로를 통해 왕도로 수송된다. 흉작 시 이 도로가 막히면 왕도 식량 위기 발생. Ergal 백작 관할.
