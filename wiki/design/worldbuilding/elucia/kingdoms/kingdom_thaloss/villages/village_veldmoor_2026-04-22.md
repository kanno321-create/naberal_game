# Veldmoor (벨드무어)

| 항목 | 내용 |
|------|------|
| **위치** | Southveld 공작령 중심 · 산맥 남기슭 평원 |
| **인구** | 약 600~900 |
| **산업** | 보리·호밀·채소 재배·가축 사육 |
| **어원** | *veld* (평야·들판, 게르만) + *-moor* (습지) = "들판 습지" |
| **분위기** | 탈로스에서 유일하게 "평범한" 농촌 풍경 · 산이 뒤에 병풍처럼 선 절경 |
| **특징** | Southveld 백작 Ergal 의 거처가 있음 · 탈로스 내 "먹을 것을 키우는 사람들"로 불림 |
