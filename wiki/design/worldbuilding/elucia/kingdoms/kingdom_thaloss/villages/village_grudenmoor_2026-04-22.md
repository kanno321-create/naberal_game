# Grudenmoor (그루덴무어)

| 항목 | 내용 |
|------|------|
| **위치** | Southveld 공작령 중심 평원 |
| **인구** | 약 800~1,200 |
| **산업** | 보리 재배·호밀 재배·탈로스 흑맥주 원재료 공급 |
| **어원** | *gruden* (게르만계 조어 · 지반·밭) + *-moor* (습지·저지대) = "밭 습지" |
| **분위기** | 탈로스에서 보기 드문 열린 풍경 · 짧은 여름 초록이 눈부심 |
| **특징** | 탈로스 왕국 자급 식량 중 보리 30% 이상 생산 · Ergal 백작 직속 |
| **Q-CORE** | "이름 없는 방랑 학자가 흉작 시 보리 병충해 대응 주문을 알려줬다"는 마을 전설 (간접 단서) |
