# Deepwatch (딥워치)

| 항목 | 내용 |
|------|------|
| **위치** | Icehelm 북방 10km · Norvend 진입 직전 |
| **인구** | 약 100~200 |
| **산업** | 군 전진 기지·봉화 연락·광부 초소 |
| **어원** | *deep* (깊은·산 깊숙이) + *-watch* (감시) = "산 깊숙이의 감시소" |
| **분위기** | 상시 경계 · 민간인 거의 없음 · 군인만의 마을 |
| **특징** | Icehelm 에서 Nordwatch 로 가는 중간 봉화 중계 지점 |
