## 요약

탈로스 건축은 극한 기후 생존과 군사 방어를 최우선으로 한다. 지상은 요새, 지하는 생활. 장식 최소화, 난방 시스템 최대화. 석재는 Norvend Range 자체 조달이므로 건물은 무겁고 두꺼우며 수백 년 내구성을 가진다.

## 핵심 원칙

| 원칙 | 내용 |
|------|------|
| **방어 우선** | 모든 건물이 최소 요새 기능 보유 |
| **지하 확장** | 동절기 생활 공간·비상 식량·피난처 |
| **난방 최우선** | 벽 두께 60~120cm · 중앙 벽난로 필수 |
| **석성 재료** | Norvend 화강암 자체 조달 · 목재 최소 |
| **장식 부재** | 표면 조각 거의 없음 — 실용 구조만 |

## 건물 유형별 특성

### 왕궁·귀족 저택
| 항목 | 내용 |
|------|------|
| 구조 | 4층 지상 + 2층 지하 |
| 벽 두께 | 150~200cm 화강암 |
| 창문 | 매우 작음 (열 손실 방지) |
| 난방 | 중앙 대형 화로 + 지하 온수 채널 (추정) |
| 특징 | 탑은 감시 기능 — 장식 탑 없음 |

### 광부 주거
| 항목 | 내용 |
|------|------|
| 구조 | 2층 지상 + 1층 지하 |
| 재료 | 석재 + 두꺼운 목재 천장 |
| 난방 | 가정용 소형 화로 · 연돌 굵음 |
| 지하 | 식량 창고·겨울 피난처 |
| 배치 | 가족 여러 대 같이 사용하는 연립 구조 |

### 요새·성문
| 항목 | 내용 |
|------|------|
| 구조 | 두꺼운 이중 성벽 · 해자는 없음 (산악 지형) |
| 특징 | 협곡 지형과 일체화 — 자연 절벽이 성벽 기능 |
| 관문 | 철제 이중 포틀리스 |
| 물자 | 3년치 비상 식량 지하 비축 원칙 |

### 광산 갱도
| 항목 | 내용 |
|------|------|
| 구조 | 수평 갱도 + 수직 환기 통 |
| 안전 | 목재 지지대 + 철 강화 |
| 특징 | 갱도 내부에 소형 주거 공간 (장기 채굴 시) |

## 지하 층 문화

탈로스 건축의 가장 독특한 특성:
- **겨울 지하 생활**: 혹한기 6개월 중 2~3개월은 지하에서 주로 생활
- **지하 시장**: 겨울에는 지하 공간을 활용한 소규모 교역
- **지하 의례**: 광부 조합 비밀 회의·철의 맹세 일부 지하에서 진행
- **비상 피난**: 전시 어린이·노약자 전원 지하 이동 체계
