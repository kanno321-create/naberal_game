# Stonecrown (스톤크라운) — Whitecrest Saddle 서쪽 고원 도시

| 항목 | 내용 |
|------|------|
| **위치** | Whitecrest Saddle 서쪽 · Maerith 왕국 경계 |
| **규모** | 인구 약 10,000~15,000 |
| **기능** | Thaloss-Maerith 경계 교역 · 고원 목축 |
| **어원** | *stone* (돌) + *-crown* (왕관 모양 지형) = "돌 왕관 도시" |
| **분위기** | 광막한 고원. 돌풍이 강함. 짧은 여름에 고산 화원이 아름다움 |
| **특징** | Maerith 왕국과의 경계 도시 → 양쪽 문화 혼재 |
