# Embervane (엠버베인) — Norvend Shadow 기슭 제련 도시

| 항목 | 내용 |
|------|------|
| **위치** | Norvend Shadow 숲 기슭 · Icehelm 동쪽 60km |
| **규모** | 인구 약 15,000~20,000 |
| **기능** | 철광 제련·무기·갑옷 제조·대륙 무기 수출 |
| **어원** | *ember* (불씨·제련) + *-vane* (잔물결·회전, 풀무 연상) = "불씨 바람의 도시" |
| **분위기** | 붉은 연기·쇳소리·화로 열기. 24시간 쉬지 않는 제련소 |
| **특징** | Thaloss 최대 수출품 = 무기류. 교황군도 여기서 구매 (실질 경제 레버리지) |
