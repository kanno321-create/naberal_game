# Irongate (아이언게이트) — Greygate Pass 관문 요새

| 항목 | 내용 |
|------|------|
| **위치** | Greygate Pass 남쪽 입구 · Pass 통행 통제 |
| **규모** | 인구 약 8,000~12,000 |
| **기능** | 관문 통행료·군사 통제·북방 무역 독점 |
| **어원** | *iron* (철·강함) + *-gate* (관문) |
| **분위기** | 겨울 5개월 이상 폐쇄. 개방 기간만 상인들로 북적 |
| **특징** | Greygate Pass 폐쇄 결정권 = 실질적 북방 교통 독점 권력 |
