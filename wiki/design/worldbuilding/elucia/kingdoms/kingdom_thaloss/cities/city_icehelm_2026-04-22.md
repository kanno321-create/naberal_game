# Icehelm (아이스헬름) — Thaloss 왕국 수도·산악 요새

## 요약

Norvend Range 남쪽 기슭, Greygate Pass 남방 도로와 Eloryn 강 상류 사이의 천혜 요새 지형에 건설된 산악 왕국의 수도. Norvend 최고봉 Icehelm Peak 에서 이름을 따왔다.

---

## 본문

| 항목 | 내용 |
|------|------|
| **위치** | Norvend Range 남사면 · Greygate Pass 남방 20km |
| **규모** | 인구 약 20,000~30,000 |
| **지형** | 협곡 안 자연 요새 지형 · 3면 절벽 1면 관문 |
| **어원** | *ice* (얼음·산정) + *-helm* (산꼭대기·투구) = "산꼭대기 도시" · Icehelm Peak 계승 |
| **분위기** | 겨울 반년 · 눈과 얼음의 도시. 강인하고 폐쇄적. 방문자에게 냉담 |
| **경제** | 광업(철광·구리·희귀 광석) · 모피 교역 · 산악 무기 제조 |
| **군사** | 교황군이 진입하기 극히 어려운 자연 방어 → 실질 독립 강함 |

**주요 랜드마크** (추정):
- 광부 왕의 궁전 — 광산 수익으로 지은 석재 요새 궁전
- 대광산 입구 — 도시 북쪽 Norvend 진입 직전 최대 광산
- 광부 조합 청사 — 사실상 왕국 실권 보유
