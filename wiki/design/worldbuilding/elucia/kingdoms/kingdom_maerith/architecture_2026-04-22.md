## 요약

Maerith 왕국 건축은 고지 혹한·강풍·몬스터 위협에 대응하는 실용적 방어 구조가 근본이다. 돌 구조물, 두꺼운 벽, 작은 창, 지하 저장고가 4대 원칙이다. 장식적 요소는 최소화하며, 내구성과 단열 성능이 건축의 최고 미덕이다.

---

## 4대 건축 원칙

| 원칙 | 기술 방식 |
|------|---------|
| **돌 구조** | 화강암 + 석회암 혼합 사용 · 목재 최소화 |
| **두꺼운 벽** | 외벽 최소 80cm · 귀족 저택 1.2m 이상 |
| **작은 창** | 창 크기 30×40cm 기준 · 방풍·방어 최우선 |
| **지하 저장고** | 모든 건물에 지하층 필수 · 6개월 식량 비축 기준 |

---

## 건물 유형별 특징

### 왕궁·성채

| 항목 | 내용 |
|------|------|
| **재료** | 화강암 + 철제 보강 |
| **벽 두께** | ~2m (성벽) · ~1.5m (내벽) |
| **탑** | 7기 · 원형 탑 (바람 저항 최적화) |
| **지하** | 지하 3층 · 피신로 연결 |
| **창** | 활구멍 + 소형 방어창 |

### 대성당

| 항목 | 내용 |
|------|------|
| **외관** | 수직 첨탑 · 안개 위로 솟음 |
| **재료** | 전면 화강암 + 첨탑 석회암 |
| **내부** | 단열층 이중벽 · 석재 바닥 |
| **지하** | 납골당 + 성물 보관고 |
| **특이점** | 성좌국 양식 + 고지 별 문양 혼합 |

### 귀족 저택

| 항목 | 내용 |
|------|------|
| **벽** | 1.0~1.2m 돌 외벽 |
| **난방** | 지하 온수 순환 + 대형 벽난로 |
| **창고** | 지하 2층 (식량+귀중품 분리) |
| **마구간** | 건물과 통로 연결 (겨울 동물 관리) |

### 서민 가옥

| 항목 | 내용 |
|------|------|
| **기초** | 암반 위 직접 건설 |
| **벽** | 돌 + 진흙 혼합 80cm |
| **지붕** | 경사도 45° 이상 · 적설 방지 |
| **난방** | 중앙 화덕 · 연기 굴뚝 |
| **지하** | 소형 감자·치즈 저장고 필수 |

## 고지 건축 고유 기법

| 기법 | 설명 |
|------|------|
| **암반 파운데이션** | 고지 동결토 대응 · 암반까지 파서 기초 설치 |
| **이중 지붕** | 외부 경사 지붕 + 내부 수평 천장 · 단열층 형성 |
| **지하 온수로** | 귀족 건물 한정 · 천연 온천수 연결 (Northmere 일부 지역) |
