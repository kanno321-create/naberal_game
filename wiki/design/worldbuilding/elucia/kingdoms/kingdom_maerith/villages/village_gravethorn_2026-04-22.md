# Gravethorn (그레이브손)

| 항목 | 내용 |
|------|------|
| **위치** | Ashcrag 남방 · 고원 서부 변경 |
| **인구** | 약 100~180 |
| **산업** | 가시 식물 약재·독소 추출·소규모 농업 |
| **어원** | *grave* (회색빛·무덤) + *thorn* (가시) |
| **분위기** | 회색 가시 덤불이 마을을 둘러쌈. 이방인 접근 꺼려함. |
| **특이사항** | Maerith 내에서 가장 폐쇄적인 마을. 약초 독소 전문 지식 보유. |
