# Saddlecross (새들크로스)

| 항목 | 내용 |
|------|------|
| **위치** | Whitecrest Saddle 남측 · Crestwatch 공작령 내 |
| **인구** | 약 350~500 |
| **산업** | 통행 보급·여관·마구간 운영 |
| **어원** | *saddle* (산안장·고개) + *cross* (교차로) |
| **분위기** | 상인·사절 통행이 잦음. 가장 개방적인 마을. 다양한 언어가 들림. |
| **특이사항** | Thaloss 상인 정기 기착 마을. Vorn 공작 직할 과세 지점. |
