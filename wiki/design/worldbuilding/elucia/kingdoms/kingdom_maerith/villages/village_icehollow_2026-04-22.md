# Icehollow (아이스할로우)

| 항목 | 내용 |
|------|------|
| **위치** | Coldmere 북방 10km · 빙호 직접 연안 |
| **인구** | 약 120~200 |
| **산업** | 빙하 얼음 채취·보관·남부 수출 |
| **어원** | *ice* (얼음) + *hollow* (속이 빈 곳 — 얼음 저장 동굴) |
| **분위기** | 얼음 빛 반사. 조용한 채취 소리. 말이 거의 없음. |
| **특이사항** | Elucia 유일의 천연 얼음 수출 마을. 고가 교역품. |
