# Maerholt (매어홀트) — Auryn 고원 수도

| 항목 | 내용 |
|------|------|
| **위치** | Auryn 고원 중심부 · 고지대 방어 거점 |
| **규모** | 인구 약 25,000~40,000 |
| **기능** | 고지 왕국 행정·군사·상업 중심 |
| **공작령** | Duchy of Aurynseat (왕가 직할) |
| **해발** | ~1,400m (추정) |
| **어원** | *Maer-* (왕국 어근) + *-holt* (숲·요새) |
| **분위기** | 차갑고 바람 센 고원. 두꺼운 돌 성벽. 안개가 잦음 |
| **특징** | Aurion Divide 동쪽에서 가장 높은 고도의 수도 |

## 주요 인물

| 인물 | 역할 |
|------|------|
| 현왕 Aldrath Maern | 왕궁 거주 · 왕국 최고 통치자 |
| 왕비 Sinevara Maern | 왕실 내정 실질 운영 |
| 왕세자 Edric Maern | 고지 수호단 본부 정기 훈련 |

## 행정·특산

- **행정**: Maern 왕가 직할 · Aurynseat 공작령 중심
- **특산**: 고지 수호단 제련 무기 · Grey Curd 치즈 · Icehelm Brew 약초 술
- **시장**: Frost Market (서리 광장) — 고지 전체 교역 집결지

## Rev.3 서사 접점

- 주인공 일행 왕도 통과 시 고지 수호단 검문 (추정)
- Frost Market 에서 정보 수집 가능
- 왕세자 Edric 의 협조·방해 분기 가능성
