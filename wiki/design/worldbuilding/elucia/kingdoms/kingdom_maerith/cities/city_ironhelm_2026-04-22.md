# Ironhelm (아이언헬름) — Auryn 광산 중심 도시

| 항목 | 내용 |
|------|------|
| **위치** | Auryn 고원 서남부 광맥 지대 |
| **규모** | 인구 약 10,000~15,000 |
| **기능** | 철·구리 광산 채굴·제련 |
| **어원** | *iron* (철) + *-helm* (투구·방어) |
| **분위기** | 광산 매연. 단단한 돌 건물. 대장장이 소리가 끊이지 않음 |
| **특징** | Maerith 왕국 군사 장비 대부분이 이곳에서 제련됨 |
| **공작령** | Duchy of Northmere (Drak 가문) |
| **주요 인물** | Corvath Drak 공작 (영주) · Ironweld Hall 길드장 |

## 행정·특산

- **행정**: Drak 공작 직할 · 광업 특허 마을 포함 관리
- **특산**: 구리 제품·철제 무기·제련 부품
- **주요 기관**: Ironweld Hall (광물 길드) · 대형 제련소 3기 · 광부 병원

## Rev.3 서사 접점

- 구리 광산 이상 징후 정보 수집 가능
- Drak 공작과의 협상 분기 가능성
- 광부들의 북방 지하 이상 목격담 단서
