## 요약

Maerith 방언은 대륙 공통어(Common Tongue)를 기반으로 켈트계 음운과 알프스 고지 어휘가 혼합된 형태다. 느리고 낮은 발음, 모음 길게 끌기, 자음 탈락이 특징이다. 외래인에게는 알아듣기 어렵고 무뚝뚝하게 들린다.

---

## 음운 특징

| 특징 | 설명 | 예시 |
|------|------|------|
| **저음 발음** | 흉성 발음 선호 · 고음 단어 낮추어 발음 | *마을* → 낮은 톤으로 길게 |
| **느린 속도** | 평균 발화 속도 대륙 기준 -30% | 단어 사이 간격 있음 |
| **모음 연장** | 강세 모음을 길게 끌음 | *maer* → *maee-r* |
| **어두 자음 약화** | b·d·g 어두 약해짐 | *break* → *reak* (구어) |
| **어말 묵음** | 어말 -t·-d 묵음화 | *hold* → *hol'* |

## 계층별 언어 사용

| 계층 | 언어 사용 |
|------|---------|
| **왕실** | 공식 석상 = 공통어 표준어 · 사적 = 방언 |
| **귀족** | 공통어 능숙 + 방언 병용 |
| **기사·병사** | 방언 주 사용 · 공통어 기초 |
| **서민·농민** | 순수 방언 사용 · 공통어 제한 |
| **상인 (Highvane)** | 다국어 능숙 · 방언 약화 |
