## 요약

세렌 왕국 의상의 핵심은 **방수·실용성**이다. 습지 환경 때문에 안감에 기름칠을 한 방수 가죽이 가장 보편적 소재이며, 린넨은 더운 계절용 기본 의복이다. 계층에 상관없이 후드 달린 덧옷이 필수품이다. 귀족도 현장에서는 실용복 착용 — "사치보다 기능"이 세렌의 미적 기준.

## 공통 의복 아이템

| 아이템 | 소재 | 용도 |
|--------|------|------|
| 후드 덧옷 | 방수 가죽 / 두꺼운 린넨 | 전 계층 필수. 안개·비 차단 |
| 방수 부츠 | 기름 처리 가죽 | 수로·습지 보행 필수 |
| 린넨 튜닉 | 린넨 (하계) / 양모 (동계) | 기본 속옷 |
| 허리띠 | 가죽 · 버클식 | 도구 걸기 실용 |

## 계층별 의상

### 서민·어부

| 항목 | 내용 |
|------|------|
| 상의 | 소매 없는 린넨 튜닉 또는 짧은 소매 |
| 하의 | 무릎 길이 린넨 바지 (물에 젖어도 빨리 마름) |
| 겉옷 | 방수 처리 짧은 가죽 조끼 |
| 신발 | 방수 가죽 부츠 또는 나무 굽 덧신 |
| 색 | 자연 린넨 색 (염색 최소 — 실용) |

### 상인·장인

| 항목 | 내용 |
|------|------|
| 상의 | 소매 있는 린넨 튜닉 + 가죽 조끼 |
| 하의 | 긴 바지 |
| 겉옷 | 후드 달린 반코트 (방수 처리) |
| 색 | 회색·갈색 베이스 + 소금 흰색 포인트 |

### 귀족·기사

| 항목 | 내용 |
|------|------|
| 상의 | 린넨 셔츠 위 가죽 더블릿 |
| 하의 | 긴 바지 + 허벅지 길이 방수 부츠 |
| 겉옷 | 후드 달린 긴 코트 (고급 방수 가죽 · 안감 모피) |
| 장식 | 가문 문장 자수 (최소) · 소금 결정 브로치 |
| 색 | 회청색·은백 기조 |

### 왕족

| 항목 | 내용 |
|------|------|
| 기본 | 고급 리넨 + 비단 안감 겉옷 |
| 공식 의전 | 회청색 벨벳 코트 + 은백 소금 결정 자수 |
| 왕관 | 소금 결정 왕관 (실물은 작고 실용적) |
| 특이점 | 왕도 방문·습지 탐험 시에도 방수 부츠 필수 |
