# Canal Route: Cernmere to Loravern (수로 간선)

| 항목 | 내용 |
|------|------|
| 출발 | Cernmere 남수문 |
| 도착 | Loravern 수로 교역 허브 |
| 거리 | 약 40~55km (추정) |
| 별칭 | 왕도 수로 (King's Canal Route) / "대수로" |
| 이동 수단 | 평저선 전용 (육로 미존재) |
| 소요 시간 | 순류 기준 반나절 · 역류 시 하루 |
| 중요성 | 세렌 내부 물자 이동 핵심 · 왕도 → 삼각주 최단 경로 |
| 경유 | Fenwater (환적 기지) |
| 위험 요소 | 안개 짙은 날 수로 미아 · 습지 깊은 곳 야생 생물 |
| 수문 통제 | Cernmere 남수문 · Loravern 입구 수문 |
