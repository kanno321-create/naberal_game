# Road: Cernmere to Fenlyn (소금 가도)

| 항목 | 내용 |
|------|------|
| 출발 | Cernmere (왕도) 소금 시장 서문 |
| 도착 | Fenlyn (소금 도시) 염전 검인소 |
| 거리 | 약 60~80km (추정) |
| 별칭 | 소금 가도 (Salt Road) / "흰 길" |
| 노면 | 초반 석조 포장 → 후반 다짐 흙길 + 목재 보강 |
| 특이점 | 소금 수레 전용 통행로 (일반 화물 혼재 금지 구간 있음) |
| 경유 마을 | Saltcross → Saltfen → Fenlyn |
| 위험 요소 | 우기 침수 구간 · 밀수꾼 매복 (야간) |
| 수호단 초소 | 3개소 (Cernmere 출구 · Saltcross · Fenlyn 입구) |
