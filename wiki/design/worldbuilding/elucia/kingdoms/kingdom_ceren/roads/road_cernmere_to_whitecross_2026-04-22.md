# Road: Cernmere to Whitecross (북부 가도)

| 항목 | 내용 |
|------|------|
| 출발 | Cernmere 북문 |
| 도착 | Whitecross · Ilaris 경계 교통로 |
| 거리 | 약 90~120km (추정) |
| 별칭 | 북부 가도 (Northern Road) |
| 노면 | 습지 구간 목재 보강 → 북부 건조 지대 석조 포장 |
| 경유 마을 | Fenwater → Reedholt → Whitecross |
| 중요성 | Ceren-Ilaris 교역 주요 육로 · 왕비 미레스 정기 사신 경로 |
| 위험 요소 | 습지 구간 우기 침수 |
| 안개 기사단 초소 | Cernmere 북문 · Whitecross 국경 |
