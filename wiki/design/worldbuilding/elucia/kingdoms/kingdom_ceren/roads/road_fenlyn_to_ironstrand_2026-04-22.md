# Road: Fenlyn to Ironstrand (해안 가도)

| 항목 | 내용 |
|------|------|
| 출발 | Fenlyn 검인소 남문 |
| 도착 | Ironstrand (하구 어항) |
| 거리 | 약 45~65km (추정) |
| 별칭 | 해안 가도 (Coastal Road) · "소금 해안길" |
| 노면 | 조석 지대 목재 보강 + 방파제 겸 도로 구간 |
| 경유 | Westfen → Ironstrand |
| 중요성 | Ceren 서해 소금 수출 해안 루트 |
| 위험 요소 | 조석 밀물 시 도로 침수 구간 (조석 시간 맞춰 통행) |
| 초소 | Fenlyn 출구 · Ironstrand 항구 입구 |
