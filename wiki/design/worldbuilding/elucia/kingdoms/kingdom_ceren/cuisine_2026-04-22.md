## 요약

세렌 요리는 습지 어업·소금·소박한 곡류를 중심으로 한다. 소금을 가장 싸고 풍부하게 쓸 수 있는 왕국이므로, 절임·훈제·염장이 기본 조리법. 외부 왕국 요리사들이 세렌에 와서 처음 놀라는 것은 "이 작은 왕국에서 왜 소금을 이렇게 아낌없이 쓰는가"라는 것.

## 대표 요리

### 장어 스튜 (Eel Stew · "세른 포트")
| 항목 | 내용 |
|------|------|
| 재료 | 뱀장어 + 수생 채소 + 소금 + 갈대 새싹 + 호밀 덩어리빵 |
| 조리법 | 장어 염장 30분 → 끓임 → 채소 추가 → 호밀빵 곁들임 |
| 상징 | 세렌의 국민 요리 · 귀족도 서민도 먹음 |
| 변형 | 귀족판: 크림 추가 + 허브 · 서민판: 물 + 소금만 |

### 이탄 훈제 붕어 (Peat-Smoked Carp)
| 항목 | 내용 |
|------|------|
| 재료 | 붕어 + 이탄 연기 + 소금 |
| 특징 | 이탄 연기가 스며든 독특한 흙향 · 교역품으로도 인기 |
| 보존 | 훈제 후 2~3주 보관 가능 · 수출용으로도 제작 |

### 소금 절임 수생 채소 (Marshweed Pickle)
| 항목 | 내용 |
|------|------|
| 재료 | 미나리류 + 갈대 새싹 + 소금 + 습지 식초 |
| 용도 | 겨울철 비상식 · 장기 보존 |
| 서민 일상 | 호밀빵 + 절임 채소 = 가장 흔한 세렌 서민 끼니 |

### 호밀빵 (Rye Flatbread · "펜 빵")
| 항목 | 내용 |
|------|------|
| 재료 | 호밀 + 소금 + 물 |
| 형태 | 납작한 원형 · 불 위에 직접 굽기 |
| 용도 | 어업 출선 시 휴대용 식량 |

## 음식 문화

| 항목 | 내용 |
|------|------|
| 식사 횟수 | 2식 (아침·저녁) · 점심 생략 흔함 (어부 작업 중) |

| 소금의 지위 | "소금을 아끼는 집은 가난한 집" — 세렌 속담 |
