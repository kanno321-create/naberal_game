# Fenwater (펜워터)

| 항목 | 내용 |
|------|------|
| 위치 | Loravale 공작령 · 내부 수로 교차점 |
| 인구 | 약 500~800 |
| 산업 | 수로 안내(길잡이) · 화물 환적 · 숙박 |
| 어원 | *fen* (습지) + *water* (물) |
| 특징 | 습지 수로의 주요 환적 거점 · 수로 안내인 조합 본거지 |
| 분위기 | 외지 상인들이 드나들어 세렌에서 가장 활기찬 내부 마을 |
| 정보 | 수로 안내인들이 습지 내부 정보 집결지 |
