# Lowfen (로우펜)

| 항목 | 내용 |
|------|------|
| 위치 | Lorenfen 공작령 남부 · 습지와 목초지 경계 |
| 인구 | 약 400~600 |
| 산업 | 목축(소·양) · 수초 채취 · 소규모 농업 |
| 어원 | *low* (낮은) + *fen* (습지) |
| 특징 | 드라엔 가문 직할 목장 존재 · 세렌에서 육류를 가장 구하기 쉬운 마을 |
| 분위기 | 습지 냄새 대신 축산 냄새 · 남부 특유의 건조한 느낌 |
| 행정 | Lowfen 백작령 소속 |
