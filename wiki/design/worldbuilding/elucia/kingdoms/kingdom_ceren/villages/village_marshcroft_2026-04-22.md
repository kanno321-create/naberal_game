# Marshcroft (마쉬크로프트)

| 항목 | 내용 |
|------|------|
| 위치 | Loravale 공작령 습지 중부 · 수생 약재 자생지 |
| 인구 | 약 150~250 |
| 산업 | 수생 약재 채취·건조·판매 · 소규모 어업 |
| 어원 | *marsh* (늪) + *croft* (작은 경작지) |
| 특징 | 약재 채취 가문들이 수세대 거주 · 교회 병원에 약재 공급 |
| 분위기 | 약초 향 · 말린 허브 다발이 처마에 가득 |
| Q-CORE 2 | "낯선 노인이 독성 약초 정화법을 알려줬다"는 마을 전설 |
