# Mistlyn (미스트린) — Loravel 습지 고립 섬 마을

| 항목 | 내용 |
|------|------|
| **위치** | Loravel 중부 습지 섬 (수로로만 접근) |
| **인구** | 약 150~300 |
| **산업** | 어업·달팽이·개구리·수생 약재 채취 |
| **어원** | *mist* (안개) + *-lyn* (호수변) = "안개 호수 마을" |
| **분위기** | 외부인 거의 없음. 수대에 걸친 같은 가문들만 거주. 달 숭배 전통 강함 |
| **특징** | 습지 깊은 곳 → 교황청 통제 완전 불가 · 숨겨진 전통 신앙 보존 |
