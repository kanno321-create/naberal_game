# Drenfold (드렌폴드)

| 항목 | 내용 |
|------|------|
| 위치 | Lorenfen 공작령 중심 · 드라엔 가문 역사적 발원지 |
| 인구 | 약 600~800 |
| 산업 | 가축·농업 · 드라엔 가문 고문서 보관 |
| 어원 | *dren* (뿌리·깊음 — 드라엔 어근) + *fold* (목초지) |
| 특징 | 드라엔 가문 조상 묘역·구성 (세렌 귀족 중 가장 오래된 묘역) |
| 역사 | 습지 부족 연합 이전부터 드라엔 씨족이 지배하던 정착지 |
| 분위기 | 오래된 돌담과 이끼 · 마을 전체에 무거운 역사감 |
| Q-CORE 2 | 드라엔 고문서 중 "이름 모를 치유자" 관련 매우 오래된 기록 단편 존재 (추정) |
