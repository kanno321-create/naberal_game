# Grayfen (그레이펜)

| 항목 | 내용 |
|------|------|
| 위치 | Loravale 공작령 최심부 · 수로로만 접근 가능 |
| 인구 | 약 80~120 |
| 산업 | 소규모 어업 · 갈대 채취 · 수생 조개 |
| 어원 | *gray* (회색·안개) + *fen* (습지) |
| 특징 | 세렌에서 가장 외진 마을. 수로 안내인도 잘 모르는 경로 |
| 분위기 | 안개가 1년 내내 낮게 깔림 · 소리가 습지에 흡수됨 |
| Q-CORE 2 | "안개 속에서 나타났다가 사라지는 노인"에 대한 목격 전설 다수 |
| 비고 | 타종족 은신 공동체 인접 가능성 (추정 · 대표님 미확정) |
