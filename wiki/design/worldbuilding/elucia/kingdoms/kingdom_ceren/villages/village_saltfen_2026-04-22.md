# Saltfen (솔트펜)

| 항목 | 내용 |
|------|------|
| 위치 | Fenlyn 남쪽 조석 염전 지대 내 소금 채취 마을 |
| 인구 | 약 300~500 |
| 산업 | 소금 채취 노동 · 소금 자루 제작 |
| 어원 | *salt* + *fen* (습지) |
| 특징 | 소금 수호단 검인관 1명 상주 · 주민 대부분이 시즌 노동자 |
| 분위기 | 흰 소금 먼지 가득 · 일 끝나면 모두 침묵 속에 쉼 |
| Q-CORE 2 | 마을 공동 우물에 "이름 없는 학자가 정화 주문을 남겼다"는 전설 전해짐 |
