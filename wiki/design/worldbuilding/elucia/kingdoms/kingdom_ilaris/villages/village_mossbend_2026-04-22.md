# Mossbend (모스벤드) — Silvan 숲 이끼 계곡 취락

| 항목 | 내용 |
|------|------|
| **위치** | Silvan 中林 계곡 · Grenwick 서쪽 12km |
| **인구** | 약 150~250 |
| **산업** | 약초·이끼 채취·숲 가이드·수지 채취 |
| **어원** | *moss* (이끼) + *bend* (계곡 굽이) |
| **분위기** | 항상 촉촉·녹색 이끼로 뒤덮인 집 · 신비로운 분위기 |
| **특이점** | 삼림 깊은 곳 소문을 가장 많이 아는 마을 · 엘프 접촉 구전이 남아있는 곳 중 하나 |
