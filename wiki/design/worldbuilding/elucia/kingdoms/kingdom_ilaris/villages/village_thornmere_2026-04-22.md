# Thornmere (손미어) — Silvan 外林 가시덤불 취락

| 항목 | 내용 |
|------|------|
| **위치** | Silvenmere 북서 8km · 숲 경계 |
| **인구** | 약 200~350 |
| **산업** | 가시목 채취·방어 울타리 제조·사냥 |
| **어원** | *thorn* (가시목) + *-mere* (내지·마을) |
| **분위기** | 숲 경계의 마지막 마을. 이 너머는 변경군 허가 없이 못 들어감 |
| **특이점** | 삼림 변경군 초소 1 거점이 이 마을에 위치 |
| **Q-CORE 간접** | "어느 해 역병이 돌 때 이름 없는 영감이 우물 정화 주문을 가르쳐 갔다" 구전 존재 |
