# Woldhaven (월드헤이번) — Deepsilvan 삼림꾼 피난처

| 항목 | 내용 |
|------|------|
| **위치** | Deepsilvan 공작령 서부 · 숲 심부 접경 |
| **인구** | 약 120~200 |
| **산업** | 삼림 경비 지원·야생 사냥·약초 채취 |
| **어원** | *wold* (삼림·황야) + *haven* (피난처·안식지) |

| **특이점** | 탐험대 실종 전 마지막으로 목격되는 마을 |
