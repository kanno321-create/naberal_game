# Ironbarrow (아이언바로) — Deepsilvan 변경군 보급 취락

| 항목 | 내용 |
|------|------|
| **위치** | Deepsilvan 공작령 중부 · 변경 초소 2 거점 인접 |
| **인구** | 약 400~600 |
| **산업** | 군수 보급·무기 수리·삼림 군마 사육 |
| **어원** | *iron* (철) + *barrow* (보급 창고·언덕) |
| **분위기** | 항상 변경군이 드나듦 · 무기 수리 소리 · 군마 냄새 |
| **특이점** | Bruiden 공작의 포획 작전 출발 기지 마을 · 포획된 타종족 임시 수용소 의혹 |
