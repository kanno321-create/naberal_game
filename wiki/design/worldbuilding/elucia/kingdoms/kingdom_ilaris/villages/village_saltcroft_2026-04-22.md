# Saltcroft (솔트크로프트) — Westshore 남부 소금 어촌

| 항목 | 내용 |
|------|------|
| **위치** | Westshore 해안 중부 · Ceren 접경 40km 북 |
| **인구** | 약 300~500 |
| **산업** | 어업·소금 증발·해산물 건조 |
| **어원** | *salt* + *croft* (작은 농장·어촌) |
| **분위기** | 소금 냄새·건조 생선 걸린 집들·밀물 때 마을 남쪽이 잠김 |
| **특이점** | Ceren 소금과 경쟁하는 자체 소금 생산 → 소금 분쟁 시 입장 복잡 |
