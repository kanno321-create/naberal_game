# Ilarien → Deepsilvan 삼림 내부 도로 (Deepwood Road)

## 도로 정보

| 항목 | 내용 |
|------|------|
| **구간** | Ilarien → Silvenmere → Deepsilvan 공작 본거 |
| **길이** | 약 110km |
| **노면** | 왕도~Silvenmere: 석조 / 이후: 험한 삼림로 |
| **통행** | 허가증 필요 (Silvenmere 이후) |
| **관리** | Bruiden 공작 직속 변경군 |

---

## 경유지

```
Ilarien (수도)
    ↓ 35km
Silvenmere (숲 가장자리 · 허가증 검문)
    ↓ 30km 삼림로
Grenwick (삼림 경계 취락)
    ↓ 45km 험한 삼림로
Deepsilvan 공작 본거
```

---

## 주요 특징

- Silvenmere 이후 삼림로 = 변경군 호위 없이 통행 불가 (공식 규정)
- 포획 작전 이동 경로이기도 함
- 탐험대 실종 사건이 Grenwick 이후 구간에서 발생

---

## 위험 등급

| 구간 | 위험도 |
|------|-------|
| Ilarien → Silvenmere | 낮음 |
| Silvenmere → Grenwick | 중간 |
| Grenwick → 공작 본거 | 높음 |
| 공작 본거 → 숲 심부 | 극고 (미개척) |

---

## 대표님 미확정 사항

- 허가증 발급 절차·비용
- 탐험대 실종 기록 정확한 위치
