## 요약

Moran 왕국 문장은 청색(바다)과 은색(안개·파도)을 기본 배색으로 하며, 파도와 바다새가 핵심 모티프. 귀족 가문 문장은 이 왕국 기본 배색 위에 가문 고유 모티프를 추가.

## 왕국 기본 문장

| 항목 | 내용 |
|------|------|
| **방패형** | 분할 없는 단일 방패 |
| **배경색** | 깊은 청색 (Sea Blue) |
| **주문양** | 은색 파도 3중선 (위로 갈수록 높아지는 파도) |
| **보조문양** | 파도 정상 위 흰 바다새 (날개 펼침) |
| **왕관** | 3개 첨탑 왕관 + 각 첨탑에 진주 1개 (해산물 상징) |
| **투구장식** | 바다새 날개 한 쌍 |
| **모토** | *"Mornvel Tir Soran"* (해양 켈트계 — "바다가 길이요 고향이다") |
