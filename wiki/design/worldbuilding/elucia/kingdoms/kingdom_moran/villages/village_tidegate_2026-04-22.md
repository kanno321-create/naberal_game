# Tidegate (타이드게이트) — 조수문 관리 마을

| 항목 | 내용 |
|------|------|
| **위치** | Mornwell 강 하구 조수 역류 제어 지점 |
| **인구** | 약 250~400 |
| **산업** | 조수문 관리·강 수운 안내·소규모 어업 |
| **어원** | *tide* (조수) + *gate* (수문) |
| **분위기** | 수문 열고 닫는 소리가 마을 시간을 알림 |
| **특이점** | 조수문 관리 기술 = 왕국 수운 물류의 핵심 · 관리 가문 Morn-Tide 씨족 |
| **중요성** | 조수문 막히면 Mornheld 항구 선박 진출입 불가 → 군사 전략 요충지 |
