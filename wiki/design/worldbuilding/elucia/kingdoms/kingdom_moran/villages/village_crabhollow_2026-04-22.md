# Crabhollow (크랩홀로우) — 조개·게 특산 어촌

| 항목 | 내용 |
|------|------|
| **위치** | Havren 내만 남부 조간대 해안 |
| **인구** | 약 350~550 |
| **산업** | 조개·게·홍합 채취·파도제 재료 공급 |
| **어원** | *crab* (게) + *-hollow* (오목한 해안 지형) |
| **분위기** | 썰물 때마다 마을 전체가 바위 위로 나와 조개 캠 |
| **특산** | 조개찜 축제 재료 공급처 (파도제·출어제 소비) |
| **공동 시설** | 조개 삶는 마을 공동 솥 3기 (상시 운영) |
