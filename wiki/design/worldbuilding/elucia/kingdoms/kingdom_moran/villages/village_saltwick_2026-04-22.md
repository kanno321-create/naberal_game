# Saltwick (솔트위크) — 청어 염장 마을

| 항목 | 내용 |
|------|------|
| **위치** | Havenwick 근교 내만 소만 |
| **인구** | 약 400~600 |
| **산업** | 청어·대구 염장 처리 전문 |
| **어원** | *salt* (소금) + *-wick* (취락) |
| **분위기** | 소금 창고 냄새와 염장 작업 소리가 마을을 가득 채움 |
| **특산** | Saltwick 염장 청어 (Havenwick 브랜드 원산지 실체) |
| **공동 시설** | 마을 중앙 대형 공동 염장 창고 1동 |

**Q-CORE 2 간접**: 공동 창고 기초석에 "방부 주문" 흔적 구전 (추정)
