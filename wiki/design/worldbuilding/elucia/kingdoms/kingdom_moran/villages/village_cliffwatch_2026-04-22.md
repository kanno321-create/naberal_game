# Cliffwatch (클리프워치) — 절벽 감시 초소 마을

| 항목 | 내용 |
|------|------|
| **위치** | Morncliff Spine 중부 절벽 중간 감시 지점 |
| **인구** | 약 150~250 |
| **산업** | 파도 기사단 지원·봉화 관리·소규모 어업 |
| **어원** | *cliff* (절벽) + *watch* (감시) |
| **분위기** | 항상 바람이 강함. 귀에 익숙한 바람 소리 없으면 잠 못 든다는 주민들 |
| **특이점** | 파도 기사단 소형 초소 2개 인근 배치 |
| **봉화** | 이 마을 봉화가 Greycliff ↔ Mornheld 연락 릴레이 중간 거점 |
