## 요약

Moran 의상은 북서 해안 기후(강풍·안개·염분)에 최적화된 실용 중심 디자인. 귀족층에서도 불필요한 화려함보다 기능적 품위를 강조. 방수 가죽과 두꺼운 린넨이 핵심 재료.

---

## 계층별 의상

### 왕족·귀족 (High Class)

| 항목 | 내용 |
|------|------|
| **상의** | 두꺼운 양모 튜닉 (청색 또는 깊은 청록) + 가문 문장 자수 |
| **외투** | 무두질 방수 가죽 대망토 (Morncliff 가죽 · 은색 버클) |
| **두건** | 해풍용 두건 — 얼굴을 가릴 수 있는 깊은 구조 |
| **신발** | 방수 처리 가죽 장화 (무릎 아래) |
| **장신구** | 은 진주 목걸이·팔찌 (바다 신앙 상징) |
| **여성 귀족** | 린넨 드레스 위 짧은 가죽 조끼 + 망토 · 두건 필수 |

### 평민·어부 (Common Class)

| 항목 | 내용 |
|------|------|
| **상의** | 거친 린넨 튜닉 (염색 없음 또는 단순 청색) |
| **외투** | 어유 처리 방수 가죽 조끼 (짧은 것) |
| **두건** | 울 소재 두건 — 귀를 감싸는 밀착형 |
| **신발** | 가죽 샌들 (항구) or 방수 부츠 (항해) |
| **작업복** | 앞치마 형태 방수 가죽 덧댐 |

### 수군·해병

| 항목 | 내용 |
|------|------|
| **제복** | 진청색 튜닉 + 가죽 갑옷 조끼 |
| **두건** | 철제 링 보강 울 두건 |
| **망토** | 방수 처리 전투용 짧은 망토 |
| **계급 표식** | 은색 늑대 발톱 핀 (바다 늑대단) · 청록 파도 배지 (파도 기사단) |

## 기후 적응 요소

- **두건 착용 관습**: 안개·강풍 때 두건을 완전히 닫는 것이 예의 (얼굴이 보이지 않아도 무례 아님)
- **망토 염수 방지**: 귀족 망토는 연 1회 어유 처리 의식 (특수 장인 담당)
- **여름 경량**: 여름에도 린넨 두겹 착용 — 해풍 차단 때문
