# Havenwick (헤이번위크) — Havren 내만 어업 도시

| 항목 | 내용 |
|------|------|
| **위치** | Havren 내만 중부 · 안전한 내항 |
| **규모** | 인구 약 15,000~20,000 |
| **기능** | 어업 · 조선소 · 어물 염장·수출 |
| **어원** | *haven* (항구·안식) + *-wick* (취락) = "항구 취락" |
| **분위기** | 평화로운 어촌 도시. 절벽 항구와 달리 온화한 기후. 축제 많음 |
| **특징** | 청어·대구 염장 기술로 유명. Nomen 섬까지 수출 (추정) |
| **인구 구성** | 어부 70% · 상인 20% · 기타 10% |
| **주요 기관** | 어업 길드 지부·염장 협동조합·어부 공동 창고 |

## 심화 (Wave 4)

- **축제 중심지**: 출어제·파도제에서 Mornheld 다음으로 큰 행사
- **Saltwick 연계**: 인근 Saltwick 마을이 실제 염장 처리 담당
- **수출 루트**: Mornheld 항구 경유 → Nomen 섬 상단 납품
- **Rev.3 접점**: 어시장 씬·청어 수출 의뢰·조선소 정보 수집
