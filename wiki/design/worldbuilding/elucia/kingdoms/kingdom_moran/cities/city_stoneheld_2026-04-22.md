# Stoneheld (스톤헬드) — Morncliff Spine 석재 채굴 도시

| 항목 | 내용 |
|------|------|
| **위치** | Morncliff Spine 중부 구릉 기슭 |
| **규모** | 인구 약 12,000~18,000 |
| **기능** | 석재 채굴·가공·Mornheld 수출 |
| **어원** | *stone* (돌) + *-held* (거점) = "돌의 거점" |
| **분위기** | 먼지·채석 소음. 강인한 석공 가문들이 도시 지배 |
| **특징** | Moran 도시 건축·항구 방파제 석재 공급원. 드워프 구전에 "인간이 빼앗은 돌산" 전설 |
| **행정** | Draven 백작가 직할 · 시장 = 석공 길드장 |
| **수출** | 왕국 건축 자재 80% 이상 공급 · Stone Road 경유 |

## 심화 (Wave 4)

- **드워프 갈등**: 비공개 경계 협상 진행 중 (대표님 미확정)
- **석공 길드**: 마을 실권 장악 — 채굴·가공·운반 독점
- **Rev.3 접점**: 채굴권 분쟁·드워프 관련 시나리오 거점
