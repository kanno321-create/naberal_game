# Keldhaven (켈드헤이번) — Havren 남부 온천·치유 도시

| 항목 | 내용 |
|------|------|
| **위치** | Havren 권역 남부 · Morncliff Spine 내부 온천 분출 지점 |
| **규모** | 인구 약 10,000~15,000 |
| **기능** | 온천 치유·귀족 요양·종교 성지 |
| **어원** | *keld* (샘·원천, L3 켈트) + *-haven* (안식처) = "원천의 안식처" |
| **분위기** | 부드러운 유황 안개. 귀족 요양객·성직자·치유사 혼재 |
| **특징** | 교황청이 "신성한 샘" 으로 공인 → 치유 순례지. 실제론 지열 온천 |
| **왕비 후원** | Sorwen 왕비 개인 후원 → 해신 신전 지부 설치 |
| **경제** | 순례자 수입·귀족 요양 숙박·치유사 서비스 |

## 심화 (Wave 4)

- **교황청 갈등**: 공인 성지이나 해신 신전 지부가 같이 있어 교황청 측 불만
- **치유사 집단**: 성직자 치유사 vs 서민 허브 치유사 공존·경쟁
- **Q-CORE 2 간접**: "이름 없는 노인 의사가 수십 년 전 이 온천의 진짜 효능을 알려줬다"는 치유사 비밀 전승 존재 (추정)
- **Rev.3 접점**: 부상 치유·희귀 정보 수집·귀족 접선 장소
