## 요약

Moran 건축은 Spineback 산에서 채굴한 회색 석재를 주재료로 하며, 바닷바람·염분·습기에 강한 밀폐형 구조를 특징으로 한다. 등대는 Moran 건축 정체성의 상징. 목재는 내부 구조재로만 제한적 사용.

---

## 건축 양식 분류

### 1. 해안 석성 (Coastal Fortress Style)

- **재료**: Spineback 회색 화강암 (두께 2~3m 외벽)
- **특징**: 창문 최소화 (바람 차단) · 외벽 경사형 (파도 충격 분산)
- **대표**: Greycliff 요새 · Mornheld 왕궁 성벽
- **세부**: 해수면 아래 기초는 철 말뚝 + 석재 복합 (부식 대비)

### 2. 항구 성벽 (Harbour Wall Style)

- **재료**: 화강암 + 굴껍질 혼합 모르타르 (방수 강화)
- **특징**: 해수 접촉 부분에 특수 굴껍질 코팅 — 염분 침식 저항
- **대표**: Grand Harbour 방파제 · Navy Bastion 외벽
- **높이**: 방파제 6~9m · 포대 포함 시 12~15m

### 3. 등대 (Lighthouse Style)

- **재료**: 순수 화강암 (건물 중 가장 정밀한 석공 기술 요구)
- **특징**: 원형 탑 구조 · 등탑 창문 12방향 (360도 조망)
- **대표**: Mornheld 대등대 2기 · Greycliff 등대
- **의미**: 등대는 Moran 의 힘의 상징 → 등대 수 = 왕국 번영 척도

### 4. 어부 공동 창고 (Fisher's Storehouse Style)

- **재료**: 목재 골조 + 석재 기초 + 갈대 지붕 처리
- **특징**: 반지하 구조 (온도 유지) · 환기창 바람 방향 계산 설계
- **대표**: Fisher's Hollow 공동 창고 20동
- **사용**: 어획물 보관·염장·그물 수선 공간 겸용

### 5. 귀족 저택 (Noble Hall Style)

- **재료**: 내부 목조 + 외부 석재 클래딩
- **특징**: 절벽 지형 활용 — 지하층이 절벽 아래를 향해 돌출
- **난방**: 석재 벽난로 + 해풍을 이용한 자연 환기 시스템
- **장식**: 청색·은색 타일 모자이크 (파도 문양) 현관 바닥

## 등대 설계 세부

| 항목 | 내용 |
|------|------|
| **재료** | Spineback 화강암 (마름돌) |
| **구조** | 원형 나선 계단 내장 · 외부 사다리 없음 (접근 제한) |
| **등불** | 고래기름 + 은 반사판 (가시거리 약 15~25km) |
| **높이** | 대형 25~35m · 소형 10~15m |
| **유지** | Sorn 공작 가문 독점 관리 (3대 전통) |
