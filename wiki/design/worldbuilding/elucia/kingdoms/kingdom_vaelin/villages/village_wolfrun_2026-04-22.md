# Wolfrun (울프런) — 초원 군마 목장 마을

| 항목 | 내용 |
|------|------|
| 위치 | Plainwatch 공작령 중부 대초원 |
| 인구 | 약 700~900 |
| 산업 | 군마 목장 · 방목 · 가죽 가공 |
| 어원 | Wolf(늑대) + Run(달리는 길) — 늑대 무리 이동로 옛 지명 |
| 분위기 | 광활한 초원 · 말 냄새 · 느긋하나 강건함 |
| 특징 | 브란 가문 주요 목장 관리 기지. 연간 300두 출하 (추정) |

## 서사 접점
- 군마 거래 관련 서브퀘스트 기지 후보
