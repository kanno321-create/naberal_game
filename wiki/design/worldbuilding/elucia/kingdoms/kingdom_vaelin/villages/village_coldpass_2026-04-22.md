# Coldpass (콜드패스) — Greygate 직전 마지막 민간 마을

| 항목 | 내용 |
|------|------|
| 위치 | Greyford 공작령 최북단 · Greygate Pass 입구 3km |
| 인구 | 약 200~300 (겨울 50명으로 감소) |
| 산업 | 북방 상인 숙박 · 군마 임시 보관 |
| 어원 | Cold(차가운) + Pass(고개) |
| 분위기 | 황량 · 바람 강함 · 항상 겨울 기색 |
| 특징 | 대부분 계절성 주민. 겨울 완전 철수 관습. 봉화대 마을 옆 위치 |

## 서사 접점
- 주인공이 탈로스 방면 이동 시 최후 민간 거점
- 긴장감 높은 전방 분위기 묘사
