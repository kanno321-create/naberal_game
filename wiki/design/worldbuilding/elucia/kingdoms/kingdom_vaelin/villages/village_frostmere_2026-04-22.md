# Frostmere (프로스트미어) — 북부 늪지 개간 마을

| 항목 | 내용 |
|------|------|
| 위치 | Greyford 공작령 남쪽 · 소규모 늪지 인근 |
| 인구 | 약 200~300 |
| 산업 | 늪지 농업 (내한성 작물) · 수조 어업 |
| 어원 | Frost(서리) + Mere(늪) |
| 분위기 | 안개 많음 · 차갑고 눅눅함 · 폐쇄적 공동체 |
| 특징 | 겨울 6개월 눈으로 고립. 독립적 공동체 문화. 외지인 경계 강함 |

## 서사 접점
- 고립된 마을 = 할배 간접 단서 구전이 살아남은 곳일 가능성
