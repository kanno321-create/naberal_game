# Halvmark (할브마르크) — 선왕 할바르드 기념 마을

| 항목 | 내용 |
|------|------|
| 위치 | Vaelmark 공작령 동쪽 |
| 인구 | 약 400~550 |
| 산업 | 농업 · 기사 훈련소 부속 마을 |
| 어원 | Halvard(선왕 이름) + Mark(경계·지역) |
| 분위기 | 선왕을 숭상하는 보수적 전통 · 에드릭 지지 강함 |
| 특징 | 선왕 할바르드가 젊은 시절 훈련한 장소. 마을 광장에 기마 조각상 있음 |

## 서사 접점
- 에드릭 왕세자의 비공식 지지 기반
- 선왕 숭상 vs 현왕 온건 노선 갈등 체감
