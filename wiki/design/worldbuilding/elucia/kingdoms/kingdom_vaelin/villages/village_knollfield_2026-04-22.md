# Knollfield (놀필드) — 구릉 귀리 농촌

| 항목 | 내용 |
|------|------|
| 위치 | Vaelmark 공작령 서쪽 완만한 구릉 |
| 인구 | 약 500~650 |
| 산업 | 귀리·보리 농업 · 말 보조 사료 생산 |
| 어원 | Knoll(구릉) + Field(들판) |
| 분위기 | 완만하고 평화로운 농촌 · 계절 바람이 아름다운 곳 |
| 특징 | 군마 말 사료의 주요 공급지. 플레인워치 목장과 직거래 |
