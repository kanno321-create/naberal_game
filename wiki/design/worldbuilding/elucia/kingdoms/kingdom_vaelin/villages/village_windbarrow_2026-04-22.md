# Windbarrow (윈드배로) — 초원 언덕 목동 마을

| 항목 | 내용 |
|------|------|
| 위치 | Plainwatch 공작령 북쪽 · 언덕지대 |
| 인구 | 약 300~450 |
| 산업 | 양 목축 · 양모 1차 가공 |
| 어원 | Wind(바람) + Barrow(언덕 무덤) — 옛 기마전사 묘역 인근 |
| 분위기 | 바람 항상 강함 · 목동 생활 · 고독한 초원 |
| 특징 | 마을 뒤편에 수백 년 된 기마전사 묘역 존재. 왕국 전통 유지 강함 |

## 서사 접점
- 묘역 = 건국 초기 전사자들의 매장지 → 역사 단서 발굴 가능
