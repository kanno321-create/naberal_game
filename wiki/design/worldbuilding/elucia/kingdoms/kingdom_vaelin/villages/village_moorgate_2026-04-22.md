# Moorgate (무어게이트) — 성좌국 접경 경계 마을

| 항목 | 내용 |
|------|------|
| 위치 | Elorfeld 공작령 최남단 · 성좌국 직할지 접경 |
| 인구 | 약 300~400 |
| 산업 | 성좌세 징수 관리 · 순례자 숙박 · 교역 통관 |
| 어원 | Moor(황무지) + Gate(관문) |
| 분위기 | 교황청 경비병과 Vaelin 경비병 혼재 · 긴장 묘한 평화 |
| 특징 | 성좌국 사절·순례자·상인의 Vaelin 입국 첫 지점 |

- 성좌국 vs Vaelin 접경 긴장 체감 장소
- 교황청 감시원 vs 에드릭 지지 귀족 갈등 무대
