# Vaelthorn (바엘손) — Vaelin 왕국 수도·기마 도시

## 요약

북부 Vaelin 평원 중앙에 위치한 기마 왕국의 수도. Mornwell 강 상류와 Eloryn 강 지류가 만나는 지점. 광활한 평원에 어울리는 개방적이고 강인한 도시.

---

## 본문

| 항목 | 내용 |
|------|------|
| **위치** | Vaelin 평원 중앙, 두 강의 합류 지점 |
| **규모** | 인구 약 35,000~50,000 |
| **입지** | 평원 감제 고지 · 강 방어 이중 활용 |
| **지형** | 완만한 구릉 위 도성 + 평원 외곽 기마 훈련장 |
| **주요 기능** | 기사단 본부·군사 학교·북부 교역 집산 |
| **어원** | *Vael* (왕국명 계승) + *-thorn* (가시·강인) = "강인한 Vaelin의 심장" |
| **분위기** | 개방적·강건. 기사단 행진이 일상적. 화려함보다 실용성 |
| **종교** | 교황청 기사단 분교 운영 · 표면상 교황청 종속, 실제론 왕국 독립성 강 |

**주요 랜드마크** (추정):
- 기사단 성채 — Vaelin 왕국 군사력 상징
- 왕궁 (기마 광장 면함) — 왕이 기마로 접견 전통
- 북방 교역소 — Thaloss·Norvend 방향 대상 집결
