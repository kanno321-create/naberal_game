# Road: Vaelthorn → Plainwatch (왕도→초원 군마 도로)

| 항목 | 내용 |
|------|------|
| 시점 | Vaelthorn (동문 Plaingate) |
| 종점 | Plainwatch 성채 |
| 총 길이 | 약 200km (추정) |
| 노면 | 초원 다진 흙길 + 일부 판석 (군마 발굽 보호) |
| 이동 시간 | 기마 2일 · 도보 7일 |

| 순서 | 지점 | 거리 |

- 군마 이송 주 경로 — 연간 수천 두 이동
- 철창 기사단 초원 순찰 루트 기반
