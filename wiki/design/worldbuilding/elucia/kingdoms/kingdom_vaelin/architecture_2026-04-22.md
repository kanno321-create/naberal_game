# Vaelin 왕국 건축 양식 (Architecture)

---

## 요약

Vaelin 건축은 방어와 실용성 우선. 두꺼운 석조 성벽·목조 내부 구조 혼합. 성채마다 마구간이 거주 공간과 동급 규모. 장식보다 기능이 앞서며, 겨울 방풍·단열이 설계 최우선.

---

## 1. 건축 유형별 특성

### 1-1. 왕궁·성채

| 항목 | 내용 |
|------|------|
| 재료 | 화강암 석조 외벽 + 목조 내부 |
| 높이 | 주탑 25~35m (추정) |
| 특징 | 기마 광장 (최소 100m×100m) 필수 |
| 마구간 | 왕궁 면적의 1/5 이상 |
| 단열 | 이중 목조 내벽 + 동물 가죽 단열재 |

### 1-2. 귀족 저택

- 석조 1층 + 목조 2~3층 구조
- 1층: 창고·마구간·하인실
- 2층 이상: 가족 거주·응접실
- 지붕: 경사 급한 석판 지붕 (눈 적설 방지)

### 1-3. 평민 주택

- 전체 목조 (귀리·호밀 짚 지붕)
- 단층 혹은 다락 추가
- 가축 공간과 주거 공간 연결 (체온 활용 난방)
- 겨울 이중 문 필수

### 1-4. 마을 우물·광장

- 마을 중심 돌우물 + 나무 덮개
- 겨울 결빙 방지 짚 덮개 전통
- 광장에 말 묶는 기둥 (Hitching Post) 항상 존재
