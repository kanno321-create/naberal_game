## 요약

알드릭 왕국 건축은 호수와의 연결을 최우선으로 한다. "배 모양 지붕"이라는 독특한 지붕 형태와 수면에 직접 면한 포치(수상 포치)가 대표적 특징이다. 석재와 목재를 혼용하며, 은빛 장식과 청회색 외벽이 알드릭 건축의 시각적 정체성이다.

## 핵심 건축 특징

| 요소 | 특징 |
|------|------|
| **지붕** | 배 모양 — 용골을 뒤집은 형태. 호수 바람·강수 대응 최적화 |
| **외벽** | 청회색 석재 또는 회벽. 호수색 창틀·문 |
| **포치** | 수면에 직접 면한 목조 포치. 배 계류 기능 겸비 |
| **기둥** | 은빛 도금 또는 은빛 페인트 기둥. 공예 조각 장식 |
| **창문** | 호수 방향으로 크게. 내륙 방향은 작게 |

## 건물 유형별 양식

### 왕궁·귀족 저택
- 청회색 석조 기초 위 목조 상부 구조
- 배 모양 지붕에 은빛 기와 장식
- 수면 향하는 개인 포치 (계류 시설 포함)
- 담수 진주 상감 문 손잡이·장식판 (귀족 저택 고급 사양)

### 서민 주택
- 목조 주택 + 회벽 마감
- 배 모양 지붕 (간략화). 호수색 창틀
- 수변 주택은 간이 포치 부착
- 내륙 주택은 포치 없음 — 채광 중심

### 어부·수운 시설
- 내구성 목조 기반 부두 건물
- 지붕 경사 급함 (폭우 대응)
- 훈제 시설 전용 굴뚝 — 도시 특유의 연기 경관

### 호수 성당
- 석조 + 배 모양 지붕 결합
- 호수 방향 대형 아치형 창문 — 수면 빛 실내 유입
- 내부 진주 모자이크 장식 (공예 장인 길드 제작)
