## 요약

Sylren 왕국 문장 체계는 농경 문화를 상징하는 금·녹색·진홍을 기본 색으로 삼는다. 밀·포도·태양이 핵심 모티프이며, 귀족 가문마다 왕국 문장의 요소를 변형 계승한다.

---

## 1. 왕국 대문장 (Royal Arms)

| 항목 | 내용 |
|------|------|
| 배경색 | 금색 (Or) |
| 상단 | 진홍 태양 (Solar) — 8개 광선 |
| 중앙 | 녹색 밀 다발 3묶음 (Wheat Sheaves) |
| 하단 | 청색 물결 2줄 (Soranth 강 상징) |
| 테두리 | 진홍 외곽선 |
| 왕관 | 금빛 농경 왕관 (보리 이삭 장식) |

---

## 2. 공작령·귀족 가문 문장

| 가문 | 배경 | 주 모티프 | 색상 |
|------|------|----------|------|
| **House Varentis (왕가)** | 금색 | 황금 밀 다발 3묶음 + 진홍 강물 | 금·진홍 |
| **House Caelindra (Auravale)** | 청색 | 황금 포도송이 + 은빛 강물 | 청·금·은 |
| **House Pelorn (Herdsland)** | 녹색 | 질주하는 갈색 말 + 금빛 양모 | 녹·금·갈색 |
| **House Darnwall (Forestmark)** | 어두운 녹색 | 검은 참나무 + 은빛 화살 | 암녹·흑·은 |
| **Mervaine (Southvale·추정)** | 진홍 | 은빛 태양 + 금빛 보리 | 진홍·은·금 |

---

## 3. 기사단 문장

| 기사단 | 배경 | 문장 |
|--------|------|------|
| **금빛 보리단** | 금색 | 직립한 녹색 보리 이삭 3대 |
| **남풍 기사단** | 진홍 | 금빛 태양 광선 + 바람을 가르는 창 |

---

## 4. 문장 사용 규칙

| 상황 | 규칙 |
|------|------|
| 왕국 공식 문서 | 왕국 대문장 봉인 필수 |
| 귀족 서신 | 가문 문장 봉인 |
| 군기 | 기사단 문장 + 왕국 대문장 조합 |
| 상인 면허 | 왕국 소형 문장 날인 |
| 수확 대축제 | 왕국 대문장 + 금빛 보리단 합동 깃발 |

---

## 5. 색상 의미론

| 색상 | 상징 | 사용 맥락 |
|------|------|---------|
| 금색 (Or) | 풍요·왕권·태양 | 왕가·왕국 배경 |
| 녹색 (Vert) | 평원·농경·생명 | 공작 가문 배경 |
| 진홍 (Gules) | 태양·용기·열정 | 기사단·남방 문화 |
| 청색 (Azure) | 강·물·외교 | Auravale·수운 관련 |
| 은색 (Argent) | 순수·지식·달 | 학자·교회 관련 가문 |
