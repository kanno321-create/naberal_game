## 요약

Sylren 건축은 남부 온난 기후와 라틴·지중해 문화를 직접 반영한다. 테라코타 기와 지붕·포치(현관 기둥 복도)·중정 분수가 핵심 요소다. 내부로 열린 구조와 통풍 중시 설계가 특징이다.

---

## 1. 건축 핵심 요소

| 요소 | 설명 | 계층별 사용 |
|------|------|-----------|
| **테라코타 기와 지붕** | 붉은·주황색 구운 점토 기와 | 전 계층 (규모 차이) |
| **포치 (Porticus)** | 기둥 지지 외부 복도·반야외 공간 | 귀족 이상 |
| **중정 분수 (Atrium)** | 건물 중앙 열린 안뜰 + 분수 또는 우물 | 귀족·상인 |
| **석조 기단** | 건물 하부 석회석·사암 기단 | 영구 건물 필수 |
| **아치 구조** | 출입구·회랑·수로 교각 | 도시 건물 표준 |
| **벽면 프레스코** | 농경·수확·포도 문양 회화 | 귀족·교회 내부 |

## 2. 건물 유형별 특성

### 2-1. 왕성·귀족 저택

| 항목 | 내용 |
|------|------|
| 외벽 | 석회석·사암 흰색 또는 황토색 |
| 지붕 | 테라코타 기와 (주황~적갈) |
| 포치 | 기둥 8~12개 열주 |
| 중정 | 대형 분수·포도나무 파고라·석조 벤치 |
| 창문 | 아치형 셔터 창문·여름 통풍 우선 |
| 규모 | 2~3층·층별 테라스 |

### 2-2. 대성당·교회

| 항목 | 내용 |
|------|------|
| 외관 | 성좌국 양식 + Sylren 테라코타 지붕 혼합 |
| 내부 | 농경 성화·수확 천장화·스테인드 글라스 |
| 종탑 | 테라코타 원뿔 지붕 종탑 |

### 2-3. 도시 상업 건물

| 항목 | 내용 |
|------|------|
| 외벽 | 소석회 도료 흰색 또는 크림색 |
| 1층 | 상점·작업장 개방형 |
| 2~3층 | 거주 구역·포치 발코니 |
| 도로면 | 아치형 회랑 (비·햇빛 차단) |

### 2-4. 농촌 가옥

| 항목 | 내용 |
|------|------|
| 소재 | 일부 석조·나머지 점토 벽돌·목재 보조 |
| 지붕 | 테라코타 기와 또는 건초·갈대 (빈민) |
| 중정 | 가축 우리 겸용 소형 안마당 |
| 규모 | 단층·방 2~4개 |
