## 요약

Oryn 왕국 건축은 목재와 석조를 혼합하되, "나무를 해치지 않는 건축" 원칙을 지킨다. 숲 마을은 수백 년 된 나무를 제거하지 않고 건물을 그 주변에 지으며, 왕도 Orynthil 의 대형 건물도 살아있는 나무를 기둥으로 활용하는 특이한 양식이 있다.

---

## 건축 원칙

| 원칙 | 내용 |
|------|------|
| **"나무를 베지 않는다"** | 수령 100년 이상 수목은 제거 금지 · 주변으로 건물 설계 |
| **목재 우선** | 구조재는 목재 · 기초·성벽은 석조 혼합 |
| **자연 통합** | 살아있는 나무 기둥 활용 (Greenspire Temple 등) |
| **수분 대응** | 습한 Orenwald 기후 → 주춧돌 높임 · 목재 방수 수지 처리 |

## 유형별 건축 양식

### 왕궁 (Thornkeep)

| 항목 | 내용 |
|------|------|
| **구조** | 석조 기초 + 대형 목재 골조 + 목재 외벽 |
| **높이** | 3층 주탑 + 2층 익관 |
| **특징** | 안마당 고대 참나무 3그루 중심 → 지붕 틈을 열어 수목 생장 허용 |
| **지붕** | 녹색 이끼 덮인 석판 지붕 |

### 정령 사원 (Greenspire Temple)

| 항목 | 내용 |
|------|------|
| **구조** | 자연 암반 기초 + 살아있는 대형 나무 8그루를 기둥으로 |
| **특징** | 인공 기둥 없음 · 나무가 죽으면 사원도 개축 |
| **지붕** | 가지와 넝쿨로 엮은 천장 + 석판 부분 복합 |

### 귀족 저택

| 항목 | 내용 |
|------|------|
| **구조** | 2~3층 목조+석조 혼합 |
| **특징** | 녹색 담쟁이 외벽 유도 · 정원 필수 |
| **지붕** | 초가(草家) 또는 목재 판 지붕 |

### 서민 주택

| 항목 | 내용 |
|------|------|
| **구조** | 단층 목조 · 주춧돌 석조 |
| **특징** | 마을 전체가 나무 사이에 끼워 넣듯 배치 |
| **지붕** | 초가 · 갈대 |

### 요새·성벽

| 항목 | 내용 |
|------|------|
| **구조** | 석조 주 + 목재 순찰로·방어시설 |
| **특징** | 목재 흉장(胸墻)이 빠른 소모 · 석조 핵심부 항구 |

## 숲 마을 특유 배치

Orenwald 내부 마을은 "나무를 해치지 않는" 원칙에 따라 수목 사이 공간을 먼저 파악하고 집터를 결정한다:

```
[나무] [집] [나무]   [나무]
        |
  [마을 우물]
        |
[집] [창고] [나무] [집]
```

- 도로가 나무 사이 S자형으로 구불구불함 → 방어 시 매복 유리
- 각 집에서 숲까지 거리 5m 이내 → 긴급 피신 용이
