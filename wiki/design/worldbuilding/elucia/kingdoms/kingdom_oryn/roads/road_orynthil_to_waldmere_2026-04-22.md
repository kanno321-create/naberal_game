# 숲 내부 도로 (Forest Interior Road)
## Orynthil → Waldmere

| 항목 | 내용 |
|------|------|
| **노선** | Orynthil 동문 → Thorngate → Ironwood → Waldmere |
| **거리** | ~80km |
| **노면** | 자갈 부분 + 목재 판자 구간 (습지 구간) |
| **폭** | 3~4m (숲 내 제한) |
| **치안** | 숲 파수단 전담 · 숲 안내인 동행 권장 |
| **이용자** | 목재 운반 마차 · 약초 채취 길드 · 교황청 이단 심문관 (Waldmere 상주 위해) |
| **특이점** | Ironwood 이후 나침반 오류 발생 구간 진입 · 안내인 없이 이동 공식 금지 · 도로 양쪽이 높은 수목으로 막혀 낮에도 어두움 |

## 경유지 마커

```
Orynthil ──40km── Thorngate ──20km── Ironwood ──20km── Waldmere
```

## 전략적 의미

- Orenwald 목재 경제 핵심 도로
- 이단 심문관 Waldmere 상주 접근로 — 왕국과 교황청 긴장 축선
- Waldmere 이후 계속 이어지면 中林→深林으로 연결 (공식 이후 구간 없음)
