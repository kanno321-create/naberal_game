# 동부 릉길 (Eastern Ridge Road)
## Orynthil → Veilorngate

| 항목 | 내용 |
|------|------|
| **노선** | Orynthil 동문 → Thorngate → Stoneford → Veilorngate |
| **거리** | ~130km |
| **노면** | 자갈 포장 (왕도~Thorngate) + 부분 석판 (Stoneford~Veilorngate) |
| **폭** | 5~7m (군사 이동 고려 · 주요 군사 도로) |
| **치안** | Veilorn March 영주군 + 녹색 사슴단 합동 순찰 |
| **이용자** | 군사 이동 · Karzor 상인 (Veilorngate 이후) · 왕실 전령 |
| **특이점** | Stoneford 근처부터 Veilorn Ridge 가 보이기 시작함 · 이 구간에서 나침반 오류 보고 드물게 있음 |

---

## 경유지 마커

```
Orynthil ──40km── Thorngate (채취 구역 검문) ──50km── Stoneford ──40km── Veilorngate
```

---

## 전략적 의미

- Oryn 왕국의 가장 중요한 군사·교역 도로
- 동부 Karzor 방면 위협 시 왕실 증원군 신속 이동 경로
- Karzor 상인들이 이 도로를 통해 Orynthil 까지 진입 가능 (Veilorngate 검문 통과 시)
