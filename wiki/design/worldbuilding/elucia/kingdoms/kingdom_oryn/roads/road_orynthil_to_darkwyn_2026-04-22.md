# 오렌왈드 북부로 (Orenwald Northern Road)
## Orynthil → Darkwyn

| 항목 | 내용 |
|------|------|
| **노선** | Orynthil 북문 → Barkmill → Hazelwick → Darkwyn |
| **거리** | ~120km |
| **노면** | 초반 자갈 + 중후반 목재 판자 + 흙길 혼재 |
| **폭** | 3~4m |
| **치안** | 녹색 사슴단 + Orenwarden 영주군 교대 순찰 |
| **이용자** | 목재·약초 운반 · Maerith 접경 방향 교역상 |
| **특이점** | Darkwyn 근처에서 나침반 완전 불능 구간 진입 · 이 도로 자체가 지역 주민 기억에만 의존 (지도상 부정확) |

Orynthil ──35km── Barkmill ──40km── Hazelwick ──45km── Darkwyn

## 전략적 의미

- Maerith 왕국과의 북부 접경 방면 도로
- Darkwyn 의 자기 이상 현상 때문에 외부 상인 기피 경향
- 왕국 내 가장 고립된 도시 연결 도로
