# 소란스 남부 수운로 (Soranth Southern Road)
## Orynthil → Soranwatch

| 항목 | 내용 |
|------|------|
| **노선** | Orynthil 남문 → Willowrun → Mossford → Reedholm → Soranwatch |
| **거리** | ~110km |
| **노면** | 자갈 포장 부분 + 강변 흙길 |
| **폭** | 4~5m |
| **치안** | 녹색 사슴단 + Soranmere 영주군 |
| **이용자** | 목재 뗏목 보조 육로 · 어획물 운반 마차 · Sylren 방면 교역상 |
| **특이점** | Soranth 강 오른쪽 강변 따라 이어짐 · 뗏목 진행 방향과 동일 경로 · 강 범람 시 침수 위험 구간 있음 |
