# Darkwyn (다크윈) — Orenwald 북부 고립 도시

| 항목 | 내용 |
|------|------|
| **위치** | Orenwald 북부 · Maerith 경계 근처 |
| **규모** | 인구 약 8,000~12,000 |
| **기능** | 북부 숲 채취·광물·희귀 수종 목재 |
| **어원** | *dark* (어두운) + *-wyn* (켈트형 호수·신비) = "어두운 물의 도시" |
| **분위기** | 대낮도 어두운 깊은 숲 속. 나침반이 작동 불량인 지역으로 소문 |
| **특징** | "오렌왈드에서 나침반이 제멋대로 돈다" 소문의 원산지 |
| **행정** | Orenwarden 공작령 북쪽 거점 · Maerith 접경 인접 |
| **주요 인물** | Darkwyn 성주 (이름 미확정) · 고참 숲 파수단원 다수 상주 |
| **특이 현상** | 나침반 오류 반경 약 15km — 자기 이상 원인 불명 (추정: 지하 광물층 · 타종족 관련 구전 병존) |
| **Rev.3 접점** | 고립된 북부 도시 · 기이한 사건 발생지 가능
