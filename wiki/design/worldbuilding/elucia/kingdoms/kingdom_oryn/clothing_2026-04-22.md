## 요약

Oryn 왕국의 의상은 숲·자연·사냥 문화를 반영한 켈트·슬라브 혼합 양식이다. 녹색·갈색·자연 중간색 계열을 주색으로 하며, 가죽과 린넨을 주 소재로 사용한다. 잎·나뭇가지·사슴 뿔 모티프 장식이 계급을 불문하고 사용된다.

---

## 계급별 의상

### 왕실·고귀족

| 항목 | 내용 |
|------|------|
| **주 소재** | 고급 린넨 + 벨벳 + 수자수 비단 (수입품) |
| **색** | 깊은 녹색 + 금 + 진한 갈색 |
| **장식** | 금실 잎 자수 · 사슴 뿔 버클 · 호박(琥珀) 보석 |
| **특징** | 외투 안감에 모피 (여름은 얇은 린넨 안감) |
| **왕의 예복** | 참나무 잎 금관 + 녹색 외투 · 안에 사슴 가죽 갑옷 상시 착용 |
| **왕비 예복** | 깊은 녹색 긴 드레스 + 은 장식 + 야생화 관 |

### 중산층·상인

| 항목 | 내용 |
|------|------|
| **주 소재** | 중급 린넨 + 가죽 |
| **색** | 중간 녹색 + 갈색 + 자연 베이지 |
| **장식** | 목재 버클·나무 단추 · 간단한 잎 문양 자수 |
| **특징** | 활동성 중시 · 깊은 숲 작업 시 갈색 위주 |

### 서민·목부·어부

| 항목 | 내용 |
|------|------|
| **주 소재** | 거친 린넨 + 양가죽 + 수지 처리 방수포 |
| **색** | 자연 갈색 + 베이지 + 먹색 |
| **장식** | 없거나 단순한 나뭇가지 모티프 봉제 |
| **특징** | 뗏목꾼은 방수 가죽 앞치마 필수 |

### 기사·군인

| 항목 | 내용 |
|------|------|
| **녹색 사슴단** | 녹색 외투 + 가죽 갑옷 + 사슴 뿔 투구 장식 |
| **숲 파수단** | 갈색·초록 위장 망토 + 경량 가죽 갑옷 |
| **일반 병사** | 갈색 린넨 + 가죽 가슴받이 |
