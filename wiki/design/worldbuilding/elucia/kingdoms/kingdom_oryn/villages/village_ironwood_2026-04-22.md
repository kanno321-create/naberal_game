# Ironwood (아이언우드) — 경목 채취 전문 취락

| 항목 | 내용 |
|------|------|
| **위치** | Orenwald 중림 · Waldmere 북방 10km |
| **인구** | 350~600 |
| **산업** | 대형 경목 채취 · 성곽·대형 건물용 목재 생산 |
| **어원** | *iron* (철처럼 단단한) + *wood* (목재) — 경목 특성 |
| **분위기** | 도끼 소리 · 목재 향 · 거대한 나무 그루터기들 |
| **특이점** | 성좌국·Sylren 으로 수출되는 Orenwald 최대 건축재 산지 · 왕국 목재 수출 1위 마을 |
| **서사 접점** | 교황청 과세 분쟁 핵심 지점 — 이 마을 목재에 세금 분쟁 |
