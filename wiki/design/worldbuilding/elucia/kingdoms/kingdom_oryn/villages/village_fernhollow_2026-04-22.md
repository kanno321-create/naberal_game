# Fernhollow (펀할로우) — Orenwald 외림 고사리 마을

| 항목 | 내용 |
|------|------|
| **위치** | Orenwald 외림 서쪽 가장자리 · Orynthil 남서 18km |
| **인구** | 300~500 |
| **산업** | 고사리·양치류 채취 · 약초 초기 가공 · 목재 수지 채취 |
| **어원** | *fern* (고사리) + *-hollow* (골짜기 취락) |
| **분위기** | 항상 녹음 짙음 · 숲 안개 상시 · 발 아래 고사리 가득 |
| **특이점** | 마을 입구 수령 300년 참나무가 수호목 · 봄의 눈뜸 때 주민 전체가 이 나무 포옹 의식 |
| **서사 접점** | 약초 정보 수집 경유지 가능 |
