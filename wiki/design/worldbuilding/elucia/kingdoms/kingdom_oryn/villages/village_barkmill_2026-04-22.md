# Barkmill (바크밀) — 수피 제지 취락

| 항목 | 내용 |
|------|------|
| **위치** | Orenwald 외림 북부 · Orynthil 북쪽 35km |
| **인구** | 200~380 |
| **산업** | 자작나무 수피 채취 · 수피 종이 제조 · 문서·지도용 재료 |
| **어원** | *bark* (수피) + *-mill* (제분·제조 공장) |
| **분위기** | 흰 자작나무 숲 · 종이 냄새 · 조용한 작업음 |
| **특이점** | Oryn 왕국 내 유일한 수피 종이 생산지 · House Ashwood 약초 기록서도 이 마을 제지 사용 |
| **서사 접점** | 고문서·지도 입수 경로 |
