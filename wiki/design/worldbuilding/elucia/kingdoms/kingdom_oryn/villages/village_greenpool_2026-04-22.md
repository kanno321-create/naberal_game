# Greenpool (그린풀) — 용천수 치유 마을

| 항목 | 내용 |
|------|------|
| **위치** | Orenwald 중부 숲 안 · 용천수 발원지 |
| **인구** | 150~300 |
| **산업** | 용천수 채취·배송 · 치유 목욕 · 희귀 수생 약초 채취 |
| **어원** | *green* (녹색) + *-pool* (웅덩이) |
| **분위기** | 맑은 녹색 물 · 새소리 풍부 · 치유 분위기 |
| **특이점** | "이 물을 마시면 상처가 빨리 낫는다" 는 소문 · 실제로 약초 성분이 우물에 침출 (추정) |
| **이름 모를 학자 마법 흔적** | "어느 해 마을 어른이 가르쳐 준 물 정화 주문을 이 우물에 처음 걸었다" 는 구전 (간접 단서) |
| **서사 접점** | 나이트 부상 치료 경유지 · 치유 약초 수집 |
