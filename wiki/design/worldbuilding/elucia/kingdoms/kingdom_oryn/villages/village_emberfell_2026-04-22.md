# Emberfell (엠버펠) — 목탄 생산 취락

| 항목 | 내용 |
|------|------|
| **위치** | Orenwald 외림 서쪽 · 성좌국 접경 근처 |
| **인구** | 300~520 |
| **산업** | 목탄(숯) 생산 · 연료 수출 · 소형 제철 보조 |
| **어원** | *ember* (잉걸불) + *-fell* (산지·취락) |
| **분위기** | 항상 연기 냄새 · 검게 그을린 주민들 · 밤에 굽는 목탄 가마 빛 |
| **특이점** | 성좌국 방면으로 목탄 수출 → 성좌국 도시 난방에 기여 · Oryn 내에서 성좌국 의존도 가장 높은 마을 |
| **서사 접점** | 성좌국·Oryn 관계 경제 조사 |
