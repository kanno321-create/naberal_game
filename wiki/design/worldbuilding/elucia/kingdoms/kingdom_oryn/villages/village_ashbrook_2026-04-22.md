# Ashbrook (애시브룩) — 목재 뗏목 기착 마을

| 항목 | 내용 |
|------|------|
| **위치** | Soranth 강 상류 지류 뗏목 기착점 · Orynthil 동북 30km |
| **인구** | 450~700 |
| **산업** | 목재 뗏목 정박·보수 · 뗏목꾼 숙식 제공 · 소형 목재 가공 |
| **어원** | *ash* (물푸레나무) + *-brook* (시냇물) |
| **분위기** | 강변 목재 냄새 · 뗏목꾼들의 노래 소리 · 강 안개 |
| **특이점** | "강 상류에서 내려오는 뗏목에 가끔 타종족 물건이 실려온다" 는 소문 · 아무도 확인 못 함 |
| **서사 접점** | Soranth 강 수운 정보 수집 경유지 |
