# Thorngate (손게이트) — 채취 구역 경계 관문 마을

| 항목 | 내용 |
|------|------|
| **위치** | Orenwald 외림·중림 경계 · 채취 허가 검문 지점 |
| **인구** | 200~400 |
| **산업** | 채취 허가증 검문·발급 대행 · 안내인 공급 · 숙식 |
| **어원** | *thorn* (가시 방벽) + *-gate* (관문) |
| **분위기** | 출입 통제 분위기 · 숲 파수단 상시 2명 이상 상주 |
| **특이점** | 불법 深林 진입 시도자가 매월 2~5명 적발됨 (교황청 이단 심문관 포함) |
| **서사 접점** | 숲 파수단과 나이트의 첫 조우 지점 가능 |
