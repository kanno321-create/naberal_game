# Deerbend (디어벤드) — 전통 사냥꾼 취락

| 항목 | 내용 |
|------|------|
| **위치** | Orenwald 외림 북부 · Darkwyn 남서 20km |
| **인구** | 350~600 |
| **산업** | 사슴 수렵·모피 가공 · 뼈·뿔 공예 · 건조 육포 |
| **어원** | *deer* (사슴) + *-bend* (강 굽이·취락) |
| **분위기** | 건조된 사슴 모피와 뼈 공예품이 처마마다 걸림 · 말린 고기 냄새 |
| **특이점** | 수백 년 전통 사냥 기법 보존 · 왕실 Hart's Day 행사에 전문 사냥꾼 공급 |
| **서사 접점** | 사냥꾼 정보원 · Orenwald 內林 경로 안내 |
