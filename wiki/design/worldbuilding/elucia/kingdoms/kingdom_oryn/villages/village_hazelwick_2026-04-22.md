# Hazelwick (헤이즐위크) — 개암나무 수액 채취 마을

| 항목 | 내용 |
|------|------|
| **위치** | Orenwald 외림 남부 · Deepwald 공작령 접경 |
| **인구** | 250~450 |
| **산업** | 개암 열매·수액 채취 · 약용 수액 가공 · 꿀벌 방목 |
| **어원** | *hazel* (개암나무) + *-wick* (취락) |
| **분위기** | 가을엔 개암 냄새 · 벌집 윙윙 · 아이들이 개암 줍는 소리 |
| **특이점** | 마을 북쪽 경계에 수령 500년 개암나무 군락 — 주민이 "할배 나무" 라 부름 (우연의 명칭 · Q-CORE 2 구조 직접 서술 아님) |
| **서사 접점** | 약용 수액 채취 · Deepwald 접경 정보 |
