## 요약

Oryn 왕국의 식문화는 숲 수렵·담수 어업·약초 채취를 기반으로 한다. 사슴 고기와 담수 생선이 주 단백질이며, 약초 스튜가 일상식의 핵심이다. 꿀술(Mead)은 왕국의 국민 음료로 모든 계층이 즐긴다. 산딸기·야생 베리류가 디저트·발효 재료로 폭넓게 쓰인다.

## 대표 요리

| 요리명 | 재료 | 특징 |
|--------|------|------|
| **오렌왈드 사슴 스튜 (Orenwald Stew)** | 사슴 고기 + 약초 7종 + 버섯 + 뿌리채소 | 진한 향신료 · 겨울 필수 음식 |
| **Soranth 구운 농어** | 농어 + 레몬 허브 + 오렌왈드 소금 | 강변 도시 일상식 |
| **숲 파수단 빵 (Warden's Loaf)** | 도토리 가루 + 통밀 + 꿀 | 순찰 시 휴대식 · 단단하고 오래감 |
| **왕실 사냥 연회 구이 (Hart's Feast)** | 통 사슴 + 7종 약초 마리네이드 + 꿀글레이즈 | 사냥 축일·성인식 전용 |
| **꿀술 (Greenwood Mead)** | Orenwald 야생화 꿀 + 물 + 야생 이스트 | 모든 계층 일상 음료 |
| **산딸기 잼 린넨 케이크** | 린넨씨 반죽 + 산딸기 잼 | 봄의 눈뜸 축제 전통 과자 |
| **약초 치유탕 (Herb Broth)** | 사슴뼈 + 약초 15종 | 부상·질병 회복식 |

## 식사 관습

| 관습 | 내용 |
|------|------|
| **사냥 전 단식** | 사냥 축일 전날 저녁 금식 → 당일 사냥 후 Hart's Feast |
| **약초 스튜 공유** | 마을 단위 공동 솥에서 끓여 나눔 → 공동체 결속 |
| **꿀술 건배 구호** | "숲이 우리를 지키고, 우리가 숲을 지킨다" |
| **정령제 음식** | 희 사슴 간을 정령께 제물로 올린 후 남은 부위로 연회 |
| **이름 모를 학자 마법 흔적** | 일부 마을에서 "이름 모를 노인이 가르쳐 준 식품 보존 주문" 사용 (간접 단서) |
