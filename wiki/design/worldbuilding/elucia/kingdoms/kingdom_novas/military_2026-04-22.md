## 요약

Novas 군제는 **모병제** 기반이다. 인구 소왕국이므로 징병제는 사실상 불가. 대신 통행세 수입을 재원으로 전문 직업군인을 모집·유지한다. 관문 수비·사막 정찰·해안 어업 민병의 3계층 구조.

---

## 1. 군제 개요

| 항목 | 내용 |
|------|------|
| **징병 방식** | 모병제 (지원제) |
| **재원** | 통행세 수입 → 병력 유지비 |
| **총 병력 (추정)** | 정규군 3,000~5,000 · 민병 2,000~4,000 |
| **지휘 체계** | 국왕 → 3개 공작 → 기사단 단장 → 각 부대 |

## 2. 3계층 군사 구조

### 2-1. 관문 수비대 (Pass Guard)

| 항목 | 내용 |
|------|------|
| **주 임무** | Azim Pass 북문·Duskgate 성벽 방어 |
| **규모** | 1,500~2,500명 (추정) |
| **장비** | 붉은 사암 색 갑옷·장방패·단검 |
| **주둔지** | Duskgate Pass Gate 구역 + Azimfront 공작령 요새 |
| **특성** | 교대 근무 24시간 유지. 관문 수호단 기사 아래 배치 |

### 2-2. 사막 정찰대 (Desert Scouts)

| 항목 | 내용 |
|------|------|
| **주 임무** | Karzor 접경·사막 감시·밀수 추적 |
| **규모** | 50~120명 정예 (추정) |
| **장비** | 경갑·단검·원거리 활·사막 대비 후드 망토 |
| **주둔지** | Sandwatch |
| **특성** | 사막 척후단 소속. 장기 야전 전문 |

### 2-3. 해안 어업 민병 (Coastal Militia)

| 항목 | 내용 |
|------|------|
| **주 임무** | 해안 어업 보호·해적 대응 |

| **장비** | 간이 무장·어선 전환 전선 |

| **특성** | 평시에는 어업 종사. 비상 시 무장 소집 |

## 3. 군사 특기·전술

| 전술 | 설명 |
|------|------|
| **관문 방어전** | Duskgate 이중 성문 활용 · 낙문·투석기 배치 |
| **사막 지구전** | 물·식량 장기 비축. 오아시스 지점 통제 |

| **해상 방어** | 어선 전환 전선으로 소규모 해적 격퇴 |
