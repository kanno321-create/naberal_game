# Ironbell (아이언벨)

| 항목 | 내용 |
|------|------|
| **위치** | Duskgate → Duskwatch 왕도로 km 45 |
| **인구** | 약 400~600 |
| **산업** | 철기 제련·무기 수리·말 편자 |
| **어원** | *iron* (철) + *bell* (종·교차로) |
| **분위기** | 대장간 소리가 아침부터 울림. 관문 수호단 장비 수리 중간 기착지 |
| **특이점** | 관문 수호단 갑옷 긴급 수리 공방 4개 운영. 왕국 내 유일한 전문 군장 수리 마을 |
