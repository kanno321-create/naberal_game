# Salthallow (솔트할로우)

| 항목 | 내용 |
|------|------|
| **위치** | 왕도 동쪽 15km · 사막 경계 소금 사구 지대 |
| **인구** | 약 150~250 |
| **산업** | 천연 소금·전갈 사향 소금 채취 |
| **어원** | *salt* (소금) + *hallow* (신성한 땅·오목한 지형) |
| **분위기** | 흰 소금 결정이 땅을 덮음. 더운 낮엔 신기루 |
| **특이점** | "전갈 사향 소금" 의 유일 채취지. 관문제 양 구이 향신료의 핵심 원료 |
