# Dustfall (더스트폴)

| 항목 | 내용 |
|------|------|
| **위치** | 사막 상단로 km 40 · 모래 사구 경계 |
| **인구** | 약 80~150 |
| **산업** | 낙타 거래·사막 안내인 |
| **어원** | *dust* (먼지·모래) + *fall* (낙하·가장자리) |
| **분위기** | 바람이 불면 모래가 지붕 위로 낙하함. 집들이 반쯤 모래에 묻혀있음 |
| **특이점** | Karzor 낙타 상인이 Elucia 측에 낙타를 파는 유일한 비공식 거래 장소 |
