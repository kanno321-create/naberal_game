# Ashridge (애시리지)

| 항목 | 내용 |
|------|------|
| **위치** | Duskgate → Moorfield 구릉로 km 40 |
| **인구** | 약 350~500 |
| **산업** | 목축 집산·건초 거래·마차 수리 |
| **어원** | *ash* (물푸레나무·재) + *ridge* (능선) |
| **분위기** | 구릉 능선 위 뻔한 목축 마을. 항상 양 냄새 |
| **특이점** | 구릉로 주요 중간 기착지. 여관·마구간 운영으로 생계 |
