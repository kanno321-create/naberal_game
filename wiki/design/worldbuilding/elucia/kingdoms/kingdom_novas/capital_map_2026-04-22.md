## 요약

**Duskgate** (더스크게이트) 는 Novas 왕국의 왕도이자 Azim Pass 북단 공식 관문 도시다. 인구 약 20,000~35,000. 붉은 사암 성벽과 Karzor 식 아치 문이 혼합된 독특한 건축으로 유명하다. 도시 전체가 통행세 수집·검문·환전·숙박을 중심으로 구성된 **관문 경제 도시**다.

## 구역 상세

### 1. 왕성구 — Voss 궁성 (Crown Quarters)

| 항목 | 내용 |
|------|------|
| 위치 | 도시 북서 고지대 · 가장 방어하기 쉬운 위치 |
| 핵심 건물 | Voss 왕궁 · 국왕 집무실 · 근위대 막사 · 왕실 마굿간 |
| 건축 양식 | 붉은 사암 성채 + Karzor 식 아치형 문루 |
| 특이점 | 왕비 Ilena 의 전용 동방 양식 거처 (궁 동쪽 별채) — 반공식 지위 상징 |
| 접근 제한 | 국왕 허가 없이 출입 불가 |

### 2. 귀족구 — Highward

| 항목 | 내용 |
|------|------|
| 위치 | 왕성구 동쪽 · 고지대 |
| 핵심 건물 | Duskwatch 공작 저택 · 성좌국 감독관 관저 · 3~5개 고위 귀족 저택 |
| 특이점 | 성좌국 감독관 관저 = 통행세 수익 일부 수령처. 이 건물을 두고 왕가와 성좌국의 냉랭한 외교 긴장 상존 |
| 방어 | 독자적 담장·경비대 |

### 3. 대성당구 — Elucia 성좌 대성당 (Cathedral Quarter)

| 항목 | 내용 |
|------|------|
| 핵심 건물 | 성좌국 정규 교회 (소규모 대성당) · 신관 거처 · 병원 |
| 규모 | 도시 타 왕국 대성당보다 작음 — 성좌국이 Novas 를 완전 신뢰하지 않음의 반영 |
| 특이점 | 지하 기록실 = 양심파 교회 기록 파편 보존 가능성 (추정·Q-CORE 반영) |
| 일상 | 이름 없는 학자가 가끔 들러 서민 약 처방을 도와준다는 소문 (추정) |

### 4. 상업구 — Market Cross

| 항목 | 내용 |
|------|------|
| 위치 | 도시 중심 · 남북문 연결 도로 교차점 |
| 핵심 시설 | 관문 상인 길드 본부 · 환전소 (3개) · 대형 여관 (2개) · 향신료 시장 · 사막 과일 노점 |
| 특산품 | 석류·대추야자·사막 향신료 (Karzor 경유) · 붉은 사암 장신구 |
| 분위기 | 라틴·Karzor 상인 혼재. 두 언어 동시에 들림 |

### 5. 관문구 — Pass Gate (통관 구역)

| 항목 | 내용 |
|------|------|
| 위치 | 도시 남쪽 · 성벽 바로 안쪽 |
| 핵심 시설 | Azim Pass 북문 관문 · 세관 건물 · 통행세 징수소 · 수비대 막사 (관문 수호단 본부) · 격리 구역 |
| 규모 | 도시 전체 면적의 15% 이상 — 왕국 가장 중요한 경제 인프라 |
| 특이점 | 24시간 교대 근무 · 성좌국 감독관이 세관 내 전용 집무실 보유 |
| 격리 구역 | 전염병 의심자·분쟁 상인 임시 구금 |

### 6. 동방인 구역 — Eastward

| 항목 | 내용 |
|------|------|
| 위치 | 관문구 동쪽 · 외성 내부 |
| 핵심 시설 | Karzor 상단 숙소 · 사막 식 카페 · 향신료 창고 · 비공식 통역사 거리 |
| 주민 | Karzor Sabin 출신 상인·외교관 수행원·혼혈인 |
| 분위기 | 도시 내 가장 이국적 구역. 사막 향초 연기·낙타 냄새. 밤에는 Karzor 음악 |
| 특이점 | 왕비 Ilena Sabin-Voss 의 친족이 이 구역 숙소를 거점으로 사용 (추정) |

### 7. 어업·항구구 — Duskway Dock

| 항목 | 내용 |
|------|------|
| 위치 | 도시 서쪽 · Duskway 강 하구 |
| 핵심 시설 | 어시장 · 선착장 · 어부 조합 · 건어물 창고 |
| 생산 | 남동 해안 어업의 내륙 집산지. 청어·잉어·건조 생선 |
| 특이점 | 강 하구 어부들은 Karzor 문화 영향 없이 순수 Elucia 전통 유지 — 왕국 내 문화 이질감의 상징 |

### 8. 빈민·노동자구 — Dustend

| 항목 | 내용 |
|------|------|
| 위치 | 도시 동남 외곽 · 성벽 외부 반쯤 |
| 주민 | 변경 노동자·실직 병사·Azim Pass 밀수 적발자·타종족 소수 |
| 특이점 | 공식 지도에 없는 무허가 판잣집 구역. 왕실 행정이 미치지 않음 |
| 위험도 | 야간 순찰 없음 · 갈취·사기 빈번 |
| 서사 활용 | 주인공이 신분 숨기고 잠입하기 좋은 장소 |
