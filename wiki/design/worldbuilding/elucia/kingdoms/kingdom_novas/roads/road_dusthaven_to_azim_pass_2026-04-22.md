# 사막 상단로 (Dusthaven → Azim Pass)

## 원전 인용 증명

### [필독 1] cities/city_dusthaven_2026-04-22.md
> "Karzor 상인들이 합법적으로 체류하며 거래하는 유일한 도시"
— Dusthaven 기능 확정

---

## 요약

Dusthaven 무역 도시에서 Azim Pass 까지 이어지는 **사막 상단 전용 우회로**. 왕도 관문로를 피해 직접 Pass 로 이동하려는 상단이 사용하는 비공식 경로가 공식화된 것. 모래 지형이 험해 낙타 전용 구간이 포함된다.

---

## 도로 정보

| 항목 | 내용 |
|------|------|
| **길이** | ~55km (추정) |
| **포장** | 없음 · 낙타 발자국 고정 황토 길 |
| **노면** | 폭 불규정 |
| **경유지** | Drysand 마을 · Dustfall 마을 |
| **치안** | 사막 척후단 간헐 순찰. 완전 보장 없음 |
| **위험** | 모래폭풍·밀수업자·타종족 유랑민 |

---

## 주요 지점

| 위치 | 특이점 |
|------|--------|
| **Drysand 마을 (km 20)** | 비공식 물·식량 보급 기착지 |
| **사막 척후단 감시탑** | km 35 지점. 야간 봉화 신호 |
| **Azim Pass 서쪽 진입** | 공식 관문보다 서쪽 진입 → 세관 우회 의혹 상존 |

## 서사 접점

- 주인공이 세관 검문 회피 시 이 경로 선택 가능
- 밀수 상단과 접촉 장소로 활용
