## 요약

Novas 왕국 의상은 Elucia 라틴 중세 기반에 Karzor 사막 실용 의복이 혼합된 독특한 스타일이다. **후드 달린 망토**가 남녀 모두의 기본 아이템이며, 모래바람·강한 햇볕 차단을 위한 실용적 디자인이 귀천 구분 없이 공유된다.

## 1. 계층별 의상

### 왕족·고위 귀족

| 항목 | 내용 |
|------|------|
| **기본 구성** | 라틴 식 긴 튜닉 + 허리 가죽 벨트 + 진홍 망토 |
| **후드** | 비단·양모 혼방 후드 — 의식 시 내림, 이동 시 착용 |
| **색** | 진홍·금·갈색 — 왕국 3색 |
| **장신구** | 모래시계 문양 브로치 + Karzor 식 팔찌 (Ilena 계열 패션 영향) |
| **여성** | 허리 강조 코르셋·라틴 드레이프 + 동방 두건 혼용 |

### 평민·상인

| 항목 | 내용 |
|------|------|
| **기본 구성** | 무명 짧은 튜닉 + 가죽 바지 + 모래색 후드 망토 |
| **망토** | 가장 중요한 아이템 — 낮 햇볕·밤 추위 모두 대응 |
| **색** | 모래색·갈색·회색 — 세탁 비용·사막 오염 고려 |
| **신발** | 가죽 샌들 + 발목 감싸는 천 (사막 모래 유입 방지) |

### 병사·기사

| 항목 | 내용 |
|------|------|
| **갑옷** | 붉은 사암 색 도료 갑옷 + 사막 모래 흡수용 안감 |
| **후드 투구** | 일반 투구 안에 천 후드 착용 — 모래 유입 방지 |
| **망토** | 갈색 군용 망토 — 야영 시 담요 대용 |
| **얼굴 가리개** | 모래폭풍 시 사용하는 천 마스크 (사막 척후단 필수) |

## 2. Karzor 혼합 요소

| 요소 | 기원 | 설명 |
|------|------|------|
| **두건 (Keffiyeh 형)** | Karzor Sabin | 사막 햇볕·모래바람 차단. Eastward 구역 이주민이 도입 |
| **팔찌 중첩 착용** | Karzor 문화 | 왕비 Ilena 이후 귀족 여성 유행 |
| **사막 색 염료** | Karzor 상업 | 석류·사프란 오렌지색 염료. 향신료 시장에서 구입 |
| **발목 방어 천** | Karzor 변경 | 모래 유입 방지 실용 장비 → 전 계층 확산 |
