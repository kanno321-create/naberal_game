## 요약

Novas 건축의 핵심 재료는 **붉은 사암 (Redstone)** 이다. Elucia 서부의 석회암 흰색 건축과 달리 Novas 전역이 붉은색 계열로 덮여 있다. 라틴 지중해 풍 평지붕·아케이드에 Karzor 식 뾰족 아치와 아라베스크 타일 문양이 혼합된 독특한 혼합 양식이 특징이다.

---

## 1. 건축 양식 요소

| 요소 | 기원 | 설명 |
|------|------|------|
| **붉은 사암 석조** | Redstone 지역 | 왕국 전역 건축 재료. 석양에 더욱 진한 붉은빛 |
| **평지붕·옥상 테라스** | 라틴·지중해 | 더운 기후 대응. 저녁 생활 공간으로 활용 |
| **뾰족 아치 (Karzor Arch)** | Karzor 혼합 | 창문·문·통로에 적용. 사막 건축 특유의 냉각 효과 |
| **아케이드 (열주 회랑)** | 라틴 | 시장·귀족 저택 전면. 강한 햇볕 차단 |
| **타일 문양** | Karzor 혼합 | 공공 건물 내부 바닥·벽. 기하학 패턴 |
| **중정 (Courtyard)** | 라틴+Karzor | 건물 내부 중앙 마당. 식물·분수 배치 |

## 2. 건물 유형별 특징

### 요새·성벽 (군사 건축)

| 항목 | 내용 |
|------|------|
| **재료** | 두꺼운 붉은 사암 블록 (1m+ 두께) |
| **특징** | 창문 최소화. 관문 낙문 (철제) |
| **Karzor 영향** | 탑루에 Karzor 식 뾰족 지붕 덮개 (햇볕 차단) |

### 왕궁·귀족 저택

| 항목 | 내용 |
|------|------|
| **재료** | 정제 붉은 사암 + 흰 석회암 창틀 |
| **특징** | 2~3층 · 중정 필수 · 평지붕 테라스 |
| **Karzor 영향** | Ilena 왕비 별채 = 완전 Karzor 내장 (타일·아치·천막형 천장) |

### 대성당·교회

| 항목 | 내용 |
|------|------|
| **재료** | 붉은 사암 + 성좌국 의뢰 흰 석회암 혼합 |
| **특징** | Elucia 성좌 양식 기본 유지하나 규모 작음 |
| **내부** | 스테인드글라스보다 타일 문양 비중 높음 |

### 시장·상업 건물

| 항목 | 내용 |
|------|------|
| **재료** | 붉은 사암 + 목재 |
| **특징** | 아케이드 전면 필수 · 1층 상점·2층 거주 |
| **Eastward 구역** | 천막·직물 지붕 혼용. Karzor 식 개방형 시장 구조 |
