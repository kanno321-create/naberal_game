# Dusthaven (더스트헤이븐) — 황야 대상 무역 도시

> **Wave 4 심화**

| 항목 | 내용 |
|------|------|
| **위치** | Duskmoor 서쪽 · 황야-내륙 경계 |
| **인구** | 약 7,000~11,000 |
| **기능** | 사막 대상(카라반) 물자 보급 · 내륙 연결 · Karzor 상단 공식 체류 |
| **행정** | Duskwatch 공작령 내 · Hael 가 세수 대행 |
| **경제** | 체류세·환전 수수료·창고 임대료. 통행세 다음 두 번째 수입원 |
| **문화** | Karzor 상인 장기 체류 → 향신료 시장·동방 음식점·낙타 거래소 발달 |
| **특이점** | Karzor Sabin 출신 Ilena 왕비의 친족 상인이 이 도시 거점 사용. 비공식 외교 채널 |
| **갈등** | 성좌국 감독관이 이 도시의 Karzor 체류자를 종교적으로 감시. 민간 마찰 |
| **Rev.3 접점** | Act 1 주인공이 Karzor 측 정보 얻을 수 있는 첫 접촉 도시 |
