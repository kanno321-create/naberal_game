## 요약

Elucia 대륙의 산악 고개는 **Norvend 주산맥 3고개** (Greygate·Ironcleft·Whitecrest) 를 핵심으로, **Morncliff Spine 2고개**, **Veilorn Ridge 2고개**, **Duskfell Range 1고개** 를 포함해 총 **8개 주요 고개** 로 구성된다. 모든 고개는 계절에 따라 개폐가 결정되며, 일부는 겨울 4~6개월 완전 폐쇄된다.

## 2. Norvend 3고개 상세

### 2-1. P-1 Greygate Pass (그레이게이트 고개) — 가장 낮은 북부 관문

**고도**: ~1,600m (확정 — mountain_ranges 원전)
**위치**: Norvend 산맥 서쪽 1/3 지점
**연결**: Thaloss 남부 ↔ Vaelin·Moran 방향 Via Imperialis VI-N

| 항목 | 내용 |
|------|------|
| 개방 기간 | 3월 중순 ~ 11월 말 (약 8.5개월) |
| 도로 폭 | 3~4m (암반 절개) |
| 도로 자재 | 석판+자갈 (제국 유지보수) |
| 방어 시설 | 남문·북문 각 1개 요새 (추정) |
| 통행세 | 고도 위험 보상 — 성인 은화 2닢 |
| 주요 물자 | 북↓: 철광·구리·모피 / 남↑: 식량·직물·무기 |

**전략 평가**: 3고개 중 최저고도 — 연간 개방 기간이 가장 길어 통행량 최다. 제국 군사 수송의 주요 북부 관문.

---

### 2-2. P-2 Ironcleft Pass (아이언클레프트 고개) — 가장 높은 군사 요충

**고도**: ~2,100m (확정 — mountain_ranges 원전)
**위치**: Norvend 산맥 중앙
**연결**: Thaloss 내부 (군사 전략적으로 외부 연결보다 내부 요새 기능)

| 항목 | 내용 |
|------|------|
| 개방 기간 | 4월 말 ~ 10월 말 (약 6개월) |
| 도로 폭 | 2~3m (암반 협곡 통과) |
| 도로 자재 | 암반 절개·계단 |
| 방어 시설 | 협곡 양쪽 천연 성벽 + 인공 요새 (추정) |
| 통행세 | 군사 허가 필요 (민간 통행 제한) |
| 특이사항 | "아이언클레프트" — 협곡 벽이 철광석층 노출, 붉은 빛 |

**전략 평가**: 가장 위험하고 가장 방어가 유리한 고개. 외적 침입 시 마지막 방어선.

---

### 2-3. P-3 Whitecrest Saddle (화이트크레스트 안부) — 동쪽 Maerith 연결

**고도**: ~1,900m (확정 — mountain_ranges 원전)
**위치**: Norvend 산맥 동쪽
**연결**: Thaloss 동쪽 ↔ Maerith 고원(Auryn 권역)

| 항목 | 내용 |
|------|------|
| 개방 기간 | 4월 초 ~ 11월 초 (약 7개월) |
| 도로 폭 | 2.5~3.5m |
| 도로 자재 | 자갈+석판 혼합 |
| 방어 시설 | Maerith 측 소형 감시탑 (추정) |
| 주요 물자 | Maerith 고원 광물·모피 ↔ Thaloss 철제품 |

**명칭 유래**: 정상부에 만년 백설이 쌓인 안부(Saddle) 형태 — "흰 정수리" (추정·대표님 미확정).

## 4. 전설 층위 (Q-CORE 간접 단서)

*구조적 진실 직접 서술 금지. 인-월드 전설·구전만.*

- Ironcleft Pass 협곡 최심부에는 **"밤마다 붉은 빛이 올라온다"** 는 Thaloss 광부 전승이 있다. 교회는 "철광석의 자연 발광" 이라 설명하나, 일부 양심파 신관은 이를 *"태초의 응집 잔재"* 로 개인 필사본에 기록했다고 전해진다. (Q-CORE 간접 단서 — 해석 금지)
- Greygate Pass 남문 요새 초석에는 **"신이 직접 놓은 돌"** 이라는 명문이 있다고 교회 기록에 나온다. 현재 초석은 마모되어 명문 확인 불가. (대표님 미확정 — 모호 보존)
