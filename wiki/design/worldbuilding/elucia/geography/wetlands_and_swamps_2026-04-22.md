## 요약

Elucia 의 습지·늪은 주로 서남부 **Loravel Wetlands** (세렌 왕국)와 남서 **Lonwyn Lakelands** (알드릭 왕국) 를 중심으로 분포하며, 대하천 하구 삼각주와 범람원이 보조 습지를 형성한다. 총 습지 면적 약 70,000 km². 습지는 경작이 어려워 인간 왕국이 실질 지배하지 못하는 구역이 많고, 이로 인해 타종족 은신 지형으로 기능한다. 동시에 어업·소금·이탄(泥炭) 의 경제 자원이기도 하다.

## 2. 대형 습지 상세

### 2-1. Loravel Wetlands — 서남 대습지 (Loravel 권역)

Elucia 최대 습지. Auravel 강 하류와 서쪽 조석이 만나 광활한 삼각주·갈대 습지·소택지를 형성한다.

| 항목 | 내용 |
|------|------|
| 면적 | ~30,000 km² |
| 평균 수심 | 0.3–2.0m (계절 변동 큼) |
| 주요 식생 | 갈대·수련·버드나무·오리나무 |
| 주민 | Ceren 왕국 어부·소금 채취인·습지 수렵인 |
| 경제 | 어업·소금·이탄 채취·철새 수렵 |
| 타종족 | 중간 — 습지 섬들에 소규모 은신 |

**습지 내부 구조**:

| 구역 | 특성 | 인간 접근 |
|------|------|---------|
| 외곽 갈대밭 | 수심 얕음·접근 가능 | 일상 어업·채취 |
| 중간 소택지 | 수심 가변·발 빠짐 위험 | 경험 있는 사공만 |
| 내부 습지 섬들 | 고립 소도·울창 수목 | **사실상 미탐사** |

---

### 2-2. Lonwyn Lakelands — 남서 호수 지대 (Lonwyn 권역)

Lowen 강이 Lonwyn 분지로 유입되며 형성된 **호수군**. 주요 호수 5–8개와 연결 수로, 주변 습초원으로 구성된다.

| 항목 | 내용 |
|------|------|
| 면적 (호수+주변 습지) | ~25,000 km² |
| 주요 호수 | Lonwyn Lake (최대, ~8,000 km²), Aldren Mere, Mistvane Pool 등 (추정) |
| 수질 | 담수 · 맑음 (상류 오염 없음) |
| 경제 | 담수 어업·소금(호변 염전)·수운 내항 |
| 전략 | 내만 항구 기능. Aldric 왕국 방어 해자 |
| 전설 | "호수 한가운데서 달빛 아래 불빛이 보인다" — Aldric 어부 구전 |

---

### 2-3. Irondelta Marsh — Eloryn 삼각주 습지

Eloryn 강 하구 Mirevane Bay 와의 접점에 형성된 삼각주 습지.

| 특성 | 내용 |
|------|------|
| 면적 | ~8,000 km² |
| 성격 | 조석 삼각주 · 강-바다 전이 생태계 |
| 산업 | 해안 어업 + 강하구 어업 복합 |
| 항구 후보 | Irondelta 북쪽 안쪽 고지에 항구 도시 입지 가능성 |

## 4. 습지와 타종족 은신 (발언 8 반영)

발언 8: *"대륙의 가장자리의 밀림이나 숲"* 항목 중 습지도 발언 8 의 "대륙 가장자리" 와 "접근 불가 지형" 의 요건을 충족한다.

- **Loravel 내부 습지 섬들**: 접근이 사실상 불가능한 고립 소도 복수 존재. 교회 순찰이 미치지 않음 — 소규모 타종족 공동체 가능성 (추정)
- **Lonwyn 호수 소도(小島)**: 호수 내부 고립 섬. Aldric 왕국 왕실도 전체 파악 불가 — 은신 지형

## 5. 전설 층위 (Q-CORE 간접 단서)
