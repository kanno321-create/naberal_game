## 요약

Elucia 의 숲은 **확정 지명 2개** (Silvan, Orenwald) 를 핵심으로, 총 8개 주요 숲 지대가 대륙 서·북·동 가장자리와 내부 구릉에 분포한다. 발언 8 원문에 따라 숲은 타종족의 주요 은신 지형이며, Silvan 과 Orenwald 는 Elucia 내 타종족 25% 의 주요 거주 후보 지대다. 숲의 규모와 밀도는 서쪽(해양성 기후)이 동쪽(대륙성)보다 크다.

## 2. 확정 숲 상세

### 2-1. Silvan Forest — 서해안 숲 (확정 지명)

| 항목 | 내용 |
|------|------|
| 공식 지명 | **Silvan** (세트 B 확정) |
| 위치 | Ilaris 왕국 (Silvan 권역) |
| 추정 면적 | ~85,000 km² (한반도 면적과 유사) |
| 수종 | 참나무·느릅나무·ash·서양주목 위주 (온대 해양성 낙엽수림) |
| 특성 | 해양 기후로 안개가 잦음. 강수량 풍부. 내부 깊이 들어갈수록 인간 미탐사 구역 |
| 타종족 은신 | **높음** — 발언 8 "숲 한가운데" 직접 해당. Elucia 25% 타종족 중 상당수 거주 추정 |
| 인간 활용 | 목재 채취·수렵·수지(樹脂) 채취·해안 선박 목재 |
| 전설 | "가장 깊은 곳에서는 나무들이 서로 속삭인다" — Ilaris 목부(木夫) 구전 |

**숲 층위 구조**:

| 층위 | 깊이 | 인간 접근 | 타종족 |
|------|------|---------|--------|
| 外林 (가장자리) | 0–5 km | 일상 채취·사냥 | 거의 없음 |
| 中林 (중부) | 5–20 km | 왕국 허가 탐사대만 | 간헐적 흔적 |
| 深林 (심부) | 20 km+ | **사실상 미탐사** | 은신 공동체 추정 |

### 2-2. Orenwald — 동부 숲 (확정 지명)

| 항목 | 내용 |
|------|------|
| 공식 지명 | **Orenwald** (세트 B 확정) |
| 위치 | Oryn 왕국 (Orenwald 권역) |
| 추정 면적 | ~120,000 km² (가장 큰 숲) |
| 수종 | 소나무·가문비나무·낙엽송 위주 + 내부는 활엽수 혼합 (대륙성 혼합림) |
| 특성 | Veilorn Ridge 서사면에 기대 있어 동쪽 건조 기류를 막음. Soranth 강 상류·Orynwood Beck 수원 |
| 타종족 은신 | **매우 높음** — 규모 최대. Deepwald Stream 비밀 수계 내부 |
| 인간 활용 | 목재·수렵·약초 채취. 왕국 Oryn 의 주요 산업 |
| 전설 | "오렌왈드 속에서는 나침반이 제멋대로 돈다" — Oryn 상인 관행 기록 (출처 불명) |

**숲의 경계 불명확성**:
Orenwald 의 동쪽 경계는 Veilorn Ridge 서사면과 맞닿아 있어, 숲 깊은 곳에서 릉 기슭까지 이어진다. 이 구간은 인간 지도상 공백이며, 교회 기록에도 "어둠의 숲" 으로만 표기된다.

## 3. 부속 숲 상세

### 3-1. Norvend Shadow (노르벤드 그림자 숲)

- Norvend Range 남사면 아고산대 (고도 800–2,000m 구간)
- 주수종: 전나무·눈잣나무·자작나무
- 겨울에 눈이 3–4개월 쌓임. Thaloss 왕국의 목재·약초 생산지
- 드워프 구전에서 "산의 수염" 이라 불린다 (현 은신 중 드워프 — 대표님 확정 전 추정)

### 3-2. Havren Thicket (하브렌 덤불 숲)

- 북서 해안 Morncliff Spine 경사면. 관목 위주 저관목 숲
- 해풍에 굽어 자라는 특이한 수형. Moran 해안 어부들에게 땔감 공급
- 규모 작고 인구 접근 多 — 타종족 은신 부적합

### 3-3. Loravel Fen-wood (로라벨 늪지 숲)

- Loravel 습지 내부에 섬처럼 점재하는 습지 숲
- 버드나무·오리나무·늪 참나무. 발이 빠지는 늪이 자연 방어벽
- 타종족 은신 중간 — 접근이 어려우나 규모 소
- Ceren 왕국 인간들도 늪 깊이는 들어가지 않음

### 3-4. Aurion Copse (오리온 소림)

- 중앙 Aurion 평야에 산재하는 소규모 잡목림
- 경작지 사이 방풍림 역할. 왕국 경계 표지 기능도 함
- 타종족 은신 **불가** — 완전히 인간 통제권 내

### 3-5. Duskveil Wood (더스크베일 숲)

- Duskmoor 구릉 Novas 왕국 남동부
- 혼합림. 내부에 석회암 지형과 결합해 **동굴·협곡 복잡** → 타종족 은신 중간 가능성
- 남동 경계 방면이라 교회·왕실 감시가 약함

### 3-6. Maerith Highwood (마에리스 고지 숲)

- 북동 Auryn 고원 Maerith 왕국
- 아고산 낙엽 활엽수. 단풍이 극적 — Maerith 문화 정체성의 일부 (추정)
- Norvend 동쪽 끝봉 Stonecrown 과 인접해 고산 분위기

## 4. 숲과 타종족 분포 (발언 8·50 통합)

발언 8 원문: *"타종족은 주변 작은 섬들이나 대륙의 가장자리의 밀림이나 숲, 사막한가운데서 숨어서 생활한다."*
발언 50 원문: *"타종족비율이 서쪽 25%동쪽75%임"*

Elucia (서쪽) 25% 타종족 분포 추정:

| 숲 | 타종족 은신 가능성 | 추정 종족 |
|----|----------------|---------|
| Silvan 深林 | **매우 높음** | 엘프 (outline Ch.15 엘프 숲 연결) |
| Orenwald 深部 | **매우 높음** | 엘프·마족 파편 |
| Norvend Shadow | 높음 | 드워프 (산맥 동굴과 연계) |
| Loravel Fen-wood | 중간 | 수인족(未確定)·소규모 마족 |
| Duskveil Wood | 중간 | 오크 인접 가능성 (Ch.17 초원과 경계) |
| Maerith Highwood | 낮음 | 용족 인근 고지 경로 |

*위 종족 배정은 발언 원문 미확정 항목 포함. 대표님 확인 전 (추정) 표기.*

## 5. 전설 층위 (Q-CORE 간접 단서)

- Silvan 深林 최심부에 **"나무들이 기억을 가진다"** 는 엘프 구전이 인간 주점에서 소문으로 떠돈다. 교회는 이를 이교 미신으로 탄압하나 완전히 지우지 못하고 있다. (Q-CORE 간접 단서)
- Orenwald 의 가장 오래된 나무는 **"이 행성이 처음 숲을 입기 시작했을 때부터 서 있었다"** 는 것이 Oryn 왕국 수도 도서관 필사본에 단 한 줄 기록되어 있다. (초고대문명 미확정 보존)
