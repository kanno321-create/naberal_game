## 요약

**남부 국경 방어 협약** 은 Elucia 최남단에 위치한 소왕국 Novas 와 인접 중왕국 Sylren 간의 방어 동맹이다(추정). Novas 는 Azim Pass 북문을 통제하며 Karzor 와 직접 접경하는 최전선 왕국이므로, 독자 방어력만으로는 동쪽의 군사 압박을 버티기 어렵다. Sylren 의 대평원 곡물과 병력이 Novas 방어를 보조하는 구조이며, 대신 Sylren 은 Novas 를 통해 Azim Pass 교역 이익을 일부 배분받는다(추정).

## 1. 협약 기본 정보

| 항목 | 내용 |
|------|------|
| **결성 동기** | Novas 단독 Karzor 방어 불가 + Sylren 교역로 확보 |
| **Novas 기여** | Azim Pass 통행세 수입 일부 배분 · 동쪽 정보 제공 |
| **Sylren 기여** | 남부 평원 식량 공급 · 유사시 병력 파견 (추정) |
| **현재 상태** | 유지 중 · Azim Pass 긴장 고조로 지원 요청 빈발 (추정) |

## 2. 전략 지형 가치

```
[Sylren 남중앙 평원] → 남하 → [Soranth 권역] → 남하 →
[Duskmoor 구릉] (Novas 영토) → [Azim Pass 북문] →
[Azim Pass 지협] → [Karzor Sabin 자치구]
```

- Sylren 이 무너지면 Novas 후방 보급선 완전 차단
- Novas 가 무너지면 Karzor 군이 Sylren 직접 공격 가능
- 상호 의존 = 협약의 물리적 기반

## 3. 협약 주요 조항 (추정)

1. **유사시 병력 파견**: Novas 가 Karzor 공격 시 Sylren 은 병력 1/3 파견 의무 (추정)
2. **식량 우선 공급**: Sylren 은 Novas 주둔 수비대에 군량 할당 보장
3. **통행세 배분**: Azim Pass 통행세 수입의 일부를 Sylren 에 배분 (추정)
4. **Azim Pass 봉쇄 협의**: 봉쇄 여부는 양국 공동 결정 의무

---

## 4. 협약 한계와 긴장

- **성좌국 간섭**: 성좌국은 Azim Pass 통행세 일부를 요구 → 양국 공동 저항
- **Novas 독자 협상 유혹**: Karzor(Sabin) 와 단독 타협 시 Sylren 불필요해짐
- **협약 비대칭**: Sylren 이 이 협약에서 얻는 직접 이익은 교역로 접근뿐 → 장기 지속 의문
