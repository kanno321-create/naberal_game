## 요약

**Azim Pass** 는 동서 두 대륙을 잇는 유일한 육지 통로로, **Novas(서쪽 북문)** 와 **Karzor Sabin 자치구(동쪽 남문)** 가 각각 통제한다. 이 단일 통로의 외교적 지위는 동서 대륙 전체 관계의 온도계 역할을 한다. 긴장 시 통행 제한 → 교역 중단 → 양측 모두 경제 타격 → 협상 복귀의 사이클이 반복된다(추정). 성좌국은 Novas 를 통해 Azim Pass 를 간접 영향권에 두지만, Karzor 측과의 직접 외교 채널은 사실상 단절 상태다(추정).

## 2. Azim Pass 외교 5대 현안

| 현안 | 현황 |
|------|------|
| **이중 통행세** | 합산 18%+ → 교역 위축 · 개혁 교착 |
| **중간 무법 구역** | 공식 거버넌스 없음 → 도적·비공식 환전 |
| **타종족 통행** | 법적 조항 부재 → 묵인 상태 |

| **군사 긴장** | 양측 경비대 소규모 충돌 반복 |

## 3. 성좌국의 딜레마

- **원하는 것**: Azim Pass 를 통한 Karzor 와의 종교·경제 영향력 확대
- **현실**: Karzor 중앙 = 성좌국 직접 대화 거부 (추정) → Novas 를 통한 간접 채널만 가능
- **결과**: 성좌국은 Azim Pass 외교를 Novas 에 위임하면서도 Novas 자율성을 견제하는 모순
