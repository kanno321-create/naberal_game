## 요약

Elucia 대륙의 공용어는 **Elucian** 이다. 라틴·게르만·켈트 혼합 어감 (네이밍 세트 B 계승) 의 표준어로, 교황청 Solaris 가 표준화를 주도한다. 문자는 교회가 관리하며 평민 문해율은 낮다. 발언 31 에 따라 마법 주문(할배 개발) 은 Elucian 어휘 기반으로 파생됐다. 방언은 별도 파일 `dialects_regional_framework` 로 분리한다.

| **마법 주문어** | Elucian 고어 방언 (할배가 체계화 — 발언 31) |

## 2. 문자 체계

### 2-1. Elucian Script 구조 (작업 가설 · 대표님 미확정)

Elucian Script 는 표음 문자 체계로, 교황청 서기관이 교회 공문서에 사용한다.

| 특성 | 내용 |
|------|------|
| **자모 수** | 26~30자 (추정) |
| **방향** | 좌→우 가로쓰기 |
| **특수 문자** | 마법 주문 기호 6종 (할배 체계 · 일반 문자와 구별) |
| **교회 독점** | 문자 교육 = 교회 권한. 귀족 가정교사 일부 예외 |
| **비식자층 소통** | 구술 전통 + 문장(紋章) 상징 |

### 2-2. 네이밍 세트 B 와의 정합

기존 확정 지명이 Elucian 발음 원칙을 보여준다:

| 지명 | 어감 패턴 |
|------|---------|
| **Elucia** | 부드러운 라틴 어미 `-ia` |
| **Solaris** | 라틴 `sol` (태양) 어간 |
| **Vaelin** | 켈트풍 `-in` 종성 |
| **Thaloss** | 게르만풍 자음군 |
| **Loravel** | 부드러운 다음절 |

신규 작명 원칙 (네이밍 세트 B 계승):
- WoW 풍 (`Auréllion`) 회피
- 발음 가능 + 한국어 표기 자연스러움
- 라틴·게르만·켈트 3요소 혼합

| 문법 항목 | 특성 |
|---------|------|
| **어순** | SOV 기본 (라틴어 계열) |
| **경어 체계** | 2단계: 경어 (`-ael` 어미) / 평어 (기본형) |
| **복수형** | `-s` 또는 `-en` (게르만 계열) |
| **시제** | 과거·현재·미래 3형 (완료형 없음 · 단순) |
| **귀족 전용 어휘** | 300~400개 어휘 평민 사용 불가 (사회적 배제 도구) |
