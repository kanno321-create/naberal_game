## 요약

Elucia 의 계절 축제는 농업 달력과 교회 달력이 결합된 구조다. 발언 46 "농업 위주의 삶" 과 "어느 마을이나 교회가 있다" 로 교회가 계절 축제를 관장하는 구조가 확정된다. 4대 공통 계절 축제를 정립하며, 왕국별 고유 축제명·행사 내용은 Wave 4 Kingdom-Detailer 담당이다.

## 2. 각 축제 상세

### 2-1. 씨앗 축제 (Spring Seeding)

봄 씨뿌리기 전 교회의 농사 축복을 받는 의식에서 유래.

| 항목 | 내용 |
|------|------|
| **기간** | 3~5일 |
| **주관** | 마을 신관 |
| **의식** | 씨앗 행렬 → 교회 마당 집결 → 신관 축복 → 봄 식사 |
| **공통 음식** | 겨우내 저장한 보존 식품·첫 봄 채소 (지역 variation) |
| **특이 사항** | 이 시기 성인식 집행 빈번 (봄 = 새 시작) |
| **타종족 금지 요소** | 축제 구역 내 타종족 금지 (마을 외곽으로 몰아냄) |

### 2-2. 하지 행군 (Midsummer March)

징병제(발언 46) 와 결합된 여름 축제. 성인 남성의 군역 신고 의식을 포함.

| 항목 | 내용 |
|------|------|
| **기간** | 1~3일 |
| **주관** | 영주·왕국 관리 (교회 보조) |
| **의식** | 무장 행진 → 징병 신고 → 무예 경연 → 연회 |
| **공통 음식** | 야외 구이·축산물 (서쪽 축산업 특성 · 발언 48) |
| **특이 사항** | 기사 서임이 이 시기 집중 |
| **양심파 연결** | 일부 신관이 이 축제에서 징병 면제 탄원 — 이단 의심 시작점 |

### 2-3. 수확 대제 (Harvest Grand Feast)

Elucia 연간 최대 축제. 교회 십일조 납부와 연동.

| 항목 | 내용 |
|------|------|
| **기간** | 3~7일 |
| **주관** | 교회 (십일조 수령) + 영주 (영지 잔치) |
| **의식** | 수확물 신관 축성 → 십일조 납부 → 영주 대연회 → 서민 광장 잔치 |
| **공통 음식** | 가을 작물·발효주·구운 고기 (지역 variation) |
| **계급 격차** | 귀족 연회(성내) vs 서민 광장 잔치 — 같은 날 다른 공간 |
| **이데올로기** | "신의 은혜로 풍작" 교리 강화 기회 (할배 생활 마법이 실제 기여하지만 교회가 신의 공으로 포장) |

### 2-4. 동지 등불제 (Solstice Lantern)

가장 긴 밤에 망자를 추모하고 새해를 기원하는 축제.

| 항목 | 내용 |
|------|------|
| **기간** | 1~3일 |
| **주관** | 교회 (추모 의식) |
| **의식** | 등불 행진 → 망자 이름 낭독 → 교회 추모 예배 → 가족 식사 |
| **공통 음식** | 따뜻한 스튜·뿌리채소·발효음료 (겨울 보관 식재료) |
| **변이 특성** | 고산 왕국 (Thaloss): 모닥불 규모 최대 · 해안 왕국: 바다에 등불 띄우기 |
| **Q-CORE 단서** | 일부 마을에서 "이름 없는 할배가 등불을 나눠줬다" 구전 (folklore_nameless_scholar 연결) |
