## 요약

Elucia (서쪽) 와 Karzor (동쪽) 는 **같은 신** 을 섬기나 교리 해석이 다르다 (발언 7). 이 공통 신앙은 교회를 통해 대륙 전체에 침투해 있으며 (발언 46 "어느 마을이나 교회가 있다"), 신앙 경제학 구조 (신의 권능 = 신앙 총량) 위에서 작동한다. 세계관 철학 "가. 불완전성" 에 따라 이 신도 완전하지 않으며, 현재는 나태와 권능 중독으로 타락한 상태다. Q-CORE 구조 (신의 실체 등) 는 교리 층위에서 **모호하게** 만 등장한다.

## 1. 신앙 기본 구조

### 1-1. "같은 신" — 대륙 공통 숭배 대상

발언 7 원문에 따라 좌우 대륙은 **동일한 신격** 을 섬긴다. 그러나 신의 이름·외형·성격은 대표님 미확정이므로 교리 층위에서만 기술한다.

| 항목 | 내용 |
|------|------|
| **공식 신격 칭호** | "첫 번째 신" (인-월드 교리 용어) (대표님 미확정 · 작업 가설) |
| **서쪽 (Elucia) 호칭** | "영원하신 분" · "빛의 근원" (작업 가설) |
| **동쪽 (Karzor) 호칭** | "성좌의 주재" · "심판의 빛" (작업 가설) |
| **공통 핵심 교리** | "인류를 위한 행성 · 타종족은 불필요" (발언 7) |
| **신의 현재 상태** | 교리상 = 전능·영원. 실제 불명 (대표님 미확정 · 교리 층위만) |

### 1-2. 해석 차이의 결과 — 적대 속 공동 목표

발언 7 "서로 적대적이긴하나 하나의 목표" 를 그대로 반영한다.

```
동일 신앙
    ├── Elucia 해석: 신 = 질서·법·교황청의 수호자
    │             → 박해 = 법제도·교회법으로 집행 (질서 있는 악)
    └── Karzor 해석: 신 = 힘·정복·성좌의 심판자
                  → 박해 = 노예시장·투기장으로 집행 (거친 악)
        ↓ 공통
    타종족 몰살 이데올로기 (발언 7 확정)
```

## 2. 교회 조직 체계 (Elucia 기준)

발언 46 "어느 마을이나 교회가 있다" 로 교회 편재성 확정.

### 2-1. 수직 교계 (위에서 아래)

| 층위 | 명칭 | 역할 |
|------|------|------|
| 1 | **교황** (성좌국 Solaris) | 최고 권위 · 신의 대리인 선언 · 권능 중독자 |
| 2 | **대주교** (왕국 수도 대성당) | 왕국 단위 교권 관장 |
| 3 | **주교** (지방 도시) | 도시 교회 감독·이단 심문 담당 |
| 4 | **신관·사제** (마을 교회) | 성인식·목걸이 등록·일상 의식 |
| 5 | **조신관·서기** (마을 교회 보조) | 기록·교육·보조 |

> 양심파 신관은 **층위 4·5** 에 주로 분포. 이단 심문과 충돌 (outline Ch.09 정합).

### 2-2. 교회 기능 목록

| 기능 | 설명 |
|------|------|
| **성인식 집전** | 목걸이 등록·의지결 장착 의식 (발언 44·45 정합) |
| **신앙 교육** | 교회학교 운영 (literacy_and_education 참조) |
| **출생·혼인·장례 기록** | 인구 관리 (교황청 중앙 등록) |
| **신성마법 분배** | 치유·축복·정화 서비스 (간접 기술 — 원래 마족 기원이나 "신의 은총"으로 포장) |
| **이단 심문** | 양심파·타종족 연대자 탄압 |
| **십일조 징수** | 왕국 세수의 일부 교황청 이전 |

### 2-3. 마을 교회 — 일상 접점

발언 46 의미: 서민이 신앙을 접하는 유일한 공식 채널.

- 마을 교회 규모: 목조 예배당 30~80석 (추정 · 대표님 미확정)
- 기능: 일요 예배·농사 축복·병자 치유 기도 (신성마법 소규모 시전)
- 신관의 이중성: 타락한 교리 전달자이면서 때로는 양심적 개인 (Ch.09 동료 모델)

## 3. 신앙 경제학 (게임 설정 §3 요약 반영)

| 구조 | 작동 방식 |
|------|---------|
| **신앙 → 신의 권능** | 신앙 총량이 클수록 신의 힘이 유지됨 |
| **인간 편애 이유** | 개체수 多 + 높은 번식력 + 신 의존도 高 |
| **교회 권력층 공모** | "신의 이름" = 통치 정당화 도구 |
| **일반 신도** | 세뇌·피해자 (설계 헌법 제3조 준수) |

## 4. 종족별 신앙 관계 (발언 정합)

| 종족 | 신앙 여부 | 관계 |
|------|---------|------|
| **인간** | 유 (교회) | 신앙 주체 · 편애 대상 |
| **엘프** | 자연 섭리 (범신론) | 공식 신앙 거부 · 박해 근거 일부 |
| **드워프** | 무신론 | 공식 신앙 거부 · 박해 근거 |
| **마족** | 원시종교 (마왕 숭배) | 이단 최고 등급 · 절멸 사냥 대상 |
