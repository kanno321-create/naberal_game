## 요약

Elucia 건축은 발언 3 "서구중세문명" 을 판타지 번역한 것으로, 고딕·로마네스크 교회 건축과 목조+석조 혼합 세속 건축이 기반이다. 발언 46 "어느 마을이나 교회가 있다" 로 교회 건물이 마을 최고 건축물임이 확정된다. 기후대별 건축 재료 variation 축을 정립하며, 왕국별 고유 건축 세부는 Wave 4 Kingdom-Detailer 담당이다.

## 1. 건축 유형 분류

| 유형 | 기능 | 주요 건물 |
|------|------|---------|
| **종교 건축** | 교회·신앙 의식 | 대성당·마을 예배당·수도원 |
| **군사 건축** | 방어·지배 | 성·요새·성벽 |
| **귀족 세속 건축** | 거주·행정 | 영주 저택·시청·법원 |
| **서민 건축** | 주거·상업 | 목조 가옥·시장 건물·여관 |
| **농촌 건축** | 농업·축산 | 농가·축사·창고·물레방아 |

## 2. 건축 양식 3분류

### 2-1. 고딕 양식 (Gothic-Elucian) — 교회 중심

| 특성 | 내용 |
|------|------|
| **원형** | 서구 중세 고딕 판타지 번역 |
| **용도** | 대성당·교황청 건물·왕국 대교회 |
| **특징** | 첨탑·높은 천장·스테인드글라스(마력 수정 빛 연출) · 아치 |
| **재료** | 백색·회색 석재 (고가) |
| **상징** | "신의 위엄" 과시 · 교황 권위 외화 |
| **마법 건축 요소** | 마력 수정 장식 (이데올로기 도구) · 신성마법 보조 구조체 |

> 집필 활용: 대성당 내부 = 압도적 규모 + 창문으로 쏟아지는 빛 + 신관 행렬. 주인공이 처음 보는 대도시 교회 장면.

### 2-2. 로마네스크 양식 (Romanesque-Elucian) — 성·귀족 건축

| 특성 | 내용 |
|------|------|
| **원형** | 서구 중세 로마네스크 판타지 번역 |
| **용도** | 성채·요새·귀족 저택·왕궁 |
| **특징** | 두꺼운 벽·둥근 아치·작은 창문·육중한 탑 |
| **재료** | 회색·갈색 석재·벽돌 |
| **상징** | 군사력·봉건 권위 |

### 2-3. 목조+석조 혼합 양식 (Timber-Stone) — 서민·농촌

| 특성 | 내용 |
|------|------|
| **원형** | 중세 유럽 반목조 건물 판타지 번역 |
| **용도** | 마을 가옥·여관·시장·소규모 마을 교회 |
| **특징** | 목재 기둥·짚 지붕·석재 기초·백회 마감 |
| **재료** | 목재(숲 근접) 또는 석재(광업지) 기후 따라 variation |
| **상징** | 서민 생활의 평범함 · 타종족은 이런 집도 못 가짐 |

## 3. 기후대별 Variation 축

| 기후대 | 주재료 | 건축 특성 |
|--------|--------|---------|
| **A 해양성온대** | 목재·벽돌 | 경사 지붕(빗물 대비)·습기 방지 높은 기초 |
| **B 대륙성온대** | 석재·목재 균형 | 표준 중세 마을 경관 · 사계절 대응 |
| **C 고산·아고산** | 석재 집중 | 두꺼운 벽·작은 창문 방한·납 지붕·지하 저장소 |
| **D 한대전이** | 석재+목재 극한 방한 | 눈 하중 대응 경사 급 지붕·이중 벽 |
| **E 내륙반건조** | 건조 목재·석재 | 평평한 지붕(강수 적음)·넓은 창문(채광) |
| **F 습지삼각주** | 목재·말뚝 기초 | 습지 위 고상 건물·수로 기반 도시 구조 |

## 4. 마을 경관 공통 구조

발언 46 "어느 마을이나 교회가 있다" 기반.

```
마을 중심 광장
    ├── 교회 (가장 큰 건물 · 석조 또는 목조)
    ├── 영주 또는 촌장 저택 (2번째 크기)
    ├── 시장·상점 거리 (광장 주변)
    ├── 여관 (상인·여행자)
    └── 가옥 (평민 · 목조+석조)

마을 외곽
    ├── 농경지·축사
    ├── 물레방아 (강 인접 시)
    └── 공동묘지 (교회 뒤편)
```
