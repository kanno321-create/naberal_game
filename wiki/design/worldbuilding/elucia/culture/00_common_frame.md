## 요약

Elucia (서쪽 대륙) 의 문화는 발언 3 "서구중세문명" 을 기반으로 한다. 왕국·기사·교회·봉건제가 사회 골격을 이루며, 발언 7 의 종교 구조(같은 신 · 해석 차이) 가 문화 전반의 이데올로기 기반을 형성한다. Wave 2 Culturalist 는 11 왕국 + 교황청 제국에 공통으로 적용되는 **통일 프레임** 과 **variation 축** 을 정립한다. 왕국별 고유 의상·축제·음식·건축 세부는 Wave 4 Kingdom-Detailer 가 확장한다.

## Variation 축 요약 (Wave 4 참조용)

Wave 4 Kingdom-Detailer 는 아래 variation 축을 기준으로 각 왕국의 고유 문화를 작성한다.

| 축 | 공통 프레임 | 변이 방향 |
|----|-----------|---------|
| **기후** | 6개 기후대 (climate_zones 참조) | 해양성 → 목초·안개 문화 / 고산 → 광업·방한 문화 |
| **산업** | 농업+축산업 (Elucia 공통) | 해안 왕국 → 어업 보조 / 내륙 → 소·말 중심 |
| **교회 영향도** | 모든 마을 교회 존재 | 성좌국 직할 → 최강 / 변경 왕국 → 상대적 약 |
| **봉건 밀도** | 왕→공작→백작→남작→기사→평민 | 제국권 → 층위 촘촘 / 변경 → 단순화 |
| **타종족 접촉** | 숲·습지·가장자리에 은신 | 가장자리 왕국 → 접촉 빈번 / 중앙 → 희소 |
| **방언** | Elucian 공용어 기반 | 해안 → 항해 어휘 / 산악 → 광업 어휘 |
