## 요약

Elucia 의 성인식은 교회 주관 하에 의지결 목걸이 장착·신앙 서약·징병 등록 3기능이 결합된 의식이다. 발언 45 "성인이라면 목걸이" 가 핵심이며, 목걸이 = 신분 증명 + 의지결 슬롯(마법 기관) 의 이중 기능이 있다. Q-CORE 상 의지결의 실제 기원은 직접 서술 금지. 왕국별 성인 나이·구체 의식 절차는 Wave 4 Kingdom-Detailer 담당.

## 1. 성인식 개요

| 항목 | 내용 |
|------|------|
| **주관** | 마을 신관 (대도시 = 주교) |
| **대상** | 인간 남녀 (성인 나이 = [대표님 미확정 · 14~16세 추정]) |
| **타종족** | 성인식 대상 아님 (노예 신분 고정) |
| **시기** | 봄 씨앗 축제 기간 집중 (new start 상징) · 개인 의식은 연중 가능 |
| **교회 수입** | 성인식 봉헌금 (계층별 금액 상이) |

## 2. 성인식 3단계 의식

### 2-1. Step 1 — 신앙 서약 (Faith Oath)

| 항목 | 내용 |
|------|------|
| **장소** | 마을 교회 제단 앞 |
| **의식** | 신관 앞에서 "첫 번째 신"에 대한 신앙 서약 낭독 |
| **증인** | 부모·마을 대표 (성인 2인 이상) |
| **교리적 의미** | 신의 보호 공동체 공식 편입 · 이단 의심 대상에서 제외 |
| **숨겨진 기능** | 교회 신자 명부 등록 → 이단 추적·징병 명부 활용 |

### 2-2. Step 2 — 목걸이 교부 및 의지결 장착

| 항목 | 내용 |
|------|------|
| **목걸이** | 신관이 계층 소재 목걸이 교부 (평민 = 목제·철, 귀족 = 은·금) |
| **의지결 장착** | 목걸이 슬롯에 의지결 기관 연결 — 발언 45 "슬롯이 딱 하나" |
| **교리 설명** | "신의 은총이 몸속에 깃드는 의식" — 실제 기원 교리층 불투명 |
| **마법 효과** | 의지결 목걸이 장착 후 생활 마법 1종 사용 가능 (기원 불명의 고대 기술 기반 · 교리상 "신의 은총") |
| **계층 차이** | 귀족 목걸이 = 고급 의지결 슬롯 (추정) → 더 강력한 마법 장착 가능 |

> ⚠️ 의지결의 실제 기원은 교리 층위에서 "신의 은총" 으로만 제시. 집필 시 Ch.12+ 에서 양심파 신관이 의문을 갖는 복선으로 활용.

### 2-3. Step 3 — 공동체 편입 선언

| 항목 | 내용 |
|------|------|
| **선언자** | 신관 |
| **내용** | "[이름]이 신의 백성이자 [왕국명] 왕국의 성인으로 등록됨" 공식 선언 |
| **징병 연계** | 남성: 성인식 당일 또는 직후 징병 명부 등록 |
| **사회적 의미** | 성인 = 계약 체결 가능 · 결혼 가능 · 재산 상속 가능 |
| **이단 보호** | 성인식 미이수 = 이단 의심 대상 → 사실상 강제 참여 |

## 3. 계층별 성인식 Variation

| 계층 | 규모 | 목걸이 | 봉헌금 | 특이 사항 |
|------|------|--------|--------|---------|
| **왕족** | 왕국 전체 행사 · 왕도 대성당 | 금·보석 | 국가 차원 기부 | 교황·대주교 주관 · 전국 선포 |
| **귀족** | 영지 규모 행사 | 은·준보석 | 거액 | 기사 서임 동시 진행 (남성) |
| **성직 지망** | 교회 특별 의식 | 교회 전용 목걸이 | 교회 규정 | 신관 입문 시험 직전 |
| **시민·상인** | 마을 규모 | 철·구리 | 중간 | 길드 등록과 연계 가능 |
| **농노·평민** | 마을 단위 집단 성인식 | 목제·뼈 | 소액 또는 현물 | 집단 처리 → 개인 의식 희박 |

## 4. 성인식 거부·실패 시 결과

| 상황 | 결과 |
|------|------|
| **성인식 미이수** | 이단 의심 대상 · 사회 계약 체결 불가 · 결혼 불인정 |
| **신앙 서약 거부** | 이단 심문 대상 즉시 회부 |
| **목걸이 분실·파손** | 교회에 신고 의무 · 재교부 수수료 |
| **타종족 위장 시도** | 발각 시 처형 또는 노예 강등 (극형) |
