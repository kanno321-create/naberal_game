## 요약

Elucia 의 신분 계급은 왕족·귀족·성직·시민·농노·노예의 6계층 구조에 타종족(비인간)이 추가된 7계층이다. 발언 45 성인 목걸이가 신분 등록·증명의 실물 시스템으로 기능한다. 발언 9 타종족 = 노예 취급 구조는 사회 최하층에 비인간 계층을 고정시킨다. 왕국별 세부 특권·법령·신분 이동 조건은 Wave 4 Kingdom-Detailer 담당.

## 1. 7계층 신분 구조

### 1-1. 전체 계층표

| 계층 | 세부 신분 | 인구 비중 (추정) | 목걸이 소재 | 신분 이동 가능성 |
|------|---------|--------------|-----------|--------------|
| **0 왕족** | 왕·왕족 직계 | 극소 (<0.1%) | 금·보석 | 출생 고정 |
| **1 상위 귀족** | 공작·백작 | 극소 (<0.5%) | 금 | 출생 고정 (왕령 예외) |
| **2 하위 귀족·기사** | 남작·기사 | 소 (1~3%) | 은 | 공훈·왕명으로 기사 승격 가능 |
| **3 성직** | 교황·대주교~조신관 | 소 (1~2%) | 교회 전용 | 교회 입문으로 이동 가능 |
| **4 시민·상인** | 자유 시민·상인 길드 | 중 (10~20%) | 철·구리 | 재력 축적 → 상위 이동 제한적 |
| **5 농노·평민** | 농노·소작농·일용노동자 | 대 (60~70%) | 목제·뼈 | 사실상 이동 불가 (이론상 해방 가능) |
| **6 봉공 계약인·타종족** | 엘프·오크·기타 종족 (서쪽 공식 용어 = 봉공 계약인 / 동쪽 = 노예) | 변동 (2~5%) | 노예 표식 | 원칙적 이동 불가 (해방 매우 드묾) |

> ⚠️ 인구 비중은 작업 가설 · 대표님 미확정.

## 2. 각 계층 상세

### 2-1. 왕족 (Royalty)

| 항목 | 내용 |
|------|------|
| **구성** | 왕·왕비·직계 왕자·공주 |
| **권한** | 영토 최종 통치권·선전포고권·귀족 서임권 |
| **의무** | 교회 십일조 납부 (왕국 단위) · 교황청 형식적 충성 |
| **목걸이** | 왕가 문장 금제 목걸이 (의지결 고급 슬롯) |
| **약점** | 교황청 파문 = 왕권 위기 · 귀족 연합 반란 위험 |

### 2-2. 상위 귀족 (High Nobility)

| 항목 | 내용 |
|------|------|
| **구성** | 공작·후작·백작 (왕이 수여한 봉토 보유) |
| **권한** | 영지 내 재판권·징세권·사병 보유 |
| **의무** | 왕령 군역·수확 대제 전후 조공 |
| **목걸이** | 금·귀석 목걸이 (가문 문장 부착) |
| **교회 관계** | 왕국 내 주요 교회 후원 (후원자 = 영향력) |

### 2-3. 하위 귀족·기사 (Low Nobility / Knight)

| 항목 | 내용 |
|------|------|
| **구성** | 남작·기사·준기사 (봉토 소규모 또는 없음) |
| **권한** | 마을 단위 영지 관리 · 기사 = 전투 특권 |
| **신분 이동** | 평민 → 기사: 공훈·왕명. 매우 드묾 |
| **목걸이** | 은 목걸이 (가문 색) |

> 집필 활용: 주인공 또는 동료 중 기사 작위를 받는 서사 — 이 계층으로 진입.

### 2-4. 성직 계층 (Clergy)

| 항목 | 내용 |
|------|------|
| **구성** | 교황(1인) → 대주교 → 주교 → 신관 → 조신관 |
| **권한** | 신성마법 공식 사용권 · 교육 독점 · 출생·혼인·사망 기록 |
| **신분 이동** | 평민 출신 신관 입문 가능 → 사실상 귀족 준하는 특권 획득 |
| **목걸이** | 교황청 규격 목걸이 (신관 등급별 문양 상이) |
| **내부 분열** | 타락파 (다수·교황청 추종) vs 양심파 (소수·은신) |

> ★ 양심파 신관 = 성직 계층 내부 갈등 서사의 핵심.

### 2-5. 시민·상인 (Citizen / Merchant)

| 항목 | 내용 |
|------|------|
| **구성** | 자유 시민·수공업자·상인·길드원 |
| **권한** | 직업 선택 자유 · 재산 소유 · 도시 내 투표권 (왕국별 차이) |
| **목걸이** | 철·구리 목걸이 |
| **약점** | 고가 사치품·향신료 접근 불가 (귀족 전유) · 길드 비가입 시 도시 영업 제한 |

### 2-6. 농노·평민 (Serf / Commoner)

| 항목 | 내용 |
|------|------|
| **구성** | 소작농·농노·일용 노동자 |
| **권한** | 최소 생존권 (왕국·영주 보호 의무) |
| **의무** | 지조·군역·교회 십일조·강제 노역 |
| **이동 제한** | 영주 허가 없이 다른 영지 이동 불가 (농노) |
| **목걸이** | 목제·뼈 목걸이 (저가) |
| **일상** | 발언 46 "농업 위주의 삶" 기반 — 농기계 없음 · 손 농업 · 교회 출석 의무 |

### 2-7. 봉공 계약인·타종족 (서쪽 공식 = 봉공 계약인 / 동쪽 = 노예 / Non-Human)

| 항목 | 내용 |
|------|------|
| **구성** | 엘프·오크·기타 타종족 노예 |
| **권한** | 법적 권한 없음 · 소유주 재산으로 분류 |
| **목걸이** | 노예 소유 표식 (성인 목걸이와 구별 · 소유주 문장) |
| **거래** | 노예시장 공개 경매 · 엘프 고가·오크 노동용 저가 (발언 9) |
| **종교 관계** | 교회 교리상 "짐승에 가까운 존재" → 신앙 접근 금지 (공식) |
| **양심파 입장** | 타종족 은신 지원 · 해방 운동 소규모 존재 (추정) |

## 4. 목걸이 신분 시스템 (발언 45 기반)

발언 45 "누구나 목걸이를 차고다니고 (성인이라면) 그목걸이로 신분을 확인가능하다(등록제)".

```
목걸이 기능:
    ├── 신분 증명 (소재·문양으로 계층 즉시 식별)
    ├── 성인 등록 (교회 성인식 후 교부)
    ├── 의지결 슬롯 1개 (마법 장착 기능 · 상위 계층 = 고급 슬롯)
    └── 노예 표식 (성인 목걸이와 구별 — 타종족 소유 표식)

목걸이 소재 위계:
    금 (보석) → 은 → 철·구리 → 목제·뼈 → 노예 표식
    [왕족·귀족]   [기사]  [시민]   [농노·평민]  [타종족]
```

> 집필 활용: 목걸이를 확인하는 것만으로 신분 파악 가능 → 신분 위장·노예 해방 장면의 긴장 도구.
