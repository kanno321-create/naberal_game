## 요약

교회 주관 종교 축제는 계절 축제와 달리 교리·권능·신앙 강화에 초점이 맞춰져 있다. 주요 행사로 성인기념일 예배, 성지 순례, 신성마법 공개 시전, 이단 척결 선포가 있다. Q-CORE 상 신성마법의 실제 기원(고대 마족)은 교리 층위에서 "신의 은총" 으로 포장되며 직접 서술하지 않는다.

## 2. 각 종교 축제 상세

### 2-1. 첫 번째 신 기념일

> ⚠️ "첫 번째 신" = 교리 내 호칭. 실체(Q-CORE) 직접 서술 금지.

| 항목 | 내용 |
|------|------|
| **시기** | (대표님 미확정 · 겨울 또는 봄 추정) |
| **규모** | 성좌국 Solaris 성대한 행사 + 전국 동시 마을 예배 |
| **핵심 의식** | 교황 또는 대주교의 신성마법 공개 시전 — 대규모 치유·광휘 연출 |
| **이데올로기** | "신의 권능이 교회를 통해 내린다" 의 시각적 증명 |
| **감춰진 진실** | 시전되는 신성마법의 원본 = (교리 기술 불가 · 파편 단서만) |
| **Variation** | 대도시 = 대규모 시연 / 마을 = 소규모 신관 시전 |

### 2-2. 정화 행진

종교적 이단 척결 선포를 목적으로 하는 군사·종교 혼합 행진.

| 항목 | 내용 |
|------|------|
| **빈도** | 연 2회 (봄·가을 추정) |
| **규모** | 도시 중심 = 주교·이단 심문관 주도 / 마을 = 신관·성인 남성 참여 |
| **의식** | 행진 → 교회 집결 → 이단 적발 선포 → 공개 처벌 (대표님 미확정) |
| **타종족 관련** | 타종족 은신지 수색 + 발각 시 체포 의식화 |
| **서사 연결** | outline Ch.09 양심파 신관이 이 행진을 거부하며 이단 지목 위기 발생 |

### 2-3. 성지 순례월

봄에 Solaris 또는 왕국 내 지역 성지를 방문하는 자발적 신앙 행위.

| 항목 | 내용 |
|------|------|
| **기간** | 봄 1개월 (추정 · 대표님 미확정) |
| **주요 목적지** | Solaris 대성당 · 왕국 내 성인 순교지 · 기적 발생 장소 |
| **참여 계층** | 귀족(긴 행렬)~부유 시민 / 가난한 시민은 왕국 내 소성지 |
| **교회 수입** | 순례 숙박·기부금 = 교회 수입원 |
| **양심파 활동** | 순례 경로 상 양심파 신관이 몰래 정보 네트워크 운영 (추정) |
| **Variation** | 가까운 성지(마을 단위) vs 먼 성지(성좌국) = 재력 차 표현 |

### 2-4. 헌납 예배

매월 정기 신앙 갱신 의식. 가장 일상적 종교 행사.

| 항목 | 내용 |
|------|------|
| **빈도** | 매월 1회 (통상 초하루) |
| **규모** | 마을 단위 · 전원 참석 의무 (사실상) |
| **의식** | 기도·찬가·신관 설교·헌금 |
| **목걸이 갱신** | 매년 1회 목걸이 신앙 등록 갱신 (헌납 예배 중 병행 · 추정) |
| **개근 압력** | 결석 = 이단 의심 대상. 왕국법이 아닌 사회적 압력 |

## 3. 신성마법 공개 시전의 문화적 기능

신성마법은 종교 축제의 핵심 스펙터클이다.

| 기능 | 설명 |
|------|------|
| **신앙 강화** | 눈에 보이는 기적 = 교리 신뢰도 강화 |
| **의료 대체** | 공개 치유 시전 = 가난한 서민 의료 수요 흡수 |
| **권능 과시** | 교황·대주교의 신성마법 규모 = 권위 서열 표현 |
| **교리 포장** | "신의 은총" 으로 제시 · 실제 기원은 교리층 불투명 (직접 서술 금지) |

> 집필 함의: Ch.01~11 신성마법은 "신의 권능" 으로만 서술. Ch.12~18 양심파 신관의 의문 첫 등장.
