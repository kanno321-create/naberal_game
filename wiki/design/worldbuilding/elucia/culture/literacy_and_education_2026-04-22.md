## 요약

Elucia 의 교육은 교회가 독점한다. 문자(Elucian Script)를 교회가 통제하므로 교회 비인가 교육은 사실상 불가능하다. 계층별 교육 기회의 극심한 격차가 존재하며, 농민 문맹률은 높고 귀족·신관만 고등 교육을 받는다. 문해율과 교육은 교회 권력 유지의 핵심 도구다. 왕국별 교육 기관·교육 과목 상세는 Wave 4 Kingdom-Detailer 담당.

## 1. 전체 문해율 추정

| 계층 | 문해율 (추정) | 비고 |
|------|------------|------|
| **왕족·귀족** | 거의 100% | 개인 교사·교회 고등 교육 |
| **성직자** | 100% | 성직 = 문해 필수 |
| **상인·시민** | 30~60% (추정) | 상업 필요성으로 자력 학습 |
| **기사** | 40~70% (추정) | 군사 문서·왕명 독해 필요 |
| **농노·평민** | 5~15% (추정) | 사실상 문맹 다수 |
| **노예 (타종족)** | 극소 | 법적 교육 금지 (추정) |

> ⚠️ 모든 수치는 작업 가설 · 대표님 미확정.

## 2. 교회 교육 기관 3단계

### 2-1. 마을 교회 기초 교육 (Primary — 종교 교육)

| 항목 | 내용 |
|------|------|
| **대상** | 마을 아동 전체 (종교 교리 교육) |
| **장소** | 마을 교회 (주일 예배 후) |
| **내용** | 교리 암송·찬가·기본 도덕 규범 |
| **문자 교육** | 없음 (구술 교육 중심) |
| **목적** | 신앙 강화·교회 복종 내면화 |

### 2-2. 도시 교회 학교 (Secondary — 신분 제한)

| 항목 | 내용 |
|------|------|
| **대상** | 귀족 자녀·부유 상인 자녀·신관 지망생 |
| **장소** | 도시 교회 부속 학교 |
| **내용** | Elucian 문자·기초 수학·교회 역사·교리 심화 |
| **문자 교육** | 있음 — Elucian Script 기초 |
| **비용** | 고액 수업료 (농민 접근 불가) |

### 2-3. 교황청 고등 교육 기관 (Tertiary — 극소수)

| 항목 | 내용 |
|------|------|
| **대상** | 신관 후보·귀족 자제·마법 연구 지망생 |
| **장소** | Solaris 교황청 부설 대학·신학교 |
| **내용** | 신학·신성마법 이론·역사·철학·고대어 |
| **문자 교육** | Elucian 고어 (마법 주문어) 포함 |
| **졸업 후** | 신관·대주교 후보·왕국 자문관 |

## 3. 비교회 교육 (제한적 존재)

| 유형 | 실태 | 위험성 |
|------|------|--------|
| **귀족 개인 교사** | 교회 인가 교사에 한해 허용 (추정) | 교회 감시 하 |
| **상인 길드 교육** | 계약서·장부 계산 실무 교육 (비공인) | 교회 묵인 (경제 필요성) |
| **민간 서적** | 교회 인가 서적 외 금서 처리 | 소지 = 이단 의심 |
| **양심파 교육** | 비밀 교육 (타종족·해방민 대상 · 추정) | 발각 = 이단 처형 |

## 4. 지식 통제 구조

```
교회 지식 통제 구조:
    ├── 문자(Elucian Script) 독점
    │     └── 비식자 = 교회 문서 의존 → 계약·법령 이해 불가
    ├── 서적 인가 제도
    │     ├── 허가 서적: 교회 교리·역사·기도서
    │     └── 금서: 대안 역사·타종족 서술·이단 사상
    ├── 교사 인가 제도
    │     └── 비인가 교사 = 이단 심문 대상
    └── 고대어 독점
          └── 마법 주문어(Elucian 고어) = 신관만 접근 가능
```

## 5. Elucia 서민의 지식 획득 방식

문맹 서민이 지식을 얻는 실제 방법:

| 방식 | 내용 |
|------|------|
| **구전·이야기꾼** | 여관·시장 이야기꾼 → 구전으로 역사·전설 전달 |
| **신관 설교** | 주일 예배 설교 = 서민의 주요 정보 수신 채널 (교회 필터링) |
| **도제 교육** | 장인·농부 수공업 기술 구술 전수 (문자 불필요) |
| **순례자 네트워크** | 성지 순례 중 타 지역 정보 습득 |

> 집필 함의: 서민은 세상을 교회 설교와 구전으로만 안다 → 타종족 박해·교리 세뇌 = 문맹 구조가 뒷받침.
