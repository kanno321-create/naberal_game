## 요약

Elucia 서민의 일상에는 6종의 생활 마법이 의지결 목걸이를 통해 활용된다. 교회는 이를 "신의 은총이 의지결에 깃든 것"으로 설명하며 공식 분배 창구 역할을 한다. Q-CORE 상 실제 기원은 직접 서술 금지. 의지결 슬롯 1개이므로 한 번에 1종만 장착 가능하며, 귀족은 고급 슬롯 의지결로 더 강력한 마법 장착 (추정).

## 1. 생활 마법 6종

이름 없는 학자 전수 마법 6종 — 교리상 "신의 은총" 으로 인식됨.

<!-- AGENT_MEMO: 생활 마법 6종의 실제 기원은 Q-CORE 2 확정 사항. 에이전트 브리핑 전용. 본문 직접 서술 절대 금지. _shared_briefing.md Q-CORE 2 섹션 참조. -->

| # | 생활 마법명 (작업 가설) | 기능 | 주 사용자 | 기후대 활용도 |
|---|-------------------|------|---------|------------|
| 1 | **불씨 유지 마법** | 화로·모닥불 불씨 장시간 유지 | 전 계층 (농민~귀족) | C 고산·D 한대전이 최고 활용 |
| 2 | **보존 마법** | 식품·약재 부패 방지 (단기) | 농민·상인·주부 | E 내륙반건조·C 고산 (저장 중시) |
| 3 | **정화 마법** | 우물·소량 물 정화 | 농민·마을 전체 | F 습지삼각주·E 내륙 (수질 불량지) |
| 4 | **상처 봉합 마법** | 가벼운 상처 지혈·봉합 | 농민·군인·주부 | 전 기후대 (기초 의료 대체) |
| 5 | **등불 마법** | 등불·초 빛 지속 | 전 계층 | 동지~봄 (긴 밤) |
| 6 | **온기 유지 마법** | 좁은 공간 온기 유지 | 농민·노인·유아 보호 | C·D 극한 지역 |

> ⚠️ 마법명은 작업 가설 · 대표님 미확정. 실제 마법 출처(할배)는 직접 서술 금지 — 교리상 "신의 은총" 으로만 표현.

## 2. 의지결 목걸이 활용 구조

발언 45 "슬롯이 딱 하나" 기반.

```
의지결 목걸이 구조:
    ├── 슬롯 1개 (성인식 목걸이에 내장)
    ├── 생활 마법 1종 장착 → 일상 사용
    ├── 교체 가능 (교회 신관 앞에서 의식 필요 · 추정)
    └── 계층별 슬롯 강도 차이 (추정)
        ├── 귀족·기사: 고급 슬롯 → 더 강한 마법 장착 가능
        └── 서민·농민: 기본 슬롯 → 생활 마법 6종 중 1종

사용 방식 (추정):
    ├── 의지 집중 → 마법 발동 (간단한 의식 동작 동반)
    ├── 마법 지속 시간: 단시간 ~ 수 시간 (마법 종류·슬롯 강도 따라 차이)
    └── 소진 시: 재충전 필요 (교회 방문 · 기도 · 추정)
```

## 3. 교리적 포장 vs 실제 (집필 가이드)

| 층위 | 표현 방식 |
|------|---------|
| **교회 교리 (공식)** | "신의 은총이 의지결에 깃들어 인간을 보호" |
| **서민 인식** | "기도하면 마법이 잘 발동된다" · "신관에게 감사" |
| **신관 인식 (타락파)** | 출처 무관심 · 권능 도구로 인식 |
| **양심파 인식** | "이 마법이 정말 신에게서 나오는가?" 의문 (Ch.09+) |
| **Q-CORE 실체** | 기원 불명 — 직접 서술 절대 금지 (_shared_briefing.md Q-CORE 2 참조) |

## 4. 계층별 마법 활용 패턴

| 계층 | 활용 마법 | 방식 |
|------|---------|------|
| **왕족·상위 귀족** | 고급 슬롯 → 강화 마법 (신성마법 근접) | 의식화·과시 |
| **기사** | 전투 보조 마법 (추정 · 대표님 미확정) | 실전 활용 |
| **신관** | 신성마법 (별도 체계) · 생활 마법 분배 권한 | 권능 도구 |
| **시민·상인** | 보존·정화 마법 (상업 활용) | 실용적 |
| **농민·농노** | 불씨·정화·온기 마법 (생존 필수) | 일상적 |
| **노예 (타종족)** | 의지결 목걸이 없음 → 생활 마법 사용 불가 | 박탈 |

## 5. 교회의 마법 통제 구조

| 통제 방식 | 내용 |
|---------|------|
| **분배 독점** | 의지결 슬롯 교체·초기화는 교회 신관만 가능 (추정) |
| **재충전 의존** | 정기 헌납 예배 출석 → 의지결 재충전 관행 (추정) |
| **이단 박탈** | 이단 심문 확정 시 의지결 목걸이 몰수 → 생활 마법 박탈 |
| **타종족 배제** | 성인식 대상 아님 → 의지결 목걸이 교부 불가 |

> 집필 함의: 이단 심문 위협 = "생활 마법 박탈" 위협 → 서민이 교회를 떠나지 못하는 실질적 이유.

## 6. 신성마법 vs 생활 마법 비교

| 항목 | 신성마법 | 생활 마법 |
|------|---------|---------|
| **사용자** | 신관 이상 · 교황청 인증 | 성인 목걸이 보유 전 계층 |
| **규모** | 대형 치유·광휘·기상 제어 (추정) | 소규모 일상 보조 |
| **출처 (교리)** | "신의 은총·권능" | "신의 은총이 의지결에 깃든 것" |
| **출처 (Q-CORE)** | 마족 창조 기술 (직접 서술 금지) | 기원 불명 — 직접 서술 금지 |
| **교회 통제** | 강함 (신관 이상만 사용) | 간접적 (분배·재충전 독점) |
