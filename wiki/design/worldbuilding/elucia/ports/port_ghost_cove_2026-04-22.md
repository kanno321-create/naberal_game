# Ghost Cove (고스트 코브) — 무소속 비공식 항구

| 항목 | 내용 |
|------|------|
| **위치** | 불명 (지도에 없음) · 서부 해안 절벽 사이 추정 |
| **등급** | C — 비공식 지하 항구 |
| **관할** | 무소속 · 여러 세력이 임시 점유 |
| **주력** | 밀수·인신매매·도망자 은신 |

## 특징

공식 지도에 존재하지 않는 항구. 자연 동굴을 확장해 만든 은신처 형태. 수면 아래로만 접근 가능한 수중 터널 입구. 이곳을 아는 자는 함구한다. 주인이 없으므로 규칙도 없다.

## 스토리 포인트

도주 · 비밀 임무 · 불법 거래가 필요한 스토리 분기에서 활용 가능. Veilglass Gate Port 허가를 받지 못한 자들이 밀항을 시도하는 곳으로도 사용될 수 있음.
