# Veilglass Gate Port (베일글래스 관문항) — 유일한 Veilglass 항로

| 항목 | 내용 |
|------|------|
| **위치** | 북서 해안 · Icehelm 해협 입구 |
| **등급** | C — 특수 전략항 |
| **관할** | Empire Choir 직속 (황제 칙령 관할) |
| **어원** | *Veilglass* (목적지명) + *Gate* (관문) + *Port* (항구) |
| **분위기** | 안개로 가득한 항구. 다른 배는 보이지 않음. 출항 의식 필수 |

## 세계관 핵심 설정

**발언 5 원문 인용**: "붉은 점 하나 — Veilglass 로 가는 유일한 항로"

이 항구만이 Veilglass 로 향하는 유일한 공식 항로를 보유한다. 이유는 다음과 같다:

1. **수정 항법 제한**: Veilglass 주변 해역은 수정 결계 구역으로 일반 항법이 작동하지 않음. 이 항구에 전속된 '결계 항법사'만이 안전 경로를 인도할 수 있음.
2. **제국 독점**: Empire Choir 성좌국이 Veilglass 항로를 정치적으로 독점. 왕복 허가증은 황제의 인장 필수.
3. **지리적 조건**: 항로 자체가 지도에 표기되지 않음. 구전으로만 전수되는 경로.

## 항구 특징

- 선착장이 단 3개. 동시 출항 제한.
- 출항 전 반드시 "안개 정화 의식" 거행 (결계 항법사 주관)
- 도착한 배의 선원들은 항구 구역을 벗어날 수 없음 (정보 차단)
- 건물: 단층 석조 창고 + 제국 관리소 + 결계 항법사 탑 1기

## 스토리 포인트

주인공이 Veilglass 로 가려면 반드시 이 항구를 통해야 함. 허가증 획득 또는 밀항이 스토리 분기점이 될 수 있음.
