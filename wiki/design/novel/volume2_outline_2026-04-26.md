## 1. Vol.2 6 챕터 outline (Q17 분류표 + Q15 차별 점층 + Q16 황금 비율 적용)

### Ch.01 — 입국 (조기 회수 ① 부츠 흙)

**위치**: 제국 서부 국경 입국 직후 → 모험가 길드 등록 → 첫 의뢰

**핵심 사건**:
1. 1권 결말부 직후. 광장 동쪽 골목 진입 → 흰 외투의 자 추적 (단서 발견)
2. 제국 모험가 길드 등록
3. 첫 의뢰 받음 — 외곽 마을 토벌·조사
4. 마을 도착 → **마을 흙 = 붉은 황토** (1권 결말 부츠 흙과 동일 색) → 첫 인지 부조화 씨앗

**Q17 회수**: 부츠 흙 (1단계 — 물리적 단서 회수)

**Q15 차별 씨앗 빈도**: 챕터당 **1회** (광장 단역의 마족 흐름 한 마디)

**Q16 페널티 빈도**: 챕터당 **2~3회** (가시화 시작 — 회복 안 되는 시야 한구석)

**[대표님 확정 대기]**:
- (1) 제국 이름 (1권 = Silvan / Ilarien — Vol.2 = 제국)
- (2) 제국 모험가 길드 이름
- (3) 첫 외곽 마을 이름 (붉은 황토 마을)
- (4) 첫 의뢰 제목 + 의뢰주
- (5) 광장 흰 외투의 자 첫 단서 발현 방식 (이름·자세·접근 방식)

### Ch.02 — 첫 의뢰 (조기 회수 ② 검은 머리 뾰족귀 + 마족 누명 방아쇠)

**위치**: 첫 외곽 마을 (붉은 황토)

**핵심 사건**:
1. 마을 도착 → **학살 잿더미 + 생존자 증언**
2. 생존자/조작 소문: *검은 머리 마족 소녀 + 날카로운 마족 어른 (180cm) 도주 목격*
3. 나이트 = 마족 범인 확신 + 분노
4. 그러나 자기 부츠 의문 + 손가락 마법진 미세 점멸 = 첫 인지 부조화 씨앗

**Q17 회수**: 검은 머리 뾰족귀 (1.5단계 — *오해된* 회수 / 마족 누명 방아쇠)

**Q15 차별 씨앗 빈도**: 챕터당 **2회** (마을 사람들 마족 욕설 + 광장 추모 분위기)

**Q16 페널티 빈도**: 챕터당 **2~3회** (잿빛 시야 + 자아 한 박자 깨짐)

**[대표님 확정 대기]**:
- (6) 학살 마을 생존자 (이름·자세·증언 방식)
- (7) 마족 어른 이름 / 외형 / 종족 분파 (세리스의 보호자)
- (8) 마을 학살 규모 (사망자 수·생존자 수)
- (9) 길드 보고 후 후속 의뢰 (다음 마을 추적인지·다른 의뢰인지)

### Ch.03 — 흰 머리 신관 (중기 회수 ③ 다른 우주적 존재 요원 가시화)

**위치**: 제국 도시 어딘가 / 또는 두 번째 마을 인근

**핵심 사건**:
1. 흰 머리 신관 (다른 요원) 첫 등장
2. 나이트 ↔ 흰 머리 신관 첫 대화 / 조우
3. 신관 = *너는 우리 중 하나다* 암시 (직접 X / 간접 박제)
4. 나이트 = 흰 머리 신관의 자세에서 *자기와 같은 종류의 자* 인지

**Q17 회수**: 흰 머리 신관 (2단계 — 다른 우주적 존재 요원 정체 가시화)

**Q15 차별 씨앗 빈도**: 챕터당 **2회** (교회의 마족 추적 명령 + 마을 사람들의 마족 사냥 행위)

**Q16 페널티 빈도**: 챕터당 **2~3회** + **시간차 누수** (행위 후 며칠 후 발현 — 사흘 전 힘 사용 대가가 이제야)

**[대표님 확정 대기]**:
- (10) 흰 머리 신관 이름 / 직위 / 정체
- (11) 흰 머리 신관 첫 등장 위치 (성당·골목·마을·길거리)
- (12) 신관과 나이트 첫 조우의 *발화점* (대화·시선·갈등)
- (13) 신관이 나이트에게 던지는 첫 결정적 한 마디

### Ch.04 — 두 번째 학살 (중기 회수 ④ 광장 흰 외투 + 인지 부조화 폭발)

**위치**: 두 번째 마을

**핵심 사건**:
1. 두 번째 마을 학살 발견 → **자기 고유 마법 흔적 (염동력 절단면) 발견**
2. 우주적 존재 (가짜 내면 자아) = *마족 잔꾀다* 라며 시야 가림 / 기만
3. 광장 흰 외투의 자 (교황청 부 추기경) 전면 등장 → *너에게 임무가 있다* 류 접근
4. 추기경 = 마족 토벌 정당화 + 나이트 도구 활용 시도
5. 단서: *단칼에 절단된 촌장 목 단면 = 나이트가 폐광에서 도적 두목 처치 시 쓴 방식 (세상에서 나만 아는 방식)*

**Q17 회수**: 광장 흰 외투 (2.5단계 — 교황청 개입 가시화 + 차별 폭발 정치 배후)

**Q15 차별 씨앗 빈도**: 챕터당 **3회** (가시화 — 광장 마족 사냥 + 길거리 마족 욕설 + 추기경 설교)

**Q16 페널티 빈도**: 챕터당 **3~4회** + 강도 1단계 (잿빛 코피 덩어리·일상 균열)

**[대표님 확정 대기]**:
- (14) 두 번째 마을 이름
- (15) 광장 흰 외투 추기경 이름 / 정확한 직위
- (16) 추기경이 나이트에게 제안하는 *임무* 의 표면적 명분
- (17) 검집 레이피어 날 끝 *처참한 이 빠짐* 발견의 trigger
- (18) 가짜 내면 자아의 첫 결정적 기만 한 마디

### Ch.05 — 의심의 가속 (후기 회수 ⑤ 어디서 봤더라 + 동족 분신 흔적)

**위치**: 제국 도시 / 길거리 / 또는 세 번째 마을 추적 길

**핵심 사건**:
1. 나이트 ↔ 적대자 (다른 요원? 또는 마족 추격대) 충돌
2. 그 자에게서 *나와 같은 흔적* (백발 / 손가락 마법진 / 두 박자) 발견
3. 내면 자아 균열 시작 — *내가 누구인가* 의문
4. 진짜 내면 vs 가짜 내면 (우주적 존재) 구분 시작
5. 추기경 / 흰 머리 신관 / 광장 흰 외투 = 모두 *우주적 존재의 도구* 인지 가능성

**Q17 회수**: *어디서 봤더라* (3단계 — 동족 분신 흔적 발견 + 자아 균열)

**Q15 차별 씨앗 빈도**: 챕터당 **3회** (마족 토벌 광기 절정 + 세리스 + 마족 어른 추격 시작)

**Q16 페널티 빈도**: 챕터당 **3~4회** + **곁의 사람 부식** 가시화 (동행자가 있다면 — 4기둥 ⑤ 수동 차원)

**[대표님 확정 대기]**:
- (19) Vol.2 동행자 (1권 록솔/안나 합류 여부 / 신규 동료 / 단독 행동)
- (20) Ch.05 적대자 = 다른 우주적 존재 요원? 또는 마족 추격? 또는 다른 종류?
- (21) *어디서 봤더라* 의 자 = 적대자와 동일 인물? 또는 별개 단서?

### Ch.06 — 세 번째 학살 (클라이맥스 회수 ⑥ 두 종류의 박자 + 일거 폭발)

**위치**: 세 번째 마을 (세리스 + 마족 어른 코너)

**핵심 사건** (캐논 § 3-3 박제 그대로):
1. 세 번째 학살 *실시간* 진행
2. 나이트 눈 뜸 → 여관방 X / **불타는 마을 한가운데** ✅
3. 손에 피 + 발밑 시체 산 + 세리스 + 마족 어른 (피투성이) 대면
4. 마족 어른: *"제발, 아이만은……!"* (피 토하며 빌음)
5. 세리스: *세상에서 가장 끔찍한 악마를 마주한 눈*
6. **결정적 진실 폭로**: 학살자 = 나이트 본인 / 마족 = 막으려 한 자
7. **두 종류의 박자 회수**: 수천 년 분신 자각 + 1권 모든 씨앗 (부츠 흙 + 검은 머리 뾰족귀 + 흰 머리 신관 + 광장 흰 외투 + 어디서 봤더라) **한 번에 의미 폭발** (오버로드식 *진실의 일거 폭발*)
